## Milestone Summary

**Decimal Phases:** None

**Key Decisions:**

- [Phase 320 / D-EXP-01]: Sheet name format `{NIP}_{FullName}` (NIP-first → collision-free). Truncate FullName dari belakang kalau total > 31 char. Exclude Excel-invalid chars (`\ / ? * [ ] :`).
- [Phase 320 / D-EXP-02]: Filter peserta yang dapat sheet = Status `Completed` + `Abandoned` only. Skip `InProgress`, `Not Started`, `Cancelled`.
- [Phase 320 / D-EXP-03]: HC permission full sama Admin — no divisi restriction.
- [Phase 320 / D-EXP-04]: PNG generate paralel `Task.WhenAll` dengan `MaxDegreeOfParallelism = Environment.ProcessorCount` (target export 50 peserta < 30 detik).
- [Phase 321 / D-EDIT-01]: Edit jawaban hanya tersedia untuk session `Status == Completed` + bukan `IsManualEntry` + bukan Assessment Proton Tahun 3 (helper `IsEditable`).
- [Phase 321 / D-EDIT-02]: GradingService refactor — extract `ComputeScoreAndETInternalAsync(session, overrideAnswers?)` no-side-effect untuk reuse di RegradeAfterEditAsync + PreviewEditScore.
- [Phase 321 / D-EDIT-03]: Transaction scope membungkus seluruh edit save flow (update PackageUserResponses + insert AssessmentEditLog + RegradeAfterEditAsync + cascade cert/TR + insert AuditLog). Rollback total kalau exception.
- [Phase 321 / D-EDIT-04]: Concurrency token via `AssessmentSession.UpdatedAt` — stale reject dengan TempData error.
- [Phase 321 / D-EDIT-05]: Per-user table action column pakai layout hybrid: `View Results` + `Activity Log` 🕐 tetap inline, sisanya (`Edit Jawaban`, `Reset`, `Akhiri Ujian`, `Reshuffle`) pindah ke dropdown ⋮.
- [Phase 322 / D-21]: Strategy D Hybrid — wrapper hx-vals → URL query string per-wrapper (NOT hx-vals inheritance). Rationale: Bug 2 prevention by-design (HTMX hx-vals inherit ke descendant, breaks Tab 2 semantic).
- [Phase 322 / D-10]: URL bookmark backward compat preserved (Tab 1 full param, Tab 2 partial — no inheritance).
