| T-341-02 | Elevation of Privilege | GET ManageOrgLevelLabels + 3 POST actions | mitigate | Method-level `[Authorize(Roles = "Admin, HC")]` di 4 action baru (D-09); class-level `[Authorize]` tetap untuk GetLevelLabels endpoint |
| T-341-03 | Tampering | POST AddLevelLabel (devtools level=99) | mitigate | D-08 server check `level != GetMaxConfiguredLevel() + 1` → friendly JSON reject (Plan 01 Task 2 Step 6) |
| T-341-04 | Tampering | POST UpdateLevelLabel / AddLevelLabel (label payload abuse) | mitigate | D-05 inline server checks: IsNullOrWhiteSpace + Trim + Length > 50 + duplicate AnyAsync (Plan 01 Task 2 Step 5-6); Pitfall 3 Trim placement |
| T-341-05 | Tampering | POST DeleteLevelLabel (devtools level=0 mid-tier) | mitigate | Server guard `level != GetMaxConfiguredLevel()` + `OrganizationUnits.AnyAsync(u => u.Level == level)` check (Plan 01 Task 2 Step 7); Pitfall 7 mitigation |
