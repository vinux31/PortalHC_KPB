# Plan 87-01: Create Dashboard Test Data Seed Action

**Goal:** Create comprehensive test data for all dashboard scenarios across 6 roles, following the established seed data pattern from Phases 85 and 90.

## Context

Phase 87 requires full role coverage (Admin, HC, SrSpv, SectionHead, Coach, Coachee) with realistic data across:
- Assessment sessions in various statuses (Open, Upcoming, Completed)
- IDP items in various statuses (Pending, In Progress, Completed)
- Proton deliverable progress for Coaching Proton stats
- Training records for Assessment Analytics
- Audit log entries for AuditLog page

The seed data action must be:
- **Idempotent**: Check existing data before insert, safe to run multiple times
- **Comprehensive**: Create realistic test scenarios for all dashboard metrics
- **Role-aware**: Create appropriate data relationships for all 6 roles

- **Proton Deliverable Progress**: Create 10-15 deliverable progress records:
