# Plan 87-02: QA Home/Index and CDP Dashboard Data Accuracy

**Goal:** Verify dashboard metrics are accurate across all roles through code review and spot-check browser verification.

## Context

Phase 87 requires deep verification of dashboard data accuracy:
- **Home/Index**: IDP progress, pending assessments, mandatory training, recent activities, upcoming deadlines
- **CDP Dashboard - Coaching Proton tab**: Progress stats per bagian/unit
- **CDP Dashboard - Assessment Analytics tab**: Assessment and training data + export

The verification approach is:
1. **Code review**: Trace each dashboard metric to controller query, verify role scoping correct
2. **Browser spot-check**: Confirm key metrics render correctly with test data
3. **Bug fix**: Fix any bugs found inline (<100 lines), flag larger issues for discussion
