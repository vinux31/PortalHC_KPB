## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client POST → Assign/Edit/Import | `AssignmentUnit` from payload becomes persisted mapping state |
| Cleanup batch → mapping mutation | auto-fix loop can overwrite persisted `AssignmentUnit` |
| Reactivate POST → mapping state | reactivation re-activates a mapping whose unit may be stale |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
