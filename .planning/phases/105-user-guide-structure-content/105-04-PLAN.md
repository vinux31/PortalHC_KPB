## Requirements
- GUIDE-CONTENT-01: Each tab displays step-by-step instructions with numbered step cards
- GUIDE-CONTENT-02: Step cards include icon, title, and description

## Problem Summary
The Admin Panel module card shows "12 panduan tersedia" but current guides combine multiple topics into single entries:
1. Cara Kelola Pekerja ✓ (exists - covers CRUD + Import/Export)
2. Cara Upload File KKJ Matrix & CPDP Files ✓ (exists)
3. Cara Mapping Coach-Coachee ✓ (exists)
4. Cara Kelola Ujian (Buat Assessment, Monitoring, Soal Ujian) ✓ (exists - covers multiple topics)
5. Audit Log & Training Record ✓ (exists - covers 2 topics)

Guides 4 and 5 combine multiple topics. To reach 12 guides and improve usability, we need to split these into focused, single-topic guides.

## Solution
Split existing combined guides into separate, focused guides for better organization and usability.
