## Notes

- Use `cubic-bezier(0.34, 1.56, 0.64, 1)` for bounce effects on cards and badges
- Use `cubic-bezier(0.68, -0.55, 0.265, 1.55)` for elastic rotations on chevrons
- Use `cubic-bezier(0.4, 0, 0.2, 1)` for smooth, premium transitions
- Always add `@media (prefers-reduced-motion: reduce)` for accessibility
- Keep animations subtle - avoid over-animating which can feel gimmicky
- Test animations on low-end devices to ensure performance
