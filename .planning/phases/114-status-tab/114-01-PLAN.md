**Backend (ProtonDataController.cs):** Add a `StatusData` action that returns JSON for all Bagian > Unit > Track combinations with silabus and guidance completeness.

1. Query `ProtonKompetensiList` where `IsActive == true`, Include `SubKompetensiList` then ThenInclude `Deliverables`. Group by (Bagian, Unit, ProtonTrackId). A group is "silabus complete" when ALL Kompetensi in that group have at least 1 SubKompetensi AND every SubKompetensi has at least 1 Deliverable.
2. Query `CoachingGuidanceFiles`, select distinct (Bagian, Unit, ProtonTrackId) into a HashSet for O(1) lookup.
3. Query `ProtonTracks` ordered by Urutan.
4. Iterate `OrganizationStructure.SectionUnits` (Bagian > Units) x tracks, build flat JSON array: `{ bagian, unit, trackId, trackName, silabusOk, guidanceOk }`. Missing combos in the silabus query = incomplete. Missing combos in guidance set = incomplete.
5. Return `Json(result)`.

Use the exact code pattern from 114-RESEARCH.md `StatusData endpoint` section.

**Frontend (Views/ProtonData/Index.cshtml):**

1. **Tab reorder:** Insert a new Status tab button BEFORE the existing Silabus tab button. Make Status tab `active` (class `nav-link active`, `aria-selected="true"`). Remove `active` from Silabus tab button (change to `nav-link`, `aria-selected="false"`).

2. **Tab pane:** Insert a new `tab-pane` div with id `statusTabContent` BEFORE the existing silabusTabContent pane. Make it `show active`. Remove `show active` from silabusTabContent (change to just `fade`). The pane contains:
   - A `div.card.border-0.shadow-sm` wrapper
   - Inside: a `table.table.table-bordered.table-sm` with thead columns: (empty header) | Silabus | Guidance
   - `tbody#statusTableBody` initially showing a loading spinner row

3. **JS function `loadStatusData()`:**
   - Show spinner in `#statusTableBody`
   - `$.get('/ProtonData/StatusData', function(data) { ... })`
   - Iterate data array. Track `currentBagian` and `currentUnit` to insert header rows:
     - Bagian row: `class="table-light"`, `<td class="fw-bold">{bagian}</td><td></td><td></td>`

4. **Event binding:**
   - Call `loadStatusData()` on `$(function() { ... })` (page load, since Status is default tab)

Use the exact JS pattern from 114-RESEARCH.md `JS rendering pattern` section.
