<objective>
Add a GenerateCertificate boolean toggle to AssessmentSession so HC can control whether an assessment produces certificates. Results page button, Certificate action, and Training Records views all respect this flag. Existing assessments default to true (backward compatible); new assessments default to false.

Purpose: Prevent "View Certificate" from appearing on trainings/assessments that do not require certificates.
Output: Migration, updated model, toggle in Create/Edit forms, guarded Results button and Certificate action, hidden certificate column in Records views.
</objective>
