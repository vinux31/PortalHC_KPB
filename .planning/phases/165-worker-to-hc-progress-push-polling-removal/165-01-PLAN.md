   - Push: `SendAsync("workerSubmitted", new { sessionId = id, workerName = user.FullName/etc, score = finalPercentage, result, status = "Completed", totalQuestions })`
   - CRITICAL: Add push in BOTH the package path (after line ~1662) AND the legacy path (after its rowsAffected check). Both paths must push.
