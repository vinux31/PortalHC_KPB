2. In `Controllers/AdminController.cs`, add TWO new actions in the COACH-COACHEE MAPPING region (after the existing CoachCoacheeMapping GET action area, before any other region):

**Action A: GET DownloadMappingImportTemplate**
- `[HttpGet]`, `[Authorize(Roles = "Admin, HC")]`
- Create XLWorkbook with worksheet "Import CoachCoachee"
- Row 1 headers (bold, green #16A34A bg, white text): "NIP Coach", "NIP Coachee"
- Row 2 example (italic, gray text): "123456", "789012"
- Row 3 note (italic, dark red): "Isi NIP Coach dan NIP Coachee. StartDate otomatis hari ini, IsActive otomatis true."
- `ws.Columns().AdjustToContents()`
- Return File with content type `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`, filename `coach_coachee_import_template.xlsx`

**Action B: POST ImportCoachCoacheeMapping**
- `[HttpPost]`, `[Authorize(Roles = "Admin, HC")]`, `[ValidateAntiForgeryToken]`
- Parameter: `IFormFile? excelFile`
- Validation: null/empty file check, extension check (.xlsx/.xls only), size check (10MB max) — on error set TempData["ImportError"] and redirect to CoachCoacheeMapping
- Load all users into a dictionary keyed by NIP: `var usersByNip = await _context.Users.Where(u => u.NIP != null).ToDictionaryAsync(u => u.NIP!, u => new { u.Id, u.Section, u.Unit });` (handle duplicate NIP with GroupBy + First if needed)
- Load all existing CoachCoacheeMapping records: `var allMappings = await _context.CoachCoacheeMappings.ToListAsync();`
- Parse Excel rows (skip row 1 header), for each row:
  - Read NIP Coach (cell 1), NIP Coachee (cell 2) — trim whitespace
  - If either NIP is empty: Status="Error", Message="NIP Coach atau NIP Coachee kosong"
  - If NIP Coach not found in usersByNip: Status="Error", Message="NIP Coach '{nipCoach}' tidak ditemukan"
  - If NIP Coachee not found: Status="Error", Message="NIP Coachee '{nipCoachee}' tidak ditemukan"
  - If coach == coachee (same user ID): Status="Error", Message="Coach tidak dapat menjadi coachee dirinya sendiri"
  - Check existing active mapping (same CoachId + CoacheeId + IsActive=true): Status="Skip", Message="Mapping sudah aktif"
  - Check existing inactive mapping (same CoachId + CoacheeId + IsActive=false): reactivate it — set IsActive=true, StartDate=DateTime.Today, EndDate=null. Status="Reactivated", Message="Mapping diaktifkan kembali"
  - Otherwise: create new CoachCoacheeMapping with CoachId, CoacheeId, IsActive=true, StartDate=DateTime.Today, AssignmentSection=coachee's Section, AssignmentUnit=coachee's Unit. Status="Success", Message="Berhasil dibuat"
- `await _context.SaveChangesAsync()`
- Log audit: `_context.AuditLogs.Add(new AuditLog { ... Action = "ImportCoachCoacheeMapping", Detail = $"Import {successCount} mapping baru, {reactivatedCount} diaktifkan kembali, {skipCount} dilewati, {errorCount} error", ... })`
- Store results in TempData as JSON: `TempData["ImportResults"] = System.Text.Json.JsonSerializer.Serialize(results);`
- Redirect to CoachCoacheeMapping action (preserving current filters not needed — just redirect to base URL)
