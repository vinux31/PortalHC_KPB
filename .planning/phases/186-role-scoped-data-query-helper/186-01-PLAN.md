**Method 2: BuildSertifikatRowsAsync** — signature: `private async Task<List<SertifikatRow>> BuildSertifikatRowsAsync()`

Implementation steps:
1. Call `GetCurrentUserRoleLevelAsync()` to get user and roleLevel.
2. Build a `List<string> scopedUserIds` based on roleLevel:
   - L1-3 (`UserRoles.HasFullAccess(roleLevel)`): `null` (no filter — query all)
   - L4 (`UserRoles.HasSectionAccess(roleLevel)`): query `_context.Users.Where(u => u.IsActive && u.Section == user.Section).Select(u => u.Id).ToListAsync()`
   - L5 (`roleLevel == 5`): query `_context.CoachCoacheeMappings.Where(m => m.CoachId == user.Id && m.IsActive).Select(m => m.CoacheeId).ToListAsync()` — then add `user.Id` to the list (coach sees own certs too)
   - L6 (`roleLevel == 6`): `new List<string> { user.Id }`

3. Query TrainingRecords:
```csharp
var trQuery = _context.TrainingRecords
    .Include(t => t.User)
    .Where(t => t.SertifikatUrl != null);
if (scopedUserIds != null)
    trQuery = trQuery.Where(t => scopedUserIds.Contains(t.UserId));
var trainingRows = await trQuery.Select(t => new SertifikatRow
{
    SourceId = t.Id,
    RecordType = RecordType.Training,
    NamaWorker = t.User != null ? t.User.FullName : "",
    Bagian = t.User != null ? t.User.Section : null,
    Unit = t.User != null ? t.User.Unit : null,
    Judul = t.Judul ?? "",
    Kategori = t.Kategori,
    NomorSertifikat = t.NomorSertifikat,
    TanggalTerbit = t.Tanggal,
    ValidUntil = t.ValidUntil,
    Status = SertifikatRow.DeriveCertificateStatus(t.ValidUntil, t.CertificateType),
    SertifikatUrl = t.SertifikatUrl
}).ToListAsync();
```

4. Query AssessmentSessions:
```csharp
var asQuery = _context.AssessmentSessions
    .Include(a => a.User)
    .Where(a => a.GenerateCertificate && a.IsPassed == true);
if (scopedUserIds != null)
    asQuery = asQuery.Where(a => scopedUserIds.Contains(a.UserId));
var assessmentRows = await asQuery.Select(a => new SertifikatRow
{
    SourceId = a.Id,
    RecordType = RecordType.Assessment,
    NamaWorker = a.User != null ? a.User.FullName : "",
    Bagian = a.User != null ? a.User.Section : null,
    Unit = a.User != null ? a.User.Unit : null,
    Judul = a.Title,
    Kategori = a.Category,
    NomorSertifikat = a.NomorSertifikat,
    TanggalTerbit = a.CompletedAt,
    ValidUntil = a.ValidUntil,
    Status = SertifikatRow.DeriveCertificateStatus(a.ValidUntil, null),
    SertifikatUrl = null
}).ToListAsync();
```

5. Merge and return:

var rows = new List<SertifikatRow>(trainingRows.Count + assessmentRows.Count);
rows.AddRange(trainingRows);
