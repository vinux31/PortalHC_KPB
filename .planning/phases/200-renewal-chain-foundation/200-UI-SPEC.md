# Phase 200 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

## Phase Nature: Backend Infrastructure Only

Phase 200 adalah fase infrastruktur data murni. Tidak ada halaman baru, tidak ada komponen UI baru, tidak ada interaksi pengguna baru yang dibuat di phase ini. Yang berubah:

- Model C# (FK properties pada AssessmentSession dan TrainingRecord)
- EF Core migration (schema DB)
- Query logic di BuildSertifikatRowsAsync (IsRenewed flag computation)

Perubahan pada BuildSertifikatRowsAsync menghasilkan satu efek visual tersembunyi: field `IsRenewed` sekarang tersedia di `SertifikatRow`, tapi belum digunakan di UI sampai Phase 204 (CDP Enhancement). Tidak ada elemen UI yang berubah tampilan di phase ini.

Kontrak desain di bawah ditetapkan sebagai baseline untuk fase-fase downstream (201–204) yang membangun di atas infrastruktur ini.

## Interaction Contract

Phase 200 tidak menambah interaksi baru. Dokumentasi ini berlaku sebagai baseline untuk fase downstream.

### Existing Interaction Patterns (dipertahankan)

| Pattern | Implementasi |
|---------|-------------|
| Loading state | Bootstrap spinner: `<div class="spinner-border text-primary" role="status">` |
| Success feedback | Bootstrap alert `alert-success`, auto-dismiss 3 detik |
| Error feedback | Bootstrap alert `alert-danger`, persistent sampai user dismiss |
| Table sort | Tidak ada client-side sort — query-based |
| Badge status | `badge rounded-pill bg-{color}` — Bootstrap 5.3 pattern |

### State Machine untuk IsRenewed (referensi fase downstream)

```
SertifikatRow.IsRenewed = false + Status = Expired    → badge "Expired" (danger)
SertifikatRow.IsRenewed = true  + Status = Expired    → badge "Expired" (danger) + badge "Sudah Diperbarui" (secondary)
SertifikatRow.IsRenewed = false + Status = AkanExpired → badge "Akan Expired" (warning)
SertifikatRow.IsRenewed = true  + Status = AkanExpired → badge "Akan Expired" (warning) + badge "Sudah Diperbarui" (secondary)
SertifikatRow.IsRenewed = false + Status = Aktif      → badge "Aktif" (success)
SertifikatRow.IsRenewed = true  + Status = Aktif      → tidak mungkin (logika bisnis)
```
