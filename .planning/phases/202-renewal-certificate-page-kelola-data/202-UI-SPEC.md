## Spacing Scale

Declared values (multiples of 4) — mengikuti Bootstrap utility classes:

| Token | Value | Bootstrap Class | Usage |
|-------|-------|----------------|-------|
| xs | 4px | `gap-1`, `p-1` | Gap antar icon dan label inline |
| sm | 8px | `gap-2`, `p-2` | Padding kompak tombol, gap antar filter |
| md | 16px | `p-3`, `mb-3` | Default elemen spacing, card body padding |
| lg | 24px | `py-4`, `mb-4` | Section padding, gap bawah filter bar |
| xl | 32px | `mb-5` | Jarak antar section major |
| 2xl | 48px | `py-5` (jarang) | Page-level padding atas/bawah (container-fluid) |
| 3xl | 64px | tidak digunakan | Tidak dipakai di halaman ini |

Exceptions:
- Touch target minimum tombol Aksi (Renew per baris): 32px tinggi via `btn btn-sm` — Bootstrap default
- Checkbox per baris: 20px x 20px via `form-check-input` — Bootstrap default
- Badge count di summary: `px-3 py-1` inline pada span badge

## Layout & Interaction Contracts

### Layout Halaman

```
container-fluid py-4
├── Header row
│   ├── h4 [icon] Renewal Sertifikat  (kiri)
│   └── [tidak ada tombol kanan — hanya breadcrumb]
├── Breadcrumb: Kelola Data > Renewal Sertifikat
├── Summary row (2 card)
│   ├── card: Expired (N) — text-danger, fs-2 fw-bold
│   └── card: Akan Expired (N) — text-warning, fs-2 fw-bold
├── Toolbar row (hanya muncul jika checkbox tercentang)
│   └── btn btn-warning [Renew Selected (N)]  — d-none default, d-flex setelah check
├── Filter bar card
│   └── row g-2: [Bagian] [Unit] [Kategori] [Status] [Reset]
└── Table container card  (id="cert-table-container")
    └── _RenewalCertificateTablePartial (AJAX refresh target)
```

### Struktur Tabel

Kolom (sesuai CONTEXT.md — locked):

| # | Kolom | Lebar | Catatan |
|---|-------|-------|---------|
| 1 | Checkbox | 40px | `form-check-input cb-select` |
| 2 | No | 48px | nomor urut per halaman |
| 3 | Nama | auto | NamaWorker |
| 4 | Judul Sertifikat | auto | row.Judul |
| 5 | Kategori | auto | row.Kategori ?? "-", `d-none d-md-table-cell` |
| 6 | Sub Kategori | auto | row.SubKategori ?? "-", `d-none d-md-table-cell` |
| 7 | Valid Until | 110px | dd MMM yyyy format |
| 8 | Status | 120px | Badge Expired/Akan Expired |
| 9 | Aksi | 90px | Tombol Renew |

Catatan: Kolom Bagian dan Unit tidak ditampilkan di tabel (tidak ada di spec CONTEXT.md kolom tabel) — hanya dipakai sebagai filter dropdown.

### Sorting Default

Expired di atas lalu Akan Expired, dalam grup sort ValidUntil ascending (paling dekat expired di atas). Sumber: CONTEXT.md locked.

### Filter Dropdowns — Urutan

1. Bagian — server-rendered options dari ViewBag.AllBagian
2. Unit — disabled awal, populate via AJAX `/CDP/GetCascadeOptions?section=` setelah Bagian dipilih
3. Kategori — server-rendered options dari ViewBag.AllCategories
4. Status — hardcoded options: "Semua Status" / "Expired" / "AkanExpired"
5. Reset button — clear semua filter, reset Unit ke disabled

Sub Kategori: tidak ada filter dropdown untuk Sub Kategori (Claude's Discretion — kolom tabel saja sudah cukup, konsisten dengan filter 4 kolom yang dikunci di CONTEXT.md).

### Bulk Select — Interaction Contract

State machine checkbox:

| State | Aksi User | Efek |
|-------|-----------|------|
| Kosong (0 tercentang) | Centang checkbox baris A (kategori X) | selectedKategori = X; semua checkbox kategori != X → disabled + title "Hanya sertifikat kategori sama"; tombol Renew Selected muncul |
| 1+ tercentang | Centang checkbox baris lain (kategori X) | Tambah ke selection jika kategori sama; skip jika disabled |
| 1+ tercentang | Uncheck semua | selectedKategori = null; semua checkbox → enabled; tombol Renew Selected hilang |
| Filter diubah | AJAX refresh tabel | selectedKategori = null; tombol Renew Selected hilang (DOM replace) |

TrainingRecord tanpa Kategori (`row.Kategori == null`): treat sebagai kategori unik `""` (string kosong) — checkbox TrainingRecord tanpa kategori hanya bisa di-bulk dengan sesama TrainingRecord tanpa kategori. Ini mencegah bug Pitfall 1 dari RESEARCH.md.

### Tombol Renew Selected

- Default: `d-none` (tersembunyi)
- Muncul: setelah minimal 1 checkbox dicentang → `d-none` dihapus, `d-flex` ditambahkan
- Label: "Renew Selected (N)" di mana N = jumlah checkbox tercentang
- Aksi: build URL multi-param JS, redirect langsung ke `/Admin/CreateAssessment?renewSessionId=...` tanpa confirm dialog
- Warna: `btn btn-warning` (konsisten dengan tombol Renew satuan)

### Empty State

Tampil ketika tidak ada sertifikat expired/akan expired yang perlu di-renew (semua sudah di-renew atau memang tidak ada):

```
[Ilustrasi: bi-patch-check-fill fs-1 text-success]
Tidak ada sertifikat yang perlu di-renew
Semua sertifikat aktif atau sudah di-renew. Kembali ke Kelola Data.
[Tombol: btn btn-outline-secondary — Kembali ke Kelola Data]
```

Implementasi: centered `div.text-center.py-5` dalam tbody atau di bawah tabel — gunakan pola `text-center text-muted py-4` yang sudah ada, dengan tambahan icon dan link kembali.
