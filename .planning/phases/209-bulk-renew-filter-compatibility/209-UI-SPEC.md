## Interaction Contract

### IC-01: Tombol "Renew N Pekerja" per Group

- **Kondisi muncul:** Saat minimal 1 checkbox `.cb-select[data-group-key="{key}"]` dalam group ter-centang
- **Kondisi hilang:** Saat 0 checkbox di group tersebut ter-centang
- **Posisi:** Di dalam `.card-header` accordion, setelah div badge, dengan `ms-auto me-2` (rata kanan, margin kanan 8px)
- **Ukuran:** `btn-sm` (padding 4px 8px)
- **Warna:** `btn-warning` (background #ffc107, teks gelap)
- **Ikon:** `bi bi-arrow-repeat me-1`
- **Teks dinamis:** "Renew [N] Pekerja" — N diperbarui setiap ada perubahan centang
- **Klik:** `event.stopPropagation()` WAJIB untuk mencegah trigger toggle accordion, lalu buka modal konfirmasi
- **Render:** Server-side di partial dengan class `d-none` awal, JS yang show/hide via `classList.remove/add('d-none')`

### IC-02: Checkbox Lock Per Group

- **Trigger:** Saat user mencentang checkbox di group mana pun
- **Efek:** Semua checkbox di group lain langsung `disabled = true`
- **Unlock:** Saat semua checkbox di active group di-uncheck, semua checkbox kembali `disabled = false`
- **Select-all:** `cb-group-select-all` di header tabel group mencentang/uncentang semua `.cb-select` dalam group-nya
- **Select-all disabled:** Jika group lain sedang aktif, klik `cb-group-select-all` diabaikan
- **Reset saat pagination:** Saat `refreshGroupTable()` selesai, `selectedGroupKey` di-reset ke `null` dan semua checkbox di-enable

### IC-03: Modal Konfirmasi Bulk Renew

- **Trigger:** Klik tombol "Renew N Pekerja" di header group
- **Konten modal:** Body menampilkan jumlah dan judul sertifikat dari group bersangkutan
- **Klik "Lanjutkan":** Redirect ke `/Admin/CreateAssessment?{params}`
- **Klik "Batal" / close:** Modal tertutup, tidak ada aksi, checkbox state tidak berubah
- **Ukuran modal:** `modal-dialog` default (tidak ada `modal-lg` atau `modal-sm`)

### IC-04: Filter — Empty State

- **Trigger:** Filter aktif tetapi semua group menghasilkan 0 anggota
- **Tampilan:** Icon `bi bi-funnel fs-1 text-muted`, heading, body teks, tombol "Reset Filter"
- **Tombol "Reset Filter":** `onclick="document.getElementById('btn-reset').click()"` — delegate ke tombol reset yang sudah ada
- **Dibedakan dari:** Empty state "tidak ada data" yang memakai icon `bi bi-patch-check-fill text-success`
- **Mekanisme:** ViewModel `RenewalGroupViewModel` perlu properti `bool IsFiltered` — di-set di controller saat ada filter param aktif

### IC-05: Summary Cards — Update via Filter

- **Mekanisme:** `updateSummaryCards()` sudah berfungsi dari Phase 208 — dipanggil otomatis di `refreshTable()`
- **Tidak ada perubahan interaksi:** FILT-02 sudah 90% handled; tidak ada IC tambahan
