**1. ManageOrganization [HttpGet, Authorize(Roles = "Admin, HC")]**
- Query roots: `_context.OrganizationUnits.Include(u => u.Children.OrderBy(c => c.DisplayOrder).ThenBy(c => c.Name)).ThenInclude(c => c.Children.OrderBy(gc => gc.DisplayOrder).ThenBy(gc => gc.Name)).Where(u => u.ParentId == null).OrderBy(u => u.DisplayOrder).ThenBy(u => u.Name).ToListAsync()`
- `ViewBag.PotentialParents = await _context.OrganizationUnits.Where(u => u.IsActive).OrderBy(u => u.Level).ThenBy(u => u.DisplayOrder).ToListAsync()`
- Jika query param `editId` ada: `ViewBag.EditUnit = await _context.OrganizationUnits.FindAsync(editId)`
- Return `View("ManageOrganization", roots)`

**2. AddOrganizationUnit [HttpPost, Authorize(Roles = "Admin, HC"), ValidateAntiForgeryToken]**
- Params: `string name, int? parentId`
- Validasi: nama kosong → TempData["Error"] = "Nama tidak boleh kosong." + TempData["ShowAddForm"] = true
- Cek duplikat: `_context.OrganizationUnits.AnyAsync(u => u.Name == name.Trim())` → TempData["Error"] = "Nama unit sudah digunakan. Gunakan nama yang berbeda." + TempData["ShowAddForm"] = true
- Hitung Level dari parent (parent == null → 0, else parent.Level + 1)
- Hitung DisplayOrder: `MaxAsync(u => (int?)u.DisplayOrder) ?? 0` di sibling yang sama + 1
- Create `new OrganizationUnit { Name = name.Trim(), ParentId = parentId, Level = level, DisplayOrder = maxOrder + 1, IsActive = true }`
- TempData["Success"] = "Unit berhasil ditambahkan."
- RedirectToAction("ManageOrganization")

**3. EditOrganizationUnit [HttpPost, Authorize(Roles = "Admin, HC"), ValidateAntiForgeryToken]**
- Params: `int id, string name, int? parentId`
- Find unit by id, return NotFound() if null
- Validasi nama kosong → error
- Cek duplikat nama (exclude self): `AnyAsync(u => u.Name == name.Trim() && u.Id != id)`
- Cek circular reference jika parentId berubah: `if (parentId.HasValue && parentId.Value == id)` → error self-reference. Juga panggil `IsDescendantAsync(id, parentId.Value)` → TempData["Error"] = "Tidak dapat memindahkan unit ke sub-unitnya sendiri (circular reference)."
- Jika parentId berubah: update Level = parent == null ? 0 : parent.Level + 1. Juga update children Level recursively via helper `UpdateChildrenLevelsAsync(unit)`
- Update Name, ParentId, Level. SaveChangesAsync.
- TempData["Success"] = "Unit berhasil diperbarui."

**4. Helper: IsDescendantAsync(int nodeId, int targetId) → bool**
- Private async method. Walk up dari targetId ke root. Jika menemui nodeId → return true.

**5. Helper: UpdateChildrenLevelsAsync(OrganizationUnit unit)**
- Private async method. Load children, set child.Level = unit.Level + 1, recurse.

**6. ToggleOrganizationUnitActive [HttpPost, Authorize(Roles = "Admin, HC"), ValidateAntiForgeryToken]**
- Param: `int id`
- Include Children. Block nonaktifkan jika `unit.IsActive && unit.Children.Any(c => c.IsActive)` → TempData["Error"] = "Nonaktifkan semua unit di bawahnya terlebih dahulu."
- Toggle `unit.IsActive = !unit.IsActive`. SaveChangesAsync.
- TempData["Success"] = `$"Status berhasil diubah menjadi {(unit.IsActive ? "Aktif" : "Nonaktif")}."`

**7. DeleteOrganizationUnit [HttpPost, Authorize(Roles = "Admin, HC"), ValidateAntiForgeryToken]**
- Param: `int id`
- Include Children, KkjFiles, CpdpFiles.
- Block: children aktif → "Hapus atau nonaktifkan unit di bawahnya terlebih dahulu."
- Block: KkjFiles.Any() || CpdpFiles.Any() → "Unit ini masih memiliki file KKJ/CPDP yang ter-assign. Hapus file terlebih dahulu."
- Block: `_context.Users.AnyAsync(u => u.Section == unit.Name || u.Unit == unit.Name)` → "Unit ini masih memiliki pekerja yang ter-assign. Pindahkan pekerja terlebih dahulu."
- Remove + SaveChangesAsync. TempData["Success"] = "Unit berhasil dihapus."

**8. ReorderOrganizationUnit [HttpPost, Authorize(Roles = "Admin, HC"), ValidateAntiForgeryToken]**
- Params: `int id, string direction`
- Find unit, get siblings ordered by DisplayOrder. Find index. Swap dengan sibling sebelum (up) atau sesudah (down). SaveChangesAsync.
- RedirectToAction("ManageOrganization")
