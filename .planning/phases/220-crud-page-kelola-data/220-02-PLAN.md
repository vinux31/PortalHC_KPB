**Tombol Aksi per baris (btn-group btn-group-sm):**

Setiap tombol wajib ada `aria-label` dan `title` sesuai UI-SPEC. Gunakan inline form POST untuk Reorder dan Toggle. Gunakan link GET untuk Edit. Gunakan `data-bs-toggle="modal"` untuk Hapus.

- **Reorder ↑**: form POST ke ReorderOrganizationUnit(id, direction="up"). Disabled jika posisi pertama di sibling. `btn btn-outline-secondary` + `bi-arrow-up`
- **Reorder ↓**: form POST ke ReorderOrganizationUnit(id, direction="down"). Disabled jika posisi terakhir. `btn btn-outline-secondary` + `bi-arrow-down`
- **Edit**: `<a>` link ke `ManageOrganization?editId=@unit.Id`. `btn btn-outline-primary` + `bi-pencil-square`. `aria-label="Edit @unit.Name"` `title="Edit"`
- **Toggle**: form POST ke ToggleOrganizationUnitActive(id). `btn btn-outline-warning` + `bi-toggle-on` jika aktif, `btn btn-outline-secondary` + `bi-toggle-off` jika nonaktif
- **Hapus**: `button` dengan `data-bs-toggle="modal" data-bs-target="#deleteModal" data-id="@unit.Id" data-name="@unit.Name"`. `btn btn-danger` + `bi-trash`. Disabled jika punya children aktif (cek `unit.Children.Any(c => c.IsActive)` di Razor)

Untuk menentukan posisi pertama/terakhir pada reorder: gunakan variabel loop. Untuk root nodes, cek `Model.IndexOf(unit)`. Untuk children, cek `unit.Children.OrderBy(c => c.DisplayOrder).ToList()` index.
