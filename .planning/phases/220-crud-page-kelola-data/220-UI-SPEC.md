## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body (sel tabel, form label) | 14px (Bootstrap default `small` / `0.875rem`) | 400 regular | 1.5 |
| Label (form label fw-bold, kolom header tabel) | 14px | 700 bold | 1.5 |
| Heading (judul halaman h2) | 24px (`h2` Bootstrap default) | 700 bold | 1.2 |
| Subtext (breadcrumb, `text-muted`, badge) | 12px (`small`) | 400 regular | 1.4 |

Catatan: Hanya 2 bobot digunakan — 400 regular dan 700 bold. Tidak ada 600 semibold.
Semua elemen yang sebelumnya menggunakan `fw-semibold` (kolom header tabel, card header) menggunakan `fw-bold`.
Tidak ada ukuran Display — halaman ini adalah admin CRUD, tidak butuh hero text.

Sumber: ManageCategories.cshtml pattern (`fw-bold`, `small`, `h2`, `text-muted`).

### 6. Tabel Indented Collapsible

**Struktur kolom (3 kolom + 1 aksi):**

| # | Kolom | Width | Align |
|---|-------|-------|-------|
| 1 | Nama (indented, dengan ikon collapse/expand untuk Bagian) | auto | left |
| 2 | Level | 100px | center |
| 3 | Status | 100px | center |
| 4 | Aksi | 180px | center |

**Row Bagian (Level 0):**
- Seluruh row `style="cursor:pointer;"` + `data-bs-toggle="collapse"` + `data-bs-target="#children-{id}"`
- Default `aria-expanded="false"` (D-02: semua collapsed saat pertama load)
- Kolom Nama: `bi-chevron-right me-2 collapse-icon` (rotate 90deg saat expanded via CSS) + nama bold
- Badge Level: `badge bg-primary` teks "Bagian"
- Badge Status: `badge bg-success` "Aktif" / `badge bg-secondary` "Nonaktif"
- Btn-group aksi: Edit + Toggle + Hapus (lihat spesifikasi tombol aksi)

**Row Unit (Level 1+) — dalam `tbody.collapse #children-{parent_id}`:**
- Kolom Nama: `ps-4` (Level 1) atau `ps-5` (Level 2) + `bi-arrow-return-right text-muted me-2 small` + nama
- Badge Level: `badge bg-info text-dark` teks "Unit"
- Badge Status: sama dengan Bagian
- Btn-group aksi: Reorder ↑↓ + Edit + Toggle + Hapus

**Thead:**
- `thead.table-light` — background `bg-light`
- Teks header: `fw-bold text-muted small text-uppercase`

**Empty state:**
- Satu `<tr>` penuh dengan `<td colspan="4" class="text-center py-5 text-muted">`
- Icon: `bi-diagram-3 d-block mb-2 fs-3`
- Teks: "Belum ada data struktur organisasi." + baris kedua `<small>Klik "Tambah Unit" untuk menambahkan.</small>`

### 7. Tombol Aksi Per Baris

Wrapper: `div.btn-group.btn-group-sm`

Semua tombol icon-only wajib menyertakan atribut `aria-label` dan `title` (tooltip). Tidak boleh ada tombol tanpa label aksesibilitas.

| Tombol | Kondisi | Kelas | Icon | aria-label | title (tooltip) |
|--------|---------|-------|------|------------|-----------------|
| Reorder ↑ | Bukan posisi pertama di sibling | `btn btn-outline-secondary` | `bi-arrow-up` | `"Naikan urutan {nama}"` | "Naikan urutan" |
| Reorder ↑ | Posisi pertama | `btn btn-outline-secondary` disabled | `bi-arrow-up` | `"Naikan urutan {nama} (tidak tersedia)"` | — |
| Reorder ↓ | Bukan posisi terakhir di sibling | `btn btn-outline-secondary` | `bi-arrow-down` | `"Turunkan urutan {nama}"` | "Turunkan urutan" |
| Reorder ↓ | Posisi terakhir | `btn btn-outline-secondary` disabled | `bi-arrow-down` | `"Turunkan urutan {nama} (tidak tersedia)"` | — |
| Edit | Selalu | `btn btn-outline-primary` | `bi-pencil-square` | `"Edit {nama}"` | "Edit" |
| Toggle | IsActive = true | `btn btn-outline-warning` | `bi-toggle-on` | `"Nonaktifkan {nama}"` | "Nonaktifkan" |
| Toggle | IsActive = false | `btn btn-outline-secondary` | `bi-toggle-off` | `"Aktifkan {nama}"` | "Aktifkan" |
| Hapus | Punya children aktif atau user/file ter-assign | `btn btn-danger` disabled | `bi-trash` | `"Hapus {nama} (tidak tersedia)"` | "Nonaktifkan atau pindahkan isi sebelum menghapus" |
| Hapus | Aman dihapus | `btn btn-danger` | `bi-trash` | `"Hapus {nama} secara permanen"` | "Hapus permanen" — trigger modal konfirmasi |

Catatan Reorder: Tombol Reorder hanya ditampilkan untuk Level 1 (Unit dalam Bagian). Level 0 (Bagian root) juga mendapat tombol reorder untuk mengatur urutan antar Bagian.

Pola `aria-label` menyertakan nama unit agar screen reader dapat membedakan tombol antar baris. Contoh: `aria-label="Edit Alkylation"`, bukan hanya `aria-label="Edit"`.

## Interaction Contract

### Expand/Collapse Tree
- Bagian (Level 0) row bertindak sebagai toggle — klik di mana saja pada row
- Bootstrap collapse digunakan pada `<tbody>` child. Jika bootstrap collapse tidak support `<tbody>` secara native (Open Question RESEARCH.md), gunakan CSS `display:none` + JavaScript toggle ringan sebagai fallback
- Ikon `bi-chevron-right` di kolom Nama: rotate 90deg via CSS transition saat `aria-expanded="true"`
  ```css
  .collapse-icon { transition: transform 0.2s ease; }
  tr[aria-expanded="true"] .collapse-icon { transform: rotate(90deg); }
  ```
- Default collapsed: tidak ada `show` class pada `tbody.collapse` saat render pertama (D-02)

### Form Tambah Toggle
- Tombol "Tambah Unit" header: `data-bs-toggle="collapse" data-bs-target="#addUnitForm"`
- Setelah POST sukses: redirect ke ManageOrganization — form kembali collapsed
- Jika validasi gagal: `TempData["ShowAddForm"]` = true → form render dengan class `show`

### Form Edit
- Tombol Edit per baris: link ke `/Admin/ManageOrganization?editId={id}` (GET dengan query param) ATAU redirect POST pattern dari ManageCategories — ikuti pattern yang sama (ViewBag.EditUnit)
- Form edit muncul di atas tabel (bukan modal) — konsisten dengan D-07
- Setelah simpan sukses: redirect ke ManageOrganization bersih (form hilang)
- Tombol "Batal": link biasa ke ManageOrganization tanpa param

### Delete Modal
```javascript
// Set data di modal saat trigger diklik
document.getElementById('deleteModal').addEventListener('show.bs.modal', function (event) {
    var button = event.relatedTarget;
    var id = button.getAttribute('data-id');
    var name = button.getAttribute('data-name');
    document.getElementById('deleteModalId').value = id;
    document.getElementById('deleteModalName').textContent = name;
});
```

### Reorder
- POST form per tombol, redirect setelah swap DisplayOrder
- Tidak ada feedback visual khusus selain urutan baris yang berubah setelah reload
