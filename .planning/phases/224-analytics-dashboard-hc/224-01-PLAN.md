**A. Di CMPController.cs, tambahkan 4 action methods:**

1. **AnalyticsDashboard()** â€” GET, `[Authorize(Roles = "Admin, HC")]`:
   ```csharp
   var vm = new AnalyticsDashboardViewModel
   {
       Sections = await _context.GetAllSectionsAsync(),
       Categories = await _context.AssessmentCategories
           .Where(c => c.ParentId == null && c.IsActive)
           .OrderBy(c => c.SortOrder)
           .Select(c => c.Name)
           .ToListAsync()
   };
   return View(vm);
   ```

2. **GetAnalyticsData(string? bagian, string? unit, string? kategori, string? subKategori, DateTime? periodeStart, DateTime? periodeEnd)** â€” GET, `[Authorize(Roles = "Admin, HC")]`, returns `JsonResult`:
   - Default periodeEnd = DateTime.Today, periodeStart = periodeEnd.AddYears(-1)
   - Base query: `_context.AssessmentSessions.Include(s => s.User).Where(s => s.IsPassed.HasValue && s.CompletedAt.HasValue && s.CompletedAt >= periodeStart && s.CompletedAt <= periodeEnd)`
   - Apply filters: bagian â†’ `s.User!.Section == bagian`, unit â†’ `s.User!.Unit == unit`, kategori â†’ `s.Category == kategori` (string match, bukan FK â€” per Pitfall 2), subKategori â†’ skip (AssessmentSession has no SubCategory field)
   - **failRate query**: GroupBy Section + Category â†’ select Total, Failed, FailRatePercent. Handle null Section â†’ "Tidak Diketahui"
   - **trend query**: GroupBy Year + Month dari CompletedAt â†’ Passed/Failed count, OrderBy Year ThenBy Month
   - **etBreakdown query**: `_context.SessionElemenTeknisScores.Include(e => e.AssessmentSession).ThenInclude(s => s.User)` dengan filter yang sama (periodeStart, periodeEnd, bagian via User.Section, unit via User.Unit, kategori via AssessmentSession.Category). GroupBy ElemenTeknis + Category â†’ Avg/Min/Max of (CorrectCount*100.0/QuestionCount where QuestionCount > 0)
   - **expiringSoon query** (filter Bagian/Unit saja, BUKAN kategori â€” per Research Open Question 2):
     - Source 1: TrainingRecords WHERE Status == "Valid" AND ValidUntil >= today AND ValidUntil <= today+30, filter Bagian/Unit via User
     - Source 2: AssessmentSessions WHERE IsPassed == true AND GenerateCertificate AND ValidUntil >= today AND ValidUntil <= today+30, filter Bagian/Unit via User
     - Concat, OrderBy TanggalExpired
   - Return `Json(new AnalyticsDataResult { FailRate = failRate, Trend = trend, EtBreakdown = etBreakdown, ExpiringSoon = expiringSoon })`

3. **GetAnalyticsCascadeUnits(string bagian)** â€” GET, `[Authorize(Roles = "Admin, HC")]`:
