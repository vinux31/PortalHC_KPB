**Scripts section** (`@section Scripts {}`):
- `<script src="https://cdn.jsdelivr.net/npm/chart.js@4/dist/chart.umd.min.js"></script>`
- JavaScript block with:

  **Variables:** `var failRateChart = null; var trendChart = null; var analyticsAbort = null;`

  **refreshAnalytics() function:**
  - Get filter values from all 6 inputs
  - Build URLSearchParams
  - Add `.dashboard-loading` to `#analytics-panels-container`, set `aria-busy="true"`
  - If analyticsAbort exists, abort it
  - Create new AbortController
  - `fetch('/CMP/GetAnalyticsData?' + params.toString(), { signal: analyticsAbort.signal })`
  - On success `.then(r => r.json()).then(data => { renderFailRate(data.failRate); renderTrend(data.trend); renderEtTable(data.etBreakdown); renderExpiringTable(data.expiringSoon); })` then remove `.dashboard-loading`
  - On error: if not AbortError, show error message, remove loading

  **renderFailRate(data) function:**
  - If data.length === 0: hide canvas, show empty state div, return
  - Show canvas, hide empty state
  - `if (failRateChart) failRateChart.destroy();`
  - Labels: `data.map(d => d.section + ' - ' + d.category)`
  - Data: `data.map(d => d.failRatePercent)`
  - Create bar chart: backgroundColor `rgba(220,53,69,0.75)`, scales.y beginAtZero + max 100, responsive true, plugins.legend display false

  **renderTrend(data) function:**
  - If data.length === 0: hide canvas, show empty, return
  - `if (trendChart) trendChart.destroy();`
  - Labels: `data.map(d => d.label)`
  - 2 datasets: "Pass" (`data.map(d => d.passed)`, borderColor `#198754`, fill false) + "Fail" (`data.map(d => d.failed)`, borderColor `#dc3545`, fill false)
  - Create line chart, responsive true, tension 0.3

  **renderEtTable(data) function:**
  - If data.length === 0: show empty, return
  - Build HTML table: `<table class="table table-sm table-hover"><thead><tr><th scope="col">Elemen Teknis</th><th scope="col">Kategori</th><th scope="col">Rata-rata</th><th scope="col">Min</th><th scope="col">Max</th><th scope="col">Sesi</th></tr></thead><tbody>`
  - For each row: td with heatmap class on avgPct cell — if avgPct >= 80 class `heatmap-high`, >= 60 class `heatmap-mid`, else `heatmap-low`. Display `avgPct.toFixed(1) + '%'`
  - Set innerHTML of `#etTableContainer`

  **renderExpiringTable(data) function:**
  - If data.length === 0: show empty, return
  - Build table: `<table class="table table-sm table-hover"><thead><tr><th scope="col">Nama Pekerja</th><th scope="col">Sertifikat</th><th scope="col">Tgl Expired</th><th scope="col">Bagian / Unit</th></tr></thead><tbody>`
  - Format tanggalExpired: `new Date(d.tanggalExpired).toLocaleDateString('id-ID')`
  - Set innerHTML of `#expiringTableContainer`

  **Cascade dropdown handlers:**
  - `filterBagian` change: if value empty → disable filterUnit, reset to "Semua Unit". Else → fetch `/CMP/GetAnalyticsCascadeUnits?bagian={val}`, populate options, enable filterUnit.
  - `filterKategori` change: if value empty → disable filterSubKategori, reset. Else → fetch `/CMP/GetAnalyticsCascadeSubKategori?kategori={val}`, populate, enable.

  **Button handlers:**
  - `btnApply` click → `refreshAnalytics()`
  - `btnReset` click → reset all selects to "", disable Unit/SubKategori, set date defaults, `refreshAnalytics()`

  **DOMContentLoaded:**
  - Set `filterStart.value` to 1 year ago (ISO format `YYYY-MM-DD`)
  - Set `filterEnd.value` to today
  - Call `refreshAnalytics()`

All canvas elements: add `role="img"` and `aria-label` (e.g., `aria-label="Grafik fail rate assessment"`)
All labels connected to selects via `for` attribute.
