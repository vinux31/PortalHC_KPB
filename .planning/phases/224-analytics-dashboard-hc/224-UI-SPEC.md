## Spacing Scale

Mengikuti Bootstrap 5 spacing utilities (base 4px = 0.25rem, 1rem = 16px):

| Token | Value | Bootstrap Class | Usage |
|-------|-------|-----------------|-------|
| xs | 4px | `gap-1`, `p-1` | Icon gaps, inline padding |
| sm | 8px | `gap-2`, `p-2` | Compact element spacing, filter label gap |
| md | 16px | `p-3`, `mb-3` | Default element spacing, card body padding |
| lg | 24px | `p-4`, `mb-4` | Section padding, card gap (`g-4`) |
| xl | 32px | `py-5` (approx) | Major section breaks |
| 2xl | 48px | `py-4 + mb-4` | Page-level top padding |

Exceptions:
- Filter bar: `py-3 px-4` (24px vertikal, 16px horizontal) untuk konsistensi dengan CDP Dashboard
- Chart canvas: min-height 280px agar chart tidak terlalu kecil di grid 2×2
- Tabel sertifikat: `table-sm` class Bootstrap (vertikal padding 4px per row) karena data bisa banyak baris

## Typography

Menggunakan Bootstrap 5 default type scale (font-size base 16px = 1rem):

| Role | Size | Weight | Line Height | Bootstrap Class | Usage |
|------|------|--------|-------------|-----------------|-------|
| Body | 14px (0.875rem) | 400 (regular) | 1.5 | `fs-6` / default | Label filter, cell tabel, keterangan chart |
| Label | 14px (0.875rem) | 600 (semibold) | 1.4 | `fw-semibold` | Header kolom tabel, badge, filter label |
| Heading | 18px (1.125rem) | 600 (semibold) | 1.2 | `h5`, `card-title` | Judul tiap panel chart/card |
| Display | 20px (1.25rem) | 700 (bold) | 1.2 | `h4` / `fw-bold` | Judul halaman "Analytics Dashboard" |

Catatan: 2 weight saja — regular (400) untuk body/data, semibold/bold (600–700) untuk heading dan label penting.

## Color

Project menggunakan Bootstrap 5 semantic colors. Tidak ada custom hex di luar Bootstrap palette.

| Role | Value | Bootstrap Token | Usage |
|------|-------|-----------------|-------|
| Dominant (60%) | `#ffffff` / `#f8f9fa` | `bg-white`, `bg-light` | Background halaman, card surface |
| Secondary (30%) | `#e9ecef` | `bg-light`, border `#dee2e6` | Filter bar background, tabel header, card border |
| Accent (10%) | `#0d6efd` | `text-primary`, `btn-primary` | Tombol "Terapkan Filter", icon box di card CMP Hub, border active filter |
| Destructive | `#dc3545` | `text-danger`, `badge bg-danger` | Badge "Expired < 7 hari" di tabel sertifikat, bar chart fail rate |

Accent reserved for:
1. Tombol "Terapkan Filter" (satu-satunya primary CTA halaman)
2. Icon box card Analytics di CMP Hub (`bg-primary bg-opacity-10 text-primary`)
3. Spinner loading (border-top-color: `#0d6efd`) — sama dengan CDP Dashboard pattern

Warna semantic tambahan untuk heatmap ET (ANLT-03, sesuai D-05):
- Hijau `rgba(40,167,69,0.20)` — skor ≥ 80%
- Kuning `rgba(255,193,7,0.20)` — skor 60–79%
- Merah `rgba(220,53,69,0.20)` — skor < 60%

Warna Chart.js:
- Bar chart fail rate: `rgba(220,53,69,0.75)` (merah) untuk fail, `rgba(25,135,84,0.75)` (hijau) untuk pass
- Line chart trend: dataset "Pass" = `#198754` (success green), dataset "Fail" = `#dc3545` (danger red)

## Interaction Contract

### Filter Cascade (D-08)

1. Saat Bagian dipilih → fetch `/CMP/GetAnalyticsCascadeUnits?bagian={val}` → populate Unit dropdown
2. Saat Bagian di-reset → Unit dropdown reset ke "Semua Unit", disable Unit
3. Saat Kategori dipilih → fetch `/CMP/GetAnalyticsCascadeSubKategori?kategori={val}` → populate SubKategori dropdown
4. Saat Kategori di-reset → SubKategori reset ke "Semua SubKategori", disable SubKategori

### AJAX Refresh (D-09)

- Trigger: klik "Terapkan Filter"
- Juga trigger: DOMContentLoaded (auto-load dengan default values D-10)
- Pattern: AbortController fetch ke `/CMP/GetAnalyticsData?{params}` — persis CDP Dashboard
- Loading state: tambah class `.dashboard-loading` ke `#analytics-panels-container` selama fetch
- Success: render chart + tabel dari JSON response, hapus `.dashboard-loading`
- Error: tampilkan error state (lihat Copywriting), hapus `.dashboard-loading`

### Chart Lifecycle

- Setiap AJAX success: destroy chart lama (`if (chart) chart.destroy()`) → init Chart.js baru
- Canvas `<canvas>` **tidak** di-replace via innerHTML — canvas bersifat persisten di DOM
- Container tabel (ANLT-03, ANLT-04): innerHTML replace aman karena tidak pakai canvas

### Default State (D-10)

- Semua dropdown filter: value kosong = "Semua ..."
- Periode Start: `today - 1 tahun` (di-set via JavaScript `new Date()` saat DOMContentLoaded)
- Periode End: `today`

## Accessibility Notes

- Semua `<canvas>` Chart.js harus memiliki `role="img"` dan `aria-label` yang deskriptif
- Tabel wajib memiliki `<thead>` dengan `scope="col"` di tiap `<th>`
- Filter dropdown wajib memiliki `<label>` eksplisit yang terhubung via `for` attribute
- Loading state `.dashboard-loading` harus disertai `aria-busy="true"` di container
- Warna heatmap tidak boleh satu-satunya indikator — tambahkan nilai % di dalam cell tabel ET
