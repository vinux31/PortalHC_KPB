4. **Section Moodle Quiz (per D-01, D-02, D-10):**

5. **Section Google Forms Quiz (per D-10):**
   - Model: Form-based quiz, grading otomatis MC
   - Admin UX Flow: Buat form → Settings → "Make this a quiz" → Add questions → set answer key → assign points → Share via link → Responses in Sheets
   - UI Description: clean material design, drag-drop questions, inline answer key editor
   - Patterns: Simplicity (< 5 menit buat basic quiz), auto-grading MC, response summary charts
   - Keterbatasan: tidak ada timer native, tidak ada randomisasi native, tidak ada per-participant monitoring, security lemah, tidak ada attempt management, tidak ada sertifikat otomatis

4. **Section SpeedExam:**
   - Monitoring dashboard: Live candidates (status Taking/Completed/Dropped), device info (browser, OS, IP, lokasi), attempt count, bulk actions (force-close, extend time, reset)
   - UI Description: list view, color coding: hijau (aktif), abu (belum mulai), biru (selesai), merah (dropped). Filter by status. Klik nama → progress detail (soal ke berapa, waktu tersisa)

   - Monitoring focus: simple dashboard (student name + status badge), hand-raise feature digital, invigilator chat per student, timer override per student
   - UI Description: minimal clean interface, real-time status badges, chat sidebar
   - Patterns: hand-raise (student minta bantuan), per-student chat, individual timer extension

6. **Section Online Proctoring Patterns 2024-2025 (per D-03):**
   - Progressive Disclosure: info kritis di depan, detail di expand
   - No-Navigation principle: admin action tanpa navigasi jauh dari monitoring view
   - Color urgency system: visual status tanpa baca teks (green/yellow/red)
   - Alert/flag system: abnormal behavior (tab switch, idle lama) langsung flagged
   - Bulk actions: close multiple sessions sekaligus
