   - Di controller AssessmentMonitoringDetail, pass `StartedAt` dan `DurationMinutes` ke view (jika belum ada di model, gunakan ViewBag atau tambahkan ke existing per-participant model)
   - Di view, untuk setiap `<tr>` peserta InProgress, tambahkan `data-started-at="@participant.StartedAt?.ToString("o")"` dan `data-duration="@Model.DurationMinutes"`
   - Pada initial render, ganti hardcoded "—" di `.timeremaining-cell` dengan computed value: `timeLeft = startedAt + durationMs - now`. Jika timeLeft <= 0, tampilkan "Waktu Habis". Jika status bukan InProgress, tetap "—".

4. Tambahkan JS countdown timer yang update setiap detik:
   ```javascript
   function updateTimeRemaining() {
     document.querySelectorAll('tr[data-started-at]').forEach(row => {
       const startedAt = new Date(row.dataset.startedAt);
       const durationMs = parseInt(row.dataset.duration) * 60000;
       const timeLeft = (startedAt.getTime() + durationMs) - Date.now();
       const cell = row.querySelector('.timeremaining-cell');
       if (timeLeft <= 0) {
         cell.textContent = 'Waktu Habis';
         cell.classList.replace('text-muted', 'text-danger');
       } else {
         const mins = Math.floor(timeLeft / 60000);
         const secs = Math.floor((timeLeft % 60000) / 1000);
         cell.textContent = `${mins}m ${secs}s`;
         cell.classList.replace('text-muted', 'text-warning');
       }
     });
   }
   setInterval(updateTimeRemaining, 1000);
   updateTimeRemaining(); // initial call
   ```
