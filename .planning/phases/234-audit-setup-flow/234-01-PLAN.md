public async Task<IActionResult> SilabusDeletePreview(int deliverableId)

**A. Fix urutan operasi GuidanceReplace (sekitar L982-1005):**

Saat ini urutan SALAH:
1. L983: `System.IO.File.Delete(oldFilePath)` — hapus dulu
2. L994: `file.CopyToAsync(stream)` — baru upload

Ubah menjadi (per D-07):
```csharp
// 1. Upload file baru DULU
var safeFileName = $"{DateTime.Now:yyyyMMddHHmmss}_{Guid.NewGuid()}{Path.GetExtension(file.FileName)}";
var newFilePath = Path.Combine(uploadDir, safeFileName);
using (var stream = new FileStream(newFilePath, FileMode.Create))
{
    await file.CopyToAsync(stream);
}

// 2. Capture path file lama SEBELUM update DB
var oldFilePath = Path.Combine(uploadDir, Path.GetFileName(guidance.FilePath));

// 3. Update DB record
guidance.FilePath = $"/uploads/guidance/{safeFileName}";
guidance.FileName = file.FileName;
await _context.SaveChangesAsync();

// 4. BARU hapus file lama (non-critical — wrapped in try-catch)
try
{
    if (System.IO.File.Exists(oldFilePath))
        System.IO.File.Delete(oldFilePath);
}
catch (Exception ex)
{
    _logger.LogWarning(ex, "Failed to delete old guidance file: {Path}", oldFilePath);
}
```
