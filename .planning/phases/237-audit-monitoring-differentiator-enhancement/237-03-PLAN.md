Implement batch HC approval per D-04, D-05, D-06, D-07:

**A. DTO class** — Tambah di CDPController.cs (atau area yang sesuai):
```csharp
public class BatchHCApproveRequest
{
    public List<int> ProgressIds { get; set; } = new();
}
```

**B. Endpoint** — Tambah di CDPController:
```csharp
[HttpPost]
[ValidateAntiForgeryToken]
[Authorize(Roles = "HC, Admin")]
public async Task<IActionResult> BatchHCApprove([FromBody] BatchHCApproveRequest req)
```
- Validate: `req.ProgressIds` tidak null/empty
- Query: `ProtonDeliverableProgresses` WHERE `req.ProgressIds.Contains(p.Id)` AND `p.Status == "Approved"` AND `p.HCApprovalStatus == "Pending"` (per D-05, hanya "Pending HC Review")
- Per Pitfall 2 (race condition): hanya proses item yang masih Pending HC — skip yang sudah berubah
- Update setiap progress: `HCApprovalStatus = "Reviewed"`, `HCReviewedById = user.Id`, `HCReviewedAt = DateTime.UtcNow`
- `SaveChangesAsync()`
- Audit log: `_auditLog.LogAsync(user.Id, user.FullName ?? user.Id, "BatchHCApprove", $"Batch HC Approve {progresses.Count} deliverable: IDs {string.Join(",", progresses.Select(p => p.Id))}", targetType: "ProtonDeliverableProgress")`
- Return JSON: `{ success: true, approvedCount: N, skippedCount: M }` dimana M = items yang sudah tidak Pending
