Fix semua hardcoded URL di 6 file ini. Rules per file:

**A. JavaScript URL (fetch, $.get, $.ajax, window.location.href, postWithToken):**
- `fetch('/Controller/Action')` -> `fetch(appUrl('/Controller/Action'))`
- `window.location.href = '/Controller/Action'` -> `window.location.href = appUrl('/Controller/Action')`
- `$.get('/Controller/Action')` -> `$.get(appUrl('/Controller/Action'))`
- `postWithToken('/Controller/Action')` -> `postWithToken(appUrl('/Controller/Action'))`

**B. HTML-in-JS strings (form action, href dalam template literal/string concat):**
- `action="/Admin/AkhiriUjian"` dalam JS string -> `action="' + basePath + '/Admin/AkhiriUjian"'`
- Atau gunakan appUrl jika memungkinkan

**C. HTML markup (href, src di Razor HTML):**
- `href="/Admin/CoachCoacheeMappingExport"` -> `asp-controller="Admin" asp-action="CoachCoacheeMappingExport"`
- `href="/Controller/Action?param=val"` -> gunakan `asp-controller` + `asp-action` + `asp-route-param`, ATAU `href="@Url.Action("Action", "Controller", new { param = val })"`

**Per-file specifics:**

1. **Views/ProtonData/Index.cshtml** (11 URL): Mostly fetch/$.get calls ke ProtonData controller. Ganti semua dengan appUrl().
2. **Views/Admin/CoachCoacheeMapping.cshtml** (10 URL): fetch calls + 1 HTML href. Fix fetch dengan appUrl(), href dengan tag helper.
3. **Views/Admin/RenewalCertificate.cshtml** (5+4 URL): fetch + window.location.href. Semua appUrl().
4. **Views/CDP/CertificationManagement.cshtml** (5+1 URL): fetch + window.location.href. Semua appUrl().
5. **Views/Shared/Components/NotificationBell/Default.cshtml** (4 URL): fetch/postWithToken calls. KRITIS: cek juga `n.actionUrl` — jika ini dari server response, prefix dengan basePath: `window.location.href = basePath + n.actionUrl` (karena server stores path tanpa PathBase prefix di database).
6. **Views/Admin/AssessmentMonitoringDetail.cshtml** (5 URL): fetch + JS-built HTML forms. Untuk form action dalam JS string, gunakan `basePath + '/Admin/...'`.

PENTING: Cari SEMUA pattern `'/`, `"/`, `fetch(`, `$.get(`, `$.ajax(`, `window.location`, `postWithToken(` di setiap file. Jangan sampai ada yang terlewat.
