**4. Tambahkan JavaScript overlay logic di `@section Scripts` — SEBELUM existing hub badge code (sebelum line 918 `// -------- Connection status badge --------`):**

```javascript
// -------- Loading Overlay Logic (D-01 through D-07) --------
var examOverlay = document.getElementById('examLoadingOverlay');
var overlayStatusEl = document.getElementById('overlayStatus');
var examContainer = document.getElementById('examContainer');
var hubConnectedForOverlay = false;

function overlayFadeOut() {
    if (overlayStatusEl) overlayStatusEl.textContent = 'Terhubung!';
    setTimeout(function() {
        if (examOverlay) {
            examOverlay.classList.add('fade-out');
            if (examContainer) examContainer.removeAttribute('inert');
            setTimeout(function() {
                examOverlay.style.display = 'none';
            }, 300);
        }
    }, 400);
}

function overlayShowError() {
    var spinner = examOverlay ? examOverlay.querySelector('.spinner-border') : null;
    if (spinner) spinner.style.display = 'none';
    if (overlayStatusEl) overlayStatusEl.textContent = 'Koneksi gagal. Periksa jaringan Anda.';
    var reloadBtn = document.getElementById('overlayReloadBtn');
    if (reloadBtn) reloadBtn.style.display = 'inline-block';
}

// D-04: minimal 1 detik + hub ready
var minDelay = new Promise(function(resolve) { setTimeout(resolve, 1000); });

if (window.assessmentHubStartPromise) {
    // assessmentHubStartPromise SELALU resolve (tidak pernah reject — lihat RESEARCH.md)
    // Jadi gunakan .then() untuk cek state, dan onclose untuk error
    Promise.all([window.assessmentHubStartPromise, minDelay])
        .then(function() {
            if (window.assessmentHub && window.assessmentHub.state === 'Connected') {
                hubConnectedForOverlay = true;
                overlayFadeOut();
            }
            // Jika state bukan Connected, onclose handler di bawah akan trigger error
        });

    // D-06: error state via onclose (karena promise tidak reject)
    if (window.assessmentHub) {
        var originalOnclose = window.assessmentHub._closedCallbacks || [];
        window.assessmentHub.onclose(function() {
            if (!hubConnectedForOverlay) {
                overlayShowError();
            }
        });
    }
} else {
    // Fallback: tidak ada promise (seharusnya tidak terjadi), hilangkan overlay setelah delay
    minDelay.then(function() {
        overlayFadeOut();
    });
}
```

PENTING — onclose handler di atas TIDAK menggantikan existing onclose handler di line 926-928. Method `onclose()` di SignalR MENAMBAHKAN callback, bukan menggantikan. Jadi kedua handler (overlay + badge) akan terpanggil. Tidak perlu modifikasi existing onclose handler.

Per D-05: Timer logic (di bagian lain file) TIDAK dimodifikasi — timer tetap berjalan karena overlay hanya visual block, tidak mempengaruhi JavaScript execution.

Per D-07: Tidak perlu kode khusus untuk resume exam — setiap page load adalah fresh, assessmentHubStartPromise dibuat baru oleh assessment-hub.js.
