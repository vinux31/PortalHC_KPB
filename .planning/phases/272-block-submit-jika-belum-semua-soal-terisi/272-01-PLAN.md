2. Setelah blok "Prevent re-submission" (setelah line 1367) dan sebelum timer enforcement (line 1369), tambah validasi completeness:
```csharp
// ---- Block incomplete submission (Phase 272) ----
if (!isAutoSubmit)
{
    // Determine total questions for this assessment
    int totalQuestions = 0;
    var pkgAssign = await _context.UserPackageAssignments
        .FirstOrDefaultAsync(a => a.AssessmentSessionId == id);
    if (pkgAssign != null)
    {
        totalQuestions = pkgAssign.GetShuffledQuestionIds().Count;
    }

    // Count answered: valid answers have optionId > 0
    int answeredCount = answers.Count(a => a.Value > 0);

    if (totalQuestions > 0 && answeredCount < totalQuestions)
    {
        int unanswered = totalQuestions - answeredCount;
        TempData["Error"] = $"Masih ada {unanswered} soal yang belum dijawab. Jawab semua soal terlebih dahulu.";
        return RedirectToAction("ExamSummary", new { id });
    }
}
```

CATATAN: Jika timer sudah habis (timerExpired server-side), auto-submit dari JavaScript akan mengirim isAutoSubmit=true, sehingga validasi di-skip. Untuk manual submit dari browser, isAutoSubmit default false sehingga validasi aktif.
