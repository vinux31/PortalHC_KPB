# Plan 275-01: Warning Pre-Test Certificate

## Objective
Tambahkan warning text dinamis di form CreateAssessment: jika judul mengandung "pre test"/"pretest" (case-insensitive) dan checkbox "Terbitkan Sertifikat" dicentang, tampilkan alert warning.

## Tasks

### Task 1: Tambah warning div dan JavaScript listener

<read_first>
- Views/Admin/CreateAssessment.cshtml (current state — lihat struktur checkbox GenerateCertificate di ~line 431-439 dan field Title di ~line 185)
</read_first>

<action>
1. Di bawah div `.form-text.text-muted` "Peserta yang lulus dapat melihat dan mencetak sertifikat" (line 438), tambahkan warning div yang hidden by default:

```html
<div id="preTestWarning" class="alert alert-warning py-2 px-3 mt-2 d-none" role="alert">
    <i class="bi bi-exclamation-triangle-fill me-1"></i>
    Judul mengandung "Pre Test". Pre Test biasanya tidak menerbitkan sertifikat.
</div>
```

2. Di bagian JavaScript (section scripts di bawah file), tambahkan fungsi untuk mendeteksi pre-test dan toggle warning:

```javascript
function checkPreTestWarning() {
    var title = document.getElementById('Title').value.toLowerCase();
    var certChecked = document.getElementById('GenerateCertificate').checked;
    var warning = document.getElementById('preTestWarning');
    var isPreTest = title.includes('pre test') || title.includes('pretest') || title.includes('pre-test');

    if (isPreTest && certChecked) {
        warning.classList.remove('d-none');
    } else {
        warning.classList.add('d-none');
    }
}

document.getElementById('Title').addEventListener('input', checkPreTestWarning);
document.getElementById('GenerateCertificate').addEventListener('change', checkPreTestWarning);
// Run on page load (for edit/renewal scenarios)
checkPreTestWarning();
```
</action>

<acceptance_criteria>
- `CreateAssessment.cshtml` contains `id="preTestWarning"`
- `CreateAssessment.cshtml` contains `pre test` detection logic with `.includes('pre test')`
- `CreateAssessment.cshtml` contains `addEventListener('input', checkPreTestWarning)`
- `CreateAssessment.cshtml` contains `addEventListener('change', checkPreTestWarning)`
- Warning div has class `d-none` by default (hidden)
- Warning div has class `alert-warning`
</acceptance_criteria>
