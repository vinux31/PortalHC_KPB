## Navigation Panel Specifications

### Layout Structure
```
.questionPanelWrapper (col-lg-3, sticky-top, top: 70px)
└── .card
    ├── .card-header → "Daftar Soal"
    └── .card-body
        └── #panelNumbers (CSS Grid container)
            └── .question-badge[] (badge items)
```

### CSS Grid Specification
```css
#panelNumbers {
    display: grid;
    grid-template-columns: repeat(10, 1fr); /* 10 kolom sama lebar */
    gap: 0.5rem; /* 8px spacing antar badges */
}

.question-badge {
    font-size: 0.85rem;
    padding: 6px 10px;
    cursor: pointer;
    transition: transform 0.1s ease;
    text-align: center;
    user-select: none;
}

.question-badge:hover {
    transform: scale(1.1); /* Sedikit zoom saat hover */
}

.question-badge:active {
    transform: scale(0.95); /* Visual feedback saat click */
}
```

### Mobile Responsive Behavior
```css
@media (max-width: 991px) {
    #questionPanelWrapper {
        display: none; /* Default hidden di mobile */
    }
    #questionPanelWrapper.visible {
        display: block; /* Show saat toggle active */
        order: -1; /* Pindahkan ke atas content */
        margin-bottom: 1rem;
    }
    #questionPanel {
        max-height: 200px;
        overflow-y: auto;
    }
}
```

### Click Behavior (JavaScript)
- Klik badge nomor soal → trigger `jumpToQuestion(qId, targetPage)`
- Jika targetPage != currentPage: switch page dulu, lalu scroll
- Jika targetPage == currentPage: scroll ke soal langsung
- Scroll behavior: instant (`behavior: 'auto'`, tanpa animasi)

### Auto-Scroll Panel
- Setiap page change, panel otomatis scroll untuk menampilkan badge pertama di halaman tersebut
- Ini membantu user tetap oriented terutama untuk assessment dengan banyak soal
