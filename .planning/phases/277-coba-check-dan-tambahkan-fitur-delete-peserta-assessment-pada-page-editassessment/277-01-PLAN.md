Implementasi:
1. Get logger via `HttpContext.RequestServices.GetRequiredService<ILogger<AdminController>>()`
2. Wrap seluruh logic dalam try/catch (pattern dari DeleteAssessment)
3. Load session: `_context.AssessmentSessions.Include(a => a.User).FirstOrDefaultAsync(a => a.Id == sessionId)`
4. Null check -> TempData["Error"] = "Session peserta tidak ditemukan." + redirect ke ManageAssessment
5. Simpan `title`, `category`, `scheduleDate`, `userName` sebelum delete untuk redirect logic
6. **Guard (per D-01):** Jika `session.StartedAt != null || session.CompletedAt != null || session.Status == "Completed"` -> TempData["Error"] = "Tidak dapat menghapus peserta: ujian sudah dimulai atau selesai." + redirect ke EditAssessment(returnToId)
7. **Cascade delete (per D-02, urutan SAMA PERSIS dari DeleteAssessment lines 2086-2127):**
   a. PackageUserResponses WHERE AssessmentSessionId == sessionId -> RemoveRange
   b. AssessmentAttemptHistory WHERE SessionId == sessionId -> RemoveRange
   c. AssessmentPackages Include Questions.ThenInclude Options WHERE AssessmentSessionId == sessionId -> foreach RemoveRange Options, Questions, then RemoveRange packages
   d. `_context.AssessmentSessions.Remove(session)`
   e. `await _context.SaveChangesAsync()`
8. **Audit log (try/catch terpisah, pattern dari DeleteAssessment lines 2132-2147):**
   - Action: "DeleteAssessmentPeserta"
   - Detail: `$"Deleted participant '{userName}' from assessment '{title}' [SessionID={sessionId}]"`
   - EntityId: sessionId, EntityType: "AssessmentSession"
9. **Redirect logic (per D-05, D-06, D-07):**
   - Query remaining siblings SETELAH SaveChangesAsync: `_context.AssessmentSessions.Where(a => a.Title == title && a.Category == category && a.Schedule.Date == scheduleDate).ToListAsync()`
   - Jika `!remainingSiblings.Any()` -> TempData["Success"] = `$"Peserta '{userName}' telah dihapus. Tidak ada peserta tersisa."` + RedirectToAction("ManageAssessment") **(D-07)**
   - Jika `sessionId != returnToId` -> TempData["Success"] = `$"Peserta '{userName}' berhasil dihapus."` + RedirectToAction("EditAssessment", new { id = returnToId }) **(D-05)**
   - Else (sessionId == returnToId) -> `var sibling = remainingSiblings.First()` + TempData["Success"] = `$"Peserta '{userName}' berhasil dihapus."` + RedirectToAction("EditAssessment", new { id = sibling.Id }) **(D-06)**
10. Catch block: logger.LogError + TempData["Error"] = "Gagal menghapus peserta. Silakan coba lagi." + RedirectToAction("ManageAssessment")
