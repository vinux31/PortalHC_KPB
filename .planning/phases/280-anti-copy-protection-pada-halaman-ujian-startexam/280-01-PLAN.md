**3. Tambahkan JS anti-copy IIFE di akhir script section (per D-05, D-06)**

Tambahkan sebelum closing `</script>` atau setelah existing code di @section Scripts:
```javascript
// Anti-copy protection (Phase 280)
(function() {
    var area = document.querySelector('.exam-protected');
    if (!area) return;

    // D-05: Block clipboard & context menu events pada area soal (silent, per D-01)
    ['copy','cut','paste','contextmenu','selectstart','dragstart'].forEach(function(evt) {
        area.addEventListener(evt, function(e) { e.preventDefault(); });
    });

    // D-06: Block keyboard shortcuts (Ctrl+C/A/U/S/P) secara global (per D-03)
    document.addEventListener('keydown', function(e) {
        if (e.ctrlKey || e.metaKey) {
            var blocked = [67, 65, 85, 83, 80]; // C, A, U, S, P
            if (blocked.indexOf(e.keyCode) !== -1) {
                e.preventDefault();
            }
        }
    });
})();
```

JANGAN:
- Block bare keys (Tab, Space, Arrow) — radio button keyboard navigation harus tetap berfungsi
- Tambahkan alert/popup saat copy attempt — D-01 sudah putuskan silent block
- Proteksi seluruh halaman — D-02 hanya area soal
