**A. AdminController — tambah 2 actions:**

1. `Maintenance` GET action dengan `[Authorize(Roles = "Admin, HC")]`:
   - Query `_context.MaintenanceModes.FirstOrDefaultAsync()`
   - Jika null, buat default object (IsEnabled=false, Message="", Scope="All")
   - Return View dengan model MaintenanceMode

2. `Maintenance` POST action dengan `[HttpPost]`, `[ValidateAntiForgeryToken]`, `[Authorize(Roles = "Admin, HC")]`:
   - Parameter: `bool isEnabled, string message, DateTime? estimatedEndTime, string scope, string? selectedModules`
   - Jika `scope == "Specific"` dan `selectedModules` tidak null, set `Scope = selectedModules` (comma-separated dari checklist)
   - Jika `scope == "All"`, set `Scope = "All"`
   - Upsert: jika row ada update, jika tidak insert baru
   - Jika enabling: set ActivatedByUserId, ActivatedByName, ActivatedAt = DateTime.UtcNow
   - Jika disabling: set DeactivatedAt = DateTime.UtcNow, IsEnabled = false
   - Invalidate cache: `_cache.Remove("MaintenanceMode_State")`
   - Audit log via `_auditLogService.LogAsync(userId, userName, isEnabled ? "MaintenanceEnabled" : "MaintenanceDisabled", $"Maintenance mode {(isEnabled ? "diaktifkan" : "dinonaktifkan")} — Scope: {scope}", maintenance.Id, "MaintenanceMode")`
   - TempData["SuccessMessage"] = isEnabled ? "Mode pemeliharaan berhasil diaktifkan." : "Mode pemeliharaan berhasil dinonaktifkan."
   - Redirect to `Maintenance` GET

@inject Microsoft.Extensions.Caching.Memory.IMemoryCache _maintenanceCache
@inject HcPortal.Data.ApplicationDbContext _maintenanceDb
@{
    HcPortal.Models.MaintenanceMode? _mState = null;
    if (User.IsInRole("Admin") || User.IsInRole("HC"))
    {
        if (!_maintenanceCache.TryGetValue("MaintenanceMode_State", out _mState))
        {
            _mState = await _maintenanceDb.MaintenanceModes.FirstOrDefaultAsync();
        }
    }
}
@if (_mState != null && _mState.IsEnabled && (User.IsInRole("Admin") || User.IsInRole("HC")))
{
    <div class="bg-warning text-dark text-center py-2 small">
        <i class="bi bi-exclamation-triangle-fill"></i>

        <a href="/Admin/Maintenance" class="btn btn-sm btn-outline-dark ms-2">Kelola</a>
    </div>
