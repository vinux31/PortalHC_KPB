### 2. Halaman /Admin/Maintenance (D-02, D-03, D-11, D-12)

Layout: Single-column form inside card, max-width container.

Komponen:
- **Header**: h2 "Mode Pemeliharaan" dengan icon `bi-cone-striped` dan badge status (Aktif/Nonaktif)
- **Toggle switch**: Bootstrap form-check form-switch untuk on/off
- **Scope selector**: Radio group — "Seluruh Website" atau "Halaman Tertentu"
- **Checklist halaman** (muncul jika "Halaman Tertentu" dipilih): Checkbox group dengan label per modul (Assessment, Coaching Proton, CDP, Kelola Data, dll)
- **Textarea pesan kustom**: 3 baris default, placeholder "Masukkan pesan untuk ditampilkan kepada pengguna..."
- **Datetime picker estimasi selesai**: Input type="datetime-local"
- **Submit button**: btn btn-primary "Simpan Pengaturan"

### 3. Maintenance Banner untuk Admin/HC (saat mode aktif)

Sticky top bar di atas navbar, full-width:
- Background: `bg-warning text-dark`
- Icon: `bi-exclamation-triangle-fill`
- Text: "Mode Pemeliharaan Aktif — {scope info}"
- Button: btn btn-sm btn-outline-dark "Kelola" (link ke /Admin/Maintenance)

### 4. Halaman Maintenance untuk User (D-07, D-08, D-14)

Full-screen, no navbar, no sidebar. Centered content:
- **Logo**: PortalHC logo, max-width 120px
- **Icon**: `bi-tools` at 48px, color text-primary
- **Judul**: Display size "Sedang Dalam Pemeliharaan"
- **Pesan kustom**: Body text dari admin, max-width 500px, text-center
- **Estimasi waktu**: Badge bg-info text-dark, icon `bi-clock`, format "Estimasi selesai: {datetime}"
- **Info partial** (D-14): Jika partial maintenance, tampilkan list halaman yang di-maintenance dalam `<ul>` sederhana
- **Footer**: Small text-muted "Hubungi admin jika ada pertanyaan mendesak"

Background: White (#ffffff). Tidak ada warna gelap/dramatis — konsisten dengan tone portal internal.

## Interaction States

### Toggle Activation Flow
1. Admin membuka /Admin/Maintenance
2. Form menampilkan state terakhir (toggle, pesan, estimasi, scope, checklist)
3. Admin mengubah pengaturan dan klik "Simpan Pengaturan"
4. POST ke server, redirect kembali dengan TempData success message
5. Jika toggle ON: banner warning muncul di semua halaman untuk Admin/HC

### User Redirect Flow
1. Non-admin/HC request halaman yang di-maintenance
2. Middleware/filter intercept, cek maintenance state dari cache
3. Redirect ke /Home/Maintenance (atau return view langsung)
4. Halaman maintenance ditampilkan tanpa layout navbar/sidebar

### Partial Maintenance Behavior
1. Jika scope = "Seluruh Website": semua halaman redirect
2. Jika scope = "Halaman Tertentu": hanya halaman yang di-checklist yang redirect
3. Halaman yang tidak di-checklist tetap bisa diakses normal
