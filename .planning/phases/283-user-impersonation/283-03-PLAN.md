## Task 1: Fix NIP "null" di Autocomplete + Fix Navigasi Impersonation

### 1a. Fix NIP null display (minor)

**File:** `Views/Shared/_Layout.cshtml` **Baris:** 289

**Ubah:**
```javascript
+ '<strong>' + u.fullName + '</strong> <small class="text-muted">' + u.nip + '</small>'
```

**Menjadi:**
```javascript
+ '<strong>' + u.fullName + '</strong>' + (u.nip ? ' <small class="text-muted">' + u.nip + '</small>' : '')
```

### 1b. Fix showAdminMenu logic (major)

**File:** `Views/Shared/_Layout.cshtml` **Baris:** 46

**Ubah:**
```csharp
var showAdminMenu = effectiveRole == null && (User.IsInRole("Admin") || User.IsInRole("HC"));
```

**Menjadi:**
```csharp
var showAdminMenu = !isImpersonating && (User.IsInRole("Admin") || User.IsInRole("HC"));
```

**Alasan:** Saat impersonate user (bukan role), `effectiveRole` tetap null karena `ImpersonateTargetRole` tidak di-set. Seharusnya cek `isImpersonating` langsung.

### 1c. Fix greeting & badge menampilkan target impersonation (major)

**File:** `Views/Shared/_Layout.cshtml` **Baris:** 8-16

**Setelah** baris 12 (`var fullName = ...`), tambahkan override logic:

```csharp
// Override display info saat impersonation aktif
var isImpersonatingEarly = Context.Items["IsImpersonating"] is true;
var displayName = isImpersonatingEarly
    ? (Context.Items["ImpersonateTargetName"]?.ToString() ?? fullName)
    : fullName;
var displayRole = isImpersonatingEarly
    ? (Context.Items["ImpersonateTargetRole"]?.ToString() ?? Context.Items["ImpersonateMode"]?.ToString() ?? userRole)
    : userRole;
var displayNameParts = displayName.Split(' ', StringSplitOptions.RemoveEmptyEntries);
var displayInitials = displayNameParts.Length >= 2
    ? $"{displayNameParts[0][0]}{displayNameParts[1][0]}".ToUpper()
    : (displayName.Length >= 2 ? displayName.Substring(0, 2).ToUpper() : "?");
```

**PERHATIAN:** `isImpersonating` baru di-declare di baris 44. Karena kita butuh di baris 8-16, kita pakai variabel `isImpersonatingEarly` terpisah.

**Lalu update semua display references:**

- Baris 126: `@currentUser.FullName` → `@displayName`
- Baris 128: `@userRole` → `@displayRole`
- Baris 135: `@initials` → `@displayInitials`
- Baris 139: `@currentUser.FullName` → `@displayName`
- Baris 140: `@userRole` → `@displayRole`
