## Component Inventory

### 1. Impersonation Banner (`_ImpersonationBanner.cshtml`)

| Property | Value |
|----------|-------|
| Position | Fixed top, di atas navbar |
| z-index | 1040 (di atas navbar 1030) |
| Background | `bg-danger` (#dc3545) |
| Text color | `text-white` |
| Padding | `py-2 px-3` (8px vertical, 16px horizontal) |
| Layout | Flex row, centered: icon + text + button |
| Icon | `bi bi-eye-fill` (Bootstrap Icons) |
| Text | `<strong>Anda melihat sebagai [Role/Nama User]</strong>` |
| Button | `btn btn-sm btn-outline-light` — label: "Kembali ke Admin" |
| Button icon | `bi bi-arrow-return-left me-1` |

Saat banner aktif, navbar harus mendapat offset `top` agar tidak tertutup. Gunakan CSS class conditional (`body.impersonating`) yang menambah `padding-top` pada body sesuai tinggi banner.

### 2. Navbar Dropdown "View As" (extend user dropdown)

| Property | Value |
|----------|-------|
| Location | Di bawah nama user di navbar dropdown, sebelum "Logout" |
| Separator | `<hr class="dropdown-divider">` sebelum section "View As" |
| Label | `Lihat Sebagai` (dropdown header, `dropdown-header` class) |
| Items | "HC" dan "User" — masing-masing `dropdown-item` |
| Icon per item | `bi bi-person-badge` (HC), `bi bi-person` (User) |
| Action | POST ke `/Admin/StartImpersonation` dengan `mode=role&targetRole=HC/User` |
| Visibility | Hanya tampil jika `User.IsInRole("Admin")` |

### 3. Navbar User Search (Impersonate User Spesifik)

| Property | Value |
|----------|-------|
| Location | Di dalam dropdown user, setelah "Lihat Sebagai" section |
| Label | `Impersonate User` (dropdown header) |
| Input | `<input type="text" class="form-control form-control-sm" placeholder="Ketik nama atau NIP...">` |
| Min characters | 2 karakter sebelum search dimulai |
| Debounce | 300ms |
| API endpoint | GET `/Admin/SearchUsersApi?q={query}` |
| Result dropdown | Max 10 items, setiap item menampilkan: `FullName` (bold) + `NIP` (muted) + `SelectedView` (badge) |
| Result item action | POST ke `/Admin/StartImpersonation` dengan `mode=user&targetUserId={id}` |
| Empty result | Teks "Tidak ada hasil" (muted, italic) |
| Loading state | Spinner kecil (`spinner-border spinner-border-sm`) di dalam input |

### 4. Disabled Submit Buttons (Read-Only Mode)

| Property | Value |
|----------|-------|
| Target | Semua `button[type=submit]`, `input[type=submit]` |
| State | `disabled` attribute + `opacity: 0.6` |
| Badge | `<span class="badge bg-danger ms-2">Mode Read-Only</span>` ditambahkan setelah setiap submit button |
| Implementation | JavaScript global di `_Layout.cshtml` — cek `document.body.dataset.impersonating === "true"` |

### 5. AJAX 403 Toast (Blocked Write via AJAX)

| Property | Value |
|----------|-------|
| Trigger | Response status 403 dengan `readOnly: true` di body |
| Position | Top-right, Bootstrap toast |
| Background | `bg-danger text-white` |
| Icon | `bi bi-shield-lock` |
| Message | "Aksi ini diblokir — Mode Read-Only aktif" |
| Auto-hide | 4 detik |

### 6. Auto-Expire Redirect

| Property | Value |
|----------|-------|
| Trigger | Middleware deteksi `StartedAt + 30 menit < now` |
| Action | Clear session keys, redirect ke `/Admin/Index` |
| TempData message | "Sesi impersonation telah berakhir (batas 30 menit)." |
| Toast style | `alert-warning` di halaman Admin/Index |

## Interaction States

### Banner Lifecycle

| State | Visual | Trigger |
|-------|--------|---------|
| Inactive | Tidak ada banner, navbar normal | Default (tidak impersonating) |
| Active (role) | Banner merah, teks "Anda melihat sebagai [Role]" | Admin klik "Lihat Sebagai HC/User" |
| Active (user) | Banner merah, teks "Anda melihat sebagai [Nama]" | Admin pilih user dari search |
| Expired | Banner hilang, redirect ke Admin/Index + warning toast | 30 menit berlalu |

### Search Autocomplete Lifecycle

| State | Visual |
|-------|--------|
| Idle | Input kosong, placeholder "Ketik nama atau NIP..." |
| Typing (< 2 char) | Input berisi teks, tidak ada dropdown |
| Loading | Input berisi teks (>= 2 char), spinner di input |
| Results | Dropdown muncul di bawah input, max 10 items |
| No results | Dropdown dengan teks "Tidak ada hasil" |
| Selected | Dropdown tutup, POST impersonation, redirect |

### Navigation Override

| Admin Role | Impersonating As | Menu Visible |
|------------|-----------------|--------------|
| Admin | (not impersonating) | Semua menu termasuk "Kelola Data" |
| Admin | HC | Menu HC (CMP, CDP, tanpa "Kelola Data" admin-only actions) |
| Admin | User | Menu User saja (CMP, CDP — persis seperti user biasa) |
