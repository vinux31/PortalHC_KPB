Tambahkan 6 fungsi baru di bawah fungsi ajaxGet() yang sudah ada. JANGAN ubah 3 fungsi existing (getAntiForgeryToken, ajaxPost, ajaxGet).

**Fungsi 1: escapeHtml(str)**
- Replace `&` -> `&amp;`, `<` -> `&lt;`, `>` -> `&gt;`, `"` -> `&quot;`, `'` -> `&#39;`
- Wajib untuk XSS safety pada innerHTML interpolation

**Fungsi 2: buildTree(flatList)**
- Terima flat array dari GetOrganizationTree
- Buat Map(id -> node), tambahkan property `children = []` pada setiap node
- Loop kedua: jika parentId === null push ke roots, else push ke parent.children
- Return roots array
- PENTING: property JSON dari endpoint menggunakan camelCase (`id`, `name`, `parentId`, `level`, `displayOrder`, `isActive`)

**Fungsi 3: renderNode(node, level)**
- Default expand: `level < 2` (Level 0 + 1 expanded, Level 2+ collapsed) — per CONTEXT Decision #2
- Icon per level: level===0 `bi-building`, level===1 `bi-diagram-3`, level>=2 `bi-dot`
- Badge: isActive ? `<span class="badge rounded-pill bg-success badge-status">Aktif</span>` : `<span class="badge rounded-pill bg-danger badge-status">Nonaktif</span>`
- Dimming: `!node.isActive` -> `style="opacity:0.5"` pada `.tree-row` div
- Chevron: hasChildren ? `<i class="bi bi-chevron-right tree-chevron${isExpanded ? ' expanded' : ''}"></i>` : `<span class="tree-chevron-placeholder"></span>`
- Children ul: `<ul class="tree-children" ${isExpanded ? '' : 'style="display:none"'}>` dengan recursive renderNode pada setiap child
- Return `<li class="tree-node" data-expanded="${isExpanded}" data-id="${node.id}">` wrapping tree-row + children

**Fungsi 4: updateExpandAllButton()**
- Query semua `.tree-node` yang punya `.tree-children`
- Jika ada yang `data-expanded === 'false'`, label button = "Expand All", else "Collapse All"
- Target: `document.getElementById('btn-expand-all')`

**Fungsi 5: initTree()**
- async function
- Get container `document.getElementById('org-tree-container')`
- Show loading spinner: `<div class="text-center py-5"><div class="spinner-border text-primary" role="status"></div><div class="mt-2 text-muted small">Memuat struktur organisasi...</div></div>`
- try: `const flat = await ajaxGet('/Organization/GetOrganizationTree')`
- buildTree(flat) -> roots
- Empty state jika roots.length === 0: `<div class="text-center text-muted py-5"><i class="bi bi-diagram-3 d-block mb-2 fs-3"></i><strong>Belum ada unit organisasi</strong><div class="small mt-1">Struktur organisasi masih kosong. Tambah Bagian baru untuk memulai.</div></div>`
- Render: `<ul class="tree-root list-unstyled mb-0">${roots.map(r => renderNode(r, 0)).join('')}</ul>`
- Panggil updateExpandAllButton()
- catch: alert danger "Gagal memuat struktur organisasi. Periksa koneksi dan muat ulang halaman."

**Fungsi 6: Event listeners (di akhir file, dalam IIFE atau langsung)**
- Event delegation pada `#org-tree-container` click:
  - `e.target.closest('.tree-row')` -> toggle expand/collapse pada `.tree-node` terdekat
  - Toggle `children.style.display`, `node.dataset.expanded`, chevron class `.expanded`
  - Panggil `updateExpandAllButton()`
- Event listener pada `#btn-expand-all` click:
  - Cek `hasCollapsed` = ada node dengan `data-expanded === 'false'` yang punya `.tree-children`
  - Jika hasCollapsed: expand semua, label "Collapse All"
  - Jika semua expanded: collapse semua, label "Expand All"

PENTING: Event listeners HARUS di-wrap dalam guard check (`if (container)`) karena file di-load di halaman lain juga. Gunakan pattern:
```javascript
document.addEventListener('DOMContentLoaded', function() {
    const container = document.getElementById('org-tree-container');
    if (!container) return;
    // attach event listeners here
});
```
