# Plan 295-01: SortableJS Drag-drop Reorder Sibling-Only

<objective>
Implement drag-and-drop reorder of organization units within the same parent using SortableJS. Add batch reorder endpoint, drag handle UI, and cross-parent blocking.
</objective>

<action>
Add new action method `ReorderBatch` in OrganizationController.cs, after the existing `ReorderOrganizationUnit` method (after line 425).

The method signature:
```csharp
[HttpPost]
[Authorize(Roles = "Admin, HC")]
[ValidateAntiForgeryToken]
public async Task<IActionResult> ReorderBatch(int? parentId, string orderedIds)
```

Implementation:
1. Parse `orderedIds` string by splitting on comma into `int[]` array. If empty or null, return `Json(new { success = false, message = "Data urutan tidak valid." })`
2. Load all units from DB where `ParentId == parentId`, ordered by DisplayOrder
3. Validate: every ID in the parsed array must belong to that parent group. If any ID is not found among siblings, return error JSON: `"ID tidak valid atau bukan sibling yang sama."`
4. Validate: count of parsed IDs must equal count of siblings. If mismatch, return error JSON: `"Jumlah item tidak sesuai."`
5. Loop through parsed IDs array. For each ID at index `i`, set the matching unit's `DisplayOrder = i + 1`
6. `await _context.SaveChangesAsync()`
7. Return `Json(new { success = true, message = "Urutan berhasil diubah." })` (no need for IsAjaxRequest check since this endpoint is AJAX-only)
</action>

- Method validates that all IDs belong to same parentId before updating

**Step D — Add initSortable() function in orgTree.js after the initTree function:**

```javascript
function initSortable() {
    if (typeof Sortable === 'undefined') return;
    document.querySelectorAll('.tree-children, .tree-root').forEach(function(ul) {
        Sortable.create(ul, {
            handle: '.drag-handle',
            animation: 150,
            ghostClass: 'sortable-ghost',
            chosenClass: 'sortable-chosen',
            group: false,
            onEnd: function(evt) {
                if (evt.oldIndex === evt.newIndex) return;
                var container = evt.from;
                var parentNode = container.closest('.tree-node');
                var parentId = parentNode ? parentNode.dataset.id : '';
                var orderedIds = Array.from(container.children)
                    .map(function(li) { return li.dataset.id; })
                    .filter(function(id) { return id; })
                    .join(',');
                ajaxPost('/Admin/ReorderBatch', { parentId: parentId, orderedIds: orderedIds })
                    .then(function(res) {
                        if (res.success) {
                            showToast(res.message, 'success');
                        } else {
                            showToast(res.message, 'danger');
                            initTree();
                        }
                    })
                    .catch(function() {
                        showToast('Gagal mengubah urutan.', 'danger');
                        initTree();
                    });
            }
        });
    });
}
```

**Step E — Call initSortable() at end of initTree():**
