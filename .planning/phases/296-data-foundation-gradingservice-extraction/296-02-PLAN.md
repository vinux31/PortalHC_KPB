**Method `GradeAndCompleteAsync(AssessmentSession session)`** — per D-02, handle 5 hal:

1. **Hitung skor** — Ekstrak dari GradeFromSavedAnswers():
   - Load PackageAssignment via `_context.UserPackageAssignments.FirstOrDefaultAsync(a => a.AssessmentSessionId == session.Id)`
   - Get shuffledIds via `packageAssignment.GetShuffledQuestionIds()`
   - Load PackageQuestions with Options
   - Load PackageUserResponses as dictionary
   - Loop per question: switch-case per `q.QuestionType ?? "MultipleChoice"` (per D-08):
     - `"MultipleChoice"`: cek response.PackageOptionId matches correct option → totalScore += q.ScoreValue
     - `"MultipleAnswer"`: `throw new NotImplementedException("MultipleAnswer grading akan diimplementasi di Phase 298.")`
     - `"Essay"`: `throw new NotImplementedException("Essay grading akan diimplementasi di Phase 298.")`
   - finalPercentage = maxScore > 0 ? (int)((double)totalScore / maxScore * 100) : 0
   - isPassed = finalPercentage >= session.PassingScore

2. **Hitung SessionElemenTeknisScores** — Ekstrak dari GradeFromSavedAnswers():
   - GroupBy ElemenTeknis (fallback "Lainnya")
   - Per group: hitung correct count, add SessionElemenTeknisScore entity

3. **Update session (race-condition-safe)** — per D-04:
   - SaveChangesAsync() dulu untuk ET scores (per CMPController pattern)
   - ExecuteUpdateAsync dengan WHERE `s.Id == session.Id && s.Status != "Completed"`
   - Set: Score=finalPercentage, Status="Completed", Progress=100, IsPassed=isPassed, CompletedAt=DateTime.UtcNow
   - Cek rowsAffected == 0 → log warning race condition, return early
   - Update PackageAssignment.IsCompleted = true via ExecuteUpdateAsync

4. **Buat TrainingRecord (dengan duplicate guard)** — Ekstrak dari kedua controller:
   - AnyAsync cek duplikasi (UserId + Judul + Tanggal yang sama)
   - Add TrainingRecord jika belum ada

5. **Generate NomorSertifikat (jika applicable)** — Ekstrak dari AkhiriUjian:
   - Kondisi: `session.GenerateCertificate && isPassed`
   - Retry loop 3x: CertNumberHelper.GetNextSeqAsync + CertNumberHelper.Build
   - ExecuteUpdateAsync WHERE NomorSertifikat == null
   - DbUpdateException catch untuk race condition sequence

6. **Notifikasi grup** — `await _workerDataService.NotifyIfGroupCompleted(session)`

**PENTING per D-03:** TIDAK handle interview Tahun 3 (Proton). Method ini hanya untuk package-based assessment.
