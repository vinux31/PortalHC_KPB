## Component Inventory

Komponen yang akan ditambahkan atau dimodifikasi di phase ini:

### 1. CreateAssessment — Dropdown AssessmentType + Dual-Section Jadwal

**Komponen:** `select.form-select` + collapsible `div#ppt-jadwal-section`

**State:** `collapse` (hidden) saat AssessmentType = Standard, `show` saat Pre-Post Test dipilih

**Layout dual-section:**
```
[Card: Jadwal Pre-Test]
  - Schedule (datetime-local)
  - DurationMinutes (number)
  - ExamWindowCloseDate (datetime-local)

[Card: Jadwal Post-Test]
  - Schedule (datetime-local, disabled jika < Pre Schedule)
  - DurationMinutes (number)
  - ExamWindowCloseDate (datetime-local)
  - GenerateCertificate (checkbox) + ValidUntil (date)
```

**Checkbox "Gunakan paket soal yang sama":** `form-check` dengan label "Gunakan paket soal yang sama untuk Pre dan Post". Saat checked, tampilkan info badge di section Post: `badge bg-info bi-info-circle "Paket soal disamakan dengan Pre-Test"`.

**Validasi tanggal:** Input Schedule Post diset `min` attribute sama dengan nilai Schedule Pre secara JavaScript (event `change` pada input Schedule Pre).

### 2. ManageAssessment — Card Pre-Post Badge

**Komponen:** `badge` tambahan pada card judul assessment

**Visual:** `badge bg-primary rounded-pill bi-arrow-left-right "Pre-Post Test"` di samping nama assessment

**Tidak ada** tombol Delete individual untuk session Pre atau Post — tombol delete hanya muncul per-grup.

### 3. AssessmentMonitoring — Expandable Pre-Post Group Row

**Komponen:** Tabel dengan parent row + 2 sub-row tersembunyi

**Parent row:**
- Badge `bi-arrow-left-right "Pre-Post"` dengan class `badge bg-primary`
- Stat gabungan: total peserta, completed = Post completed, passed = Post passed
- Tombol expand: `bi-chevron-down` / `bi-chevron-up` toggle (text-primary)
- Status derived (tidak disimpan di DB)

**Sub-row Pre-Test:**
- Indent 24px kiri (padding-left: 24px)
- Badge `badge bg-info "Pre-Test"`
- Stat: peserta, completed Pre, link ke MonitoringDetail
- Aksi: Akhiri Semua Ujian Pre, Reshuffle All Pre

**Sub-row Post-Test:**
- Indent 24px kiri
- Badge `badge bg-secondary "Post-Test"`
- Stat: peserta, completed Post, passed Post, link ke MonitoringDetail
- Aksi: Akhiri Semua Ujian Post, Reshuffle All Post

**Expand behavior:** JavaScript toggle `d-none` pada sub-row saat klik chevron di parent row.

### 4. EditAssessment — Tab Pre / Post

**Komponen:** `nav nav-tabs` dengan 2 tab: "Pre-Test" dan "Post-Test"

**Tab Pre-Test:** Form edit jadwal Pre (Schedule, DurationMinutes, ExamWindowCloseDate) + link ManagePackages Pre

**Tab Post-Test:** Form edit jadwal Post + GenerateCertificate + ValidUntil + link ManagePackages Post

**Tab styling:** `nav-link active` Bootstrap default. Badge count paket soal di dalam label tab: `badge bg-secondary ms-1`.

### 5. ManagePackages — Tombol "Copy dari Pre-Test"

**Komponen:** `btn btn-outline-info bi-clipboard "Copy dari Pre-Test"` — hanya muncul pada ManagePackages untuk session Post

**Konfirmasi inline:** Saat diklik, tampilkan `alert alert-warning` inline: "Ini akan menimpa semua paket soal Post-Test yang ada. Lanjutkan?" dengan tombol Konfirmasi (btn-warning) dan Batal (btn-secondary).

### 6. Modal Delete Grup

**Komponen:** Bootstrap modal standar

**Visual:** `modal-header bg-danger text-white` dengan ikon `bi-trash3`

**Body:** Daftar dampak — "Assessment ini akan menghapus: [jumlah] paket soal, [jumlah] jawaban peserta, [jumlah] sesi (Pre + Post)."

**Footer:** Tombol "Hapus Grup" (btn-danger) + "Batal" (btn-secondary)

## Interaction States

| Komponen | State | Perilaku |
|----------|-------|---------|
| Dropdown AssessmentType | Pilih "Pre-Post Test" | Dual-section jadwal expand (Bootstrap collapse show) |
| Dropdown AssessmentType | Pilih "Standard" | Dual-section tersembunyi, field Pre/Post di-clear |
| Input Schedule Pre | onChange | Update `min` attribute pada input Schedule Post |
| Checkbox paket soal sama | checked | Tampilkan info badge di section Post, field paket Post disabled |
| Tombol expand monitoring | click | Toggle d-none pada 2 sub-row, toggle icon chevron-down/up |
| Tombol "Copy dari Pre-Test" | click | Tampilkan alert konfirmasi inline (bukan modal) |
| Tombol delete individual Pre/Post | — | Tidak ditampilkan (tidak ada) — D-19 dari CONTEXT.md |
| Input Schedule Post | Tanggal < Pre | Diblokir oleh `min` attribute; backend juga enforce — D-06 |
