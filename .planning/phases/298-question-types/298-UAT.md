## Bugs Fixed (inline — severity low/medium)

### BUG-1: SubmitExam incomplete submission check (MEDIUM)
file: Controllers/CMPController.cs ~line 1504
problem: SubmitExam hanya menghitung `answers.Count(a => a.Value > 0)` dari form POST. Essay dan MA yang disimpan via SignalR tidak ada di dictionary `answers`, sehingga dianggap belum dijawab.
fix: Merge form answers dengan DB responses (`PackageUserResponses`) untuk menghitung total soal terjawab.

### BUG-2: ToDictionaryAsync crash pada soal MA (MEDIUM)
file: Controllers/CMPController.cs ~line 1566
problem: `ToDictionaryAsync(r => r.PackageQuestionId)` crash "duplicate key" karena MA menyimpan multiple rows per soal (satu per opsi).
fix: Ganti dengan `ToListAsync()` + `GroupBy().ToDictionary()`.

### BUG-3: Upsert loop overwrite MA/Essay answers (MEDIUM)
file: Controllers/CMPController.cs ~line 1574
problem: Loop scoring set `PackageOptionId = selectedOptId` untuk SEMUA tipe soal. Untuk MA/Essay, `selectedOptId` = null (tidak ada di form dict), menghapus jawaban yang sudah disimpan via SignalR.
fix: Refactor loop untuk handle MC, MA, Essay secara terpisah. MC upsert dari form, MA scoring dari DB rows, Essay skip (dinilai manual).
