## Interaction Contracts

### Dynamic Form Switching (HC Create/Edit Soal)

- Saat `QuestionType` dropdown berubah ke **Essay**: sembunyikan section opsi A-D (slide/collapse), tampilkan textarea rubrik. ScoreValue field tetap tampil.
- Saat `QuestionType` berubah ke **MC**: tampilkan opsi A-D dengan radio input, sembunyikan textarea rubrik. ScoreValue fixed 10 (disabled).
- Saat `QuestionType` berubah ke **MA**: tampilkan opsi A-D dengan checkbox input, sembunyikan textarea rubrik. ScoreValue fixed 10 (disabled).
- Jika ada jawaban peserta aktif dan tipe diubah: tampilkan modal konfirmasi sebelum apply perubahan.

### Essay Textarea Auto-Save (Worker StartExam)

- Debounce 2 detik setelah berhenti mengetik → kirim via SignalR hub (pola yang sudah ada).
- Counter karakter update real-time saat ketik.
- Saat counter > 90% dari max (1800/2000): counter berubah warna `text-warning`.
- Saat counter = max (2000/2000): textarea disabled untuk input tambahan, counter `text-danger`.

### MA Checkbox Auto-Save (Worker StartExam)

- Auto-save segera setiap checkbox berubah (tidak ada debounce — pola berbeda dari Essay).
- Jawaban MA disimpan sebagai comma-separated option IDs: `"ans_N": "optId1,optId2"`.
- Card soal background: `bg-success bg-opacity-10` saat setidaknya 1 opsi dipilih (konsisten dengan MC radio).

### Essay Grading Inline (HC AssessmentMonitoringDetail)

- Input skor tampil langsung (tidak ada toggle). Pre-fill dengan nilai yang sudah tersimpan jika ada.
- "Simpan Skor" → AJAX POST → response: update badge status soal dari "Belum Dinilai" ke "Sudah Dinilai".
- Jika semua soal essay sudah dinilai: muncul tombol "Selesaikan Penilaian" (sticky atau di bawah list).
- Klik "Selesaikan Penilaian" → modal konfirmasi → AJAX POST → recalculate + update IsPassed + generate sertifikat.

### Preview Modal (HC Manage Questions)

- Tombol preview di setiap row tabel soal: `<button class="btn btn-outline-secondary btn-sm"><i class="bi bi-eye me-1"></i>Preview</button>`.
- Modal menampilkan card soal persis layout StartExam (termasuk badge tipe, opsi/textarea sesuai tipe, disabled state).
- Tidak ada interaksi di dalam preview (semua input `disabled` atau `readonly`).
