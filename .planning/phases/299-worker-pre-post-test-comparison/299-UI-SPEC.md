## Spacing Scale

Declared values (multiples of 4, aligned ke Bootstrap spacing utilities):

| Token | Value | Bootstrap Util | Usage |
|-------|-------|---------------|-------|
| xs | 4px | `gap-1`, `p-1` | Icon gap, badge padding internal |
| sm | 8px | `gap-2`, `p-2` | Compact meta item gap |
| md | 16px | `p-3`, `gap-3` | Card body padding default |
| lg | 24px | `p-4`, `mb-4` | Card body padding (`p-4`), section margin |
| xl | 32px | `py-5` (approx) | Section vertikal break |
| 2xl | 48px | `py-5` | Empty state padding |
| 3xl | 64px | — | Tidak digunakan di phase ini |

Exceptions:
- Left-border linking visual: 4px solid (`border-start border-4`) — token xs
- Touch target tombol aksi: minimum 44px height via `py-2 px-3` pada `btn`

Sumber: pola eksisting `Assessment.cshtml` card body `p-4`, meta spacing `mb-3`.

## Typography

Dua font weight digunakan: **400 (regular)** untuk body dan label, **700 (bold)** untuk heading dan card header.

| Role | Size | Weight | Line Height | Bootstrap Class |
|------|------|--------|-------------|-----------------|
| Body | 14px | 400 (regular) | 1.5 | default / `.small` |
| Label | 14px | 400 (regular) | 1.4 | `.small` |
| Heading (card title) | 16px | 700 (bold) | 1.3 | `.fw-bold` (h5 card-title) |
| Display (page heading) | 20px | 700 (bold) | 1.2 | `h2 .fw-bold` |

Catatan: `fw-semibold` (600) **tidak digunakan** di phase ini. Semua elemen yang sebelumnya memerlukan penekanan visual (card-header, gain score, label aksi) menggunakan `fw-bold` (700).

Sumber: pola eksisting `Assessment.cshtml` — `h2.fw-bold`, `h5.card-title.fw-bold`, `.small.text-muted`.

## Component Inventory

### 1. Pre-Post Card Pair (WKPPT-01)

**Card Pre-Test:**
```
.card.border-0.shadow-sm.assessment-card-item
  border-start border-4 border-primary  ← linking visual (D-03)
  .card-body.p-4

    h5.card-title.fw-bold  ← judul assessment
    .assessment-meta  ← tanggal, durasi, status
    .mt-auto > [tombol aksi]
```

**Card Post-Test (Pre belum Completed — D-04):**
```
.card.border-0.shadow-sm.assessment-card-item.opacity-50  ← grayed out
  border-start border-4 border-primary
  [sama seperti card di atas]
  .mt-auto > button.btn.btn-secondary[disabled]
    "Selesaikan Pre-Test terlebih dahulu"
```

**Card Post-Test (Pre expired — D-05):**
```
  .mt-auto > span.badge.bg-danger.w-100.py-2
    "Pre-Test tidak diselesaikan"
```

**Card Post-Test (Pre Completed, jadwal belum tiba — D-06):**
```
  .mt-auto > button.btn.btn-outline-secondary[disabled]
    "Opens [DD MMM YYYY, HH:mm] WIB"
```

**Arrow connector antara 2 card (D-03):**
- Posisi: elemen `<div class="d-flex align-items-center justify-content-center my-2">` di antara dua card dalam satu `.row`
- Isi desktop: `<i class="bi bi-arrow-right-circle-fill text-primary fs-4" aria-hidden="true"></i>`
- Isi mobile: `<i class="bi bi-arrow-down-circle-fill text-primary fs-4" aria-hidden="true"></i>`
- Wrapper wrapper div harus menyertakan `aria-label="Pre-Test menuju Post-Test"` agar screen reader tetap informatif meski ikon dihide
- Breakpoint switch: `d-none d-md-flex` untuk arrow kanan, `d-flex d-md-none` untuk arrow bawah

### 2. Tab Filtering Strategy (D-08 — Claude's Discretion)

Pre-Post card pair mengikuti status card **Post-Test** untuk tab placement:
- Jika Post masih Upcoming → pair muncul di tab "Upcoming"
- Jika Post sudah Open (dan Pre juga Open) → pair muncul di tab "Open"
- Jika keduanya Completed → tidak tampil di My Assessments (masuk Riwayat)

Implementasi: attribute `data-status` pada wrapper div pasangan card diset ke status Post-Test. JS tab filter existing tidak perlu diganti — cukup extend dengan wrapper group.

### 3. Riwayat Ujian Badge (D-09)

Tabel riwayat kolom "Judul":
```
<span class="badge bg-info text-dark me-1">Pre-Test</span> Judul Assessment
<span class="badge bg-primary me-1">Post-Test</span> Judul Assessment
```

Tombol di kolom aksi untuk baris Post-Test: `btn btn-sm btn-outline-primary` → "Lihat Detail" (D-10).
