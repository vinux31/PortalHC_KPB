**3. Controller — AssessmentAdminController.cs (per D-08, D-09, D-10, D-11):**
Inject `IHubContext<AssessmentHub>` di constructor (jika belum ada). Tambahkan endpoint:
```csharp
[HttpPost]
public async Task<IActionResult> AddExtraTime(int assessmentId, int minutes)
{
    if (minutes < 5 || minutes > 120 || minutes % 5 != 0)
        return Json(new { success = false, message = "Waktu harus antara 5-120 menit, kelipatan 5." });

    // Cari semua session aktif (InProgress) dengan AssessmentId ini
    var sessions = await _context.AssessmentSessions
        .Where(s => s.AssessmentId == assessmentId && s.Status == "InProgress")
        .ToListAsync();

    if (!sessions.Any())
        return Json(new { success = false, message = "Tidak ada peserta aktif." });

    foreach (var session in sessions)
    {
        session.ExtraTimeMinutes = (session.ExtraTimeMinutes ?? 0) + minutes;
    }
    await _context.SaveChangesAsync();

    // Broadcast ke semua peserta aktif via SignalR
    var batchKey = sessions.First().AccessToken;
    await _hubContext.Clients.Group($"batch-{batchKey}")
        .SendAsync("ExtraTimeAdded", minutes * 60);

    return Json(new { success = true, message = $"Waktu ujian berhasil ditambahkan. Peserta mendapat {minutes} menit tambahan." });
}
```

Catatan: Cari field yang menentukan assessment identity (mungkin AccessToken atau AssessmentId) — gunakan yang sesuai dari kode existing. Cari juga status field name yang benar (mungkin "InProgress" atau nilai lain) dari penggunaan existing di controller.

```javascript
function addExtraTime() {
    var minutes = document.getElementById('extraTimeSelect').value;
    var assessmentId = @Model.AssessmentId; // atau ambil dari ViewBag/Model yang sesuai
    fetch('@Url.Action("AddExtraTime", "AssessmentAdmin")', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/x-www-form-urlencoded',
            'RequestVerificationToken': document.querySelector('input[name="__RequestVerificationToken"]').value
        },
        body: 'assessmentId=' + assessmentId + '&minutes=' + minutes
    })
    .then(r => r.json())
    .then(data => {
        bootstrap.Modal.getInstance(document.getElementById('extraTimeModal')).hide();
        // Toast notification — gunakan pattern toast yang sudah ada di project
        showToast(data.message, data.success ? 'success' : 'error');
    })
    .catch(() => showToast('Gagal menambah waktu. Silakan coba lagi.', 'error'));
}
```
