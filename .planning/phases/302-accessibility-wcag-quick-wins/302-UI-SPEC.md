## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing (padding tombol kecil) |
| md | 16px | Default element spacing (padding card, form group) |
| lg | 24px | Section padding (modal body padding) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Skip link: `padding: 8px 16px` (sm + md) saat visible, 0 saat hidden
- Focus outline offset: 2px (tidak perlu multiple of 4, mengikuti standar browser)
- Touch targets (dari Phase 300): min 48x48dp untuk semua tombol di mobile

## Komponen & Interaksi Phase 302

### A. Skip Link (A11Y-01)

**Sumber keputusan:** D-01, D-02, D-03 dari CONTEXT.md

**Visual contract:**
- Hidden by default: posisi absolut offscreen (`position: absolute; left: -9999px`) atau menggunakan Bootstrap `visually-hidden` class yang sudah tersedia
- Saat `:focus-visible`: muncul di atas semua konten dengan `position: fixed; top: 8px; left: 8px; z-index: 9999; background: #0d6efd; color: #ffffff; padding: 8px 16px; border-radius: 4px; text-decoration: none; font-size: 14px; font-weight: 600`
- Teks: "Lewati ke konten utama"
- Target: anchor ke `#main-content` (area soal di StartExam.cshtml)

**Implementasi CSS class baru:**
```css
.skip-link {
    position: absolute;
    left: -9999px;
    top: 8px;
    z-index: 9999;
    padding: 8px 16px;
    background: #0d6efd;
    color: #ffffff;
    border-radius: 4px;
    font-size: 14px;
    font-weight: 600;
    text-decoration: none;
}
.skip-link:focus-visible {
    left: 8px;
    outline: 2px solid #ffffff;
    outline-offset: 2px;
}
```

### B. Keyboard Navigation (A11Y-02)

**Sumber keputusan:** D-04, D-05, D-06, D-07 dari CONTEXT.md

**Interaction contract:**
- Radio buttons (MC, True/False): Arrow Up/Down untuk navigasi antar opsi — native browser behavior, tidak perlu JS tambahan
- Checkboxes (MA): Arrow Up/Down untuk navigasi, Space untuk toggle — native browser behavior
- Tab order dalam satu halaman soal: Skip link → [soal 1: label → opsi 1 → opsi 2 → ...] → [soal 2] → ... → [sticky footer: Prev → offcanvas trigger → Next/Submit]
- Essay textarea: Tab masuk ke textarea, Tab lagi keluar ke elemen berikutnya — native behavior
- Focus outline visible: `outline: 2px solid #0d6efd; outline-offset: 2px` pada semua elemen saat `:focus-visible`

**Tidak boleh diubah:** anti-copy handler di lines 1349-1358 StartExam.cshtml hanya block Ctrl+C/A/U/S/P — tidak konflik dengan Tab/Arrow/Enter/Space (D-17).

### C. Auto-Focus Saat Pindah Halaman Soal (A11Y-06)

**Sumber keputusan:** D-14, D-15 dari CONTEXT.md

**Interaction contract:**
- Setelah `performPageSwitch()` selesai render halaman baru, focus otomatis berpindah ke card soal pertama di halaman baru
- Target elemen focus: container soal pertama (elemen dengan class soal atau heading soal pertama)
- Elemen target harus memiliki `tabindex="-1"` agar bisa menerima programmatic focus tanpa masuk tab order natural
- Visual: tidak ada visual indicator khusus — focus outline standar sudah cukup

### D. Extra Time Modal (A11Y-05)

**Sumber keputusan:** D-08, D-09, D-10, D-11, D-12, D-13 dari CONTEXT.md

**Visual contract:**
- Tombol trigger: Bootstrap `btn btn-warning btn-sm` dengan label "Tambah Waktu" + icon `bi bi-clock-history`
- Posisi tombol: di halaman AssessmentMonitoringDetail, di area header/action bar assessment (bukan per baris peserta — D-08: berlaku bulk)
- Modal: Bootstrap modal standar — ukuran `modal-sm`, judul "Tambah Waktu Ujian"
- Input waktu: `<select>` dropdown dengan opsi kelipatan 5 menit: 5, 10, 15, 20, 25, 30, 45, 60, 90, 120 menit
- Label input: "Tambah waktu (menit)"
- Modal footer: tombol "Batal" (`btn-secondary`) + tombol "Tambah Waktu" (`btn-primary`)
- Setelah konfirmasi: modal tutup, timer peserta diupdate real-time via SignalR (tidak perlu refresh)

**State feedback:**
- Success toast: "Waktu ujian berhasil ditambahkan. Peserta mendapat X menit tambahan."
- Error toast: "Gagal menambah waktu. Coba lagi." (warna error: `#c62828` sesuai `assessment-toast--error`)
