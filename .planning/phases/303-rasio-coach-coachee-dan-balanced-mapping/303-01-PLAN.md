**1. GET CoachWorkload** — per D-05, D-06, D-07, D-09, D-10, D-12, D-17, D-18:
```csharp
[Authorize(Roles = "Admin, HC")]
public async Task<IActionResult> CoachWorkload(string? section)
```
- Load threshold dari `_context.CoachWorkloadThresholds.FirstOrDefaultAsync()` — jika null, gunakan default (Max=5, Warning=4)
- Load semua user via `_context.Users.Select(u => new { u.Id, u.FullName, u.Section, u.IsActive }).ToListAsync()` — buat `userDict`
- Query active mappings: `_context.CoachCoacheeMappings.Where(m => m.IsActive && !m.IsCompleted).ToListAsync()` (per D-09, D-10)
- Load coach users via `_userManager.GetUsersInRoleAsync("Coach")` lalu filter `IsActive`
- Build workload rows: GroupBy CoachId, hitung Count per coach. Untuk tiap active coach, buat anonymous object `{ CoachId, CoachName, CoachSection, CoacheeCount, Status }` dimana Status = "Overloaded" jika >= Max, "Warning" jika >= WarningThreshold, "OK" jika di bawah
- Filter by section jika parameter `section` tidak kosong
- Summary cards via ViewBag: `TotalActiveCoaches`, `TotalActiveCoachees` (count activeMappings yang ada di filtered rows), `AvgRatio` (Math.Round double), `OverloadedCount`
- ViewBag: `Threshold`, `WorkloadRows` (ordered descending by CoacheeCount), `Sections` (dari `_context.GetSectionUnitsDictAsync().Keys`), `SectionFilter`
- Generate saran reassign via private method `GenerateReassignSuggestions()` — lihat detail di bawah
- ViewBag: `ReassignSuggestions`
- Return `View()`

**Private method GenerateReassignSuggestions:**
```csharp
private List<dynamic> GenerateReassignSuggestions(
    List<dynamic> rows, List<CoachCoacheeMapping> activeMappings,
    Dictionary<string, dynamic> userDict, CoachWorkloadThreshold threshold)
```
- Identifikasi overloaded coaches (CoacheeCount > MaxCoacheesPerCoach)
- Identifikasi underloaded coaches (CoacheeCount < MaxCoacheesPerCoach)
- Untuk tiap coachee di overloaded coach:
  - Cari target coach di section yang sama dengan coachee (per D-12) yang masih underloaded
  - Jika tidak ada, cari target coach manapun yang underloaded (beban terendah)
  - Buat saran: `{ MappingId, CoacheeName, FromCoachName, ToCoachId, ToCoachName, CoacheeSection }`
  - Skip jika tidak ada target yang eligible
- Return list saran (max 20 saran untuk performa UI)
