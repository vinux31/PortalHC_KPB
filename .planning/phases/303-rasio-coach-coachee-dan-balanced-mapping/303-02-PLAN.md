## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client -> AJAX endpoints | POST data dari JavaScript (mappingId, newCoachId, threshold values) |
| View rendering | ViewBag data di-serialize ke JSON untuk Chart.js — potential XSS |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-303-07 | Tampering | AJAX calls | mitigate | Anti-forgery token di semua POST AJAX requests via `__RequestVerificationToken` header |
| T-303-08 | Information Disclosure | Chart.js data | accept | Data coach-coachee sudah visible untuk HC — no new exposure; JSON serialized via `Json.Serialize` (auto-encoded) |
| T-303-09 | Spoofing | Role check di view | mitigate | `User.IsInRole("Admin")` untuk hide/show UI elements; backend enforcement via `[Authorize]` (Plan 01) |
</threat_model>
