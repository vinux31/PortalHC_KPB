## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser DOM (admin/worker) ↔ Razor view | Helper output (label string + badge class) di-render server-side via `@QuestionTypeLabels.{Method}(...)`. Razor `@` auto-encodes — XSS-safe. |
| Browser ↔ Server (POST CreateQuestion/EditQuestion) | Form submit dengan `<option value="MultipleChoice">` (enum value DB binding) — TIDAK terpengaruh helper. Validation `if (questionType == "MultipleChoice" && correctCount != 1)` reads enum literal. |
| Controller ↔ DB (EF Core) | DB column `PackageQuestions.QuestionType` tetap value `"MultipleChoice"`/`"MultipleAnswer"`/`"Essay"` (D-17). Helper hanya translate enum→display, tidak menyentuh DB. |
| JS handler ↔ DOM dropdown | JS reads `<option value="...">` (enum value) via `dropdown.value` / `data-question-type` (D-19) — TIDAK terpengaruh label change. |
