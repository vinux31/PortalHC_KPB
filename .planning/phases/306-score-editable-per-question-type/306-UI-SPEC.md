### JS Form Submit Flow (NEW logic per D-09)

```
form.submit event fires
  ↓
isEditMode = (editQuestionId.value !== '')
  ↓ NO → submit normally
  ↓ YES
originalScore = parseInt(scoreValue.dataset.originalScore, 10)
newScore     = parseInt(scoreValue.value, 10)
affectedN    = parseInt(form.dataset.affectedSessions, 10) || 0
  ↓
delta = (newScore !== originalScore)
needsConfirm = (delta && affectedN > 0)
  ↓ NO → submit normally
  ↓ YES
event.preventDefault()
  ↓
populateModalCopy(order, originalScore, newScore, affectedN)
  ↓
editScoreWarningModal.show()
  ↓
[user clicks "Ya, Lanjutkan"]
  ↓
editScoreWarningModal.hide()
  ↓
form.submit()  // bypass listener via flag or remove-then-submit
```
