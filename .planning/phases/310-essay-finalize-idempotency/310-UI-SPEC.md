| xs | `*-1` | 4px (0.25rem) | Icon gap di tombol (`me-1` antara `bi-check-circle` dan label) |

| lg | `*-4` | 24px (1.5rem) | Section padding (`mb-4` antar session card) |

**Exceptions Phase 310:** none. Tombol "Selesaikan Penilaian" + tooltip wrapper pakai existing wrapper `<div id="finalizeSection_@session.Id">` tanpa spacing changes. Toast/alert injection pakai `mb-3` di `.container-fluid` (existing pattern dari Phase 302 extra-time alert L1385).

## Interaction Spec

### Disabled-state gate (D-02)

**Trigger condition:** `session.Status == AssessmentConstants.AssessmentStatus.Completed && session.NomorSertifikat != null`

**Visual contract:**
- Tombol tetap visible (NOT hidden via display:none).
- Class `btn btn-success btn-finalize-grading` retained.
- Bootstrap `disabled` attribute added → otomatis: opacity 0.65, cursor `not-allowed`, click handler tidak fire (Bootstrap default behavior).
- `title="Sudah selesai pada {CompletedAt} WIB"` + `data-bs-toggle="tooltip"` → hover tooltip muncul setelah ~500ms delay (Bootstrap default).
- Wrapper `<div id="finalizeSection_@session.Id">` `display:block` (kondisi `EssayPendingCount == 0` sudah pasti true untuk Status=Completed).

**Accessibility:**
- Bootstrap `disabled` attribute auto-set `aria-disabled="true"` → screen reader announce "disabled".
- Tooltip pakai `title` attribute → screen reader baca sebagai accessible name.
- Tidak butuh keyboard handler tambahan (button disabled = tab-skip).

### Friendly no-op alert (D-03)

**Trigger:** Response JSON `{ success: true, alreadyFinalized: true, message: "...", nomorSertifikat: "..." }`.

**Visual contract:**
- Inject `<div class="alert alert-info alert-dismissible fade show mb-3" role="alert">...</div>` ke `afterbegin` `.container-fluid` (top of page, di bawah header).
- Icon `bi bi-info-circle-fill me-2` di awal text.
- Label `<strong>Info:</strong>` prefix.
- Tombol close `<button class="btn-close" data-bs-dismiss="alert"></button>` di kanan.
- **Auto-dismiss:** `setTimeout(() => alertEl.querySelector('.btn-close').click(), 5000)` — 5 detik consistent dengan toast convention.
- **NO** `location.reload()` — biar admin lihat pesan dulu (ini idempotent no-op, state sudah benar).

### Error rejection (D-04)

**Trigger:** Response JSON `{ success: false, message: "..." }` untuk status Open/InProgress/Cancelled.

**Visual contract:**
- Sama dengan D-03 tapi class `alert-danger` + icon `bi-x-circle-fill` + label `<strong>Error:</strong>`.
- Re-enable tombol `this.disabled = false` (existing L1352 behavior preserved).
- Auto-dismiss 7 detik (sedikit lebih lama untuk error, agar admin sempat baca).

### Tooltip initialization

**Trigger:** Page load (DOMContentLoaded).

**Implementation:** 
```javascript
document.addEventListener('DOMContentLoaded', function() {
    var tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
    tooltipTriggerList.forEach(function(el) { new bootstrap.Tooltip(el); });
});
```

Existing pattern di `Views/Admin/AddManualAssessment.cshtml` & related — planner reuse identical snippet.
