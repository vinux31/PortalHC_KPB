## Step 1 — Admin + Open + 0 response → DELETE OK

**Pre-condition:** Login Admin, ada 1 row Status=Open tanpa peserta yang sudah jawab.

**Action:**
1. Navigasi ke `/Admin/ManageAssessment`, cari row Status=Open seed.
2. Klik dropdown action `⋮` → klik tombol "Hapus" / "Hapus Grup".
3. Modal `#deleteAssessmentModal` terbuka.
4. Verify Step 1 panel: spinner sebentar lalu konten muncul. Verify field-field berikut populate:
   - "Status saat ini: <strong>Open</strong>"
   - "0 peserta dengan jawaban tersimpan"
   - "0 sertifikat akan terhapus"
   - Counts paket soal & riwayat sesuai data
5. Klik "Lanjutkan →".
6. Step 2 panel muncul: alert merah "Tindakan ini tidak dapat dibatalkan." + cascade enumeration list 4 item.
7. Klik "🗑 Hapus Permanen".

**Expect:**
- Browser navigate ke `/Admin/ManageAssessment`.
- Banner hijau `alert-success`: "Assessment '...' berhasil dihapus." atau "Grup assessment '...' berhasil dihapus."
- Row hilang dari tabel.
- DB query: `SELECT TOP 1 ActionType, Description FROM AuditLogs ORDER BY CreatedAt DESC` — ActionType = `DeleteAssessment` (atau Group), Description match `Status=Open ResponseCount=0`.
- Cascade verify (DB): `SELECT COUNT(*) FROM PackageUserResponses WHERE AssessmentSessionId=<deletedId>` = 0; same untuk AssessmentAttemptHistory + AssessmentPackages.
