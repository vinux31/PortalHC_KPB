## U3 — CL-B(b) (Buat-tunggu: pending → exam → Siap → Konfirmasi)

Seed: worker Tahun 1, TANPA penanda final.

1. `POST BypassSave` mode `CL-B(b)` Tahun 1→Tahun 2.
2. Cek response Json: `success=true`, **`showAttachPackageReminder=true`** (W-12), `pendingId` terisi. **Cek `TempData["Warning"]` ter-set** (muncul sebagai alert warning di page render berikutnya).
3. SQL cek: `PendingProtonBypass` `Status='Menunggu'` + `LinkedAssessmentSessionId`; `AssessmentSession` bare: `Category='Assessment Proton'`, `Status='Upcoming'`, **`UserId=<worker>` + `AssessmentType='Standard'`** (B-05), TANPA `UserPackageAssignments` (tanpa paket — D-01); deliverable source Approved + history; worker BELUM pindah; TANPA penanda.
4. HC pasang paket via Kelola Assessment (step 2) → worker kerjakan exam → LULUS.
5. SQL cek: pending `Status='Siap'`; `UserNotifications` HC inisiator ada tipe `PROTON_BYPASS_READY` (worker TIDAK dapat); penanda source `Origin='Exam'` terbit.
6. `POST BypassConfirm {pendingId}` → SQL cek: pindah (source inactive, target active `Origin='Bypass'`); **target punya `ProtonDeliverableProgress` hasil bootstrap (W-02)**; pending `Status='Selesai'` + `ResolvedAt`; sesi exam linked TIDAK `'Dibatalkan'` (bukti kelulusan dipertahankan).

## U5 — Batal pending (§8.1)

> **Catatan verifier (I-03):** SC3/PBYP-06 berbunyi "belum-dikerjakan→hapus" — implementasi = **soft-cancel `Status='Dibatalkan'`** (keputusan D-14, reversible). 'hapus' DIPENUHI sebagai 'Dibatalkan', BUKAN row terhapus. JANGAN tandai gap.

a. Pending `Menunggu` + worker BELUM kerjakan (session `Upcoming`): `POST BypassCancelPending` → pending `'Dibatalkan'` + `ResolvedAt`; sesi bare `Status='Dibatalkan'`.
b. Pending `Siap` (worker sudah lulus): batal → pending `'Dibatalkan'` TAPI sesi exam + penanda `Origin='Exam'` DIPERTAHANKAN (hasil sah).
c. **W-03:** worker kerjakan-tapi-GAGAL (session `Completed`, `IsPassed=0`, pending masih `Menunggu`): batal → pending `'Dibatalkan'` TAPI session TETAP `'Completed'` (jejak kerjakan-gagal TIDAK di-overwrite).

- [ ] U5 PASS
