## Spacing Scale

> v32.1 uses Bootstrap's spacing utilities, NOT a custom 4px token system. The "scale" below documents the
> Bootstrap utility idiom already in the shipped markup. Verification preserves these; any view-only fix must
> use the same utilities (never reintroduce inline magic-number `style="font-size:..."` ‚Äî DSN-05 removed them).

| Bootstrap token | px (default) | Usage in shipped markup |
|-----------------|--------------|--------------------------|
| `gap-1` / `g-1` | 4px | Aksi button row (`.d-flex.gap-1`), icon-text gaps |
| `gap-2` / `g-2` | 8px | Toolbar button gap, filter row (`row g-2`), card-header avatar+name gap |
| `mb-3` / `g-3` / `p-3` | 16px | Card vertical rhythm (accordion card `mb-3`; summary cards `g-3`; import-result cells `p-3`) |
| `mb-4` | 24px | Major section breaks on CoachWorkload (summary ‚Üí filter ‚Üí chart ‚Üí table ‚Üí saran) |
| avatar | 36px (fixed) | `avatar-initial` width/height ‚Äî intentional fixed dimension (parity with `ManageWorkers`) |

**Exceptions (intentional, preserve as-is):** avatar `width:36px;height:36px;font-size:0.8rem` inline (matches `ManageWorkers` idiom, D-03); functional/layout inline retained by DSN-05 (`max-height:300px;overflow-y:auto` chart scroll, `min-height:150px` canvas set by Chart.js runtime, `max-width:300px` filter select, chevron `transition:transform .2s`). These are NOT magic-number font-size and were explicitly allowed (388 D-05).

**Verification rule:** Assert NO inline `style="font-size:11px/12px/..."` remains on CoachWorkload (DSN-05 cleanup must hold). `.legend-dot` must be a class (12px dot), not inline geometry; only `background:` color stays inline (data-driven status color).
