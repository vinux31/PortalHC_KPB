    - else (MC/MA): cell = `ws.Cell(row,col++).GetString().Trim()`. If empty → `SkippedBlank++; continue` (D-06 OMIT — do NOT push an empty spec). Split on ',' with `RemoveEmptyEntries|TrimEntries`.
