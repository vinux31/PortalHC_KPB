## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client (Admin/HC browser) → endpoint 410/411 | POST add/remove/restore — untrusted input (sessionId, reason, userIds) lintas ke server |
| server (controller) → SignalR client (monitor grup + worker) | broadcast event berisi fullName/removalReason (data DB) ke browser pemantau & worker |
| broadcast trigger → commit boundary | event hanya boleh dipancarkan setelah transaksi sukses (jangan bocor state rollback) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-412-01 | Tampering | broadcast SEBELUM commit → notif untuk tx rollback | mitigate | Semua `SendAsync` ditempatkan SETELAH `CommitAsync`/`SaveChangesAsync` (Task 1 a/b/c); preseden :4351 |
| T-412-02 | Information Disclosure | `examRemoved` dikirim ke worker yang salah (cross-user kick) | mitigate | `targetUserId` di-resolve server dari `session.UserId` (DB), bukan input client; `Clients.User(userId)` SignalR auto-map (preseden :4905) |
| T-412-03 | Information Disclosure | broadcast bocor ke pemantau batch lain | mitigate | Group-scoped `monitor-{batchKey}`; `JoinMonitor` (:56) sudah role-gate Admin/HC sebelum add-to-group; broadcast hanya ke grup batch terkait |
| T-412-04 | Elevation | non-Admin memicu broadcast | mitigate | Broadcast hanya dari endpoint ber-`[Authorize(Roles="Admin, HC")]` + `[ValidateAntiForgeryToken]` (sudah di 410/411 :2354/:2554/:2583) — tak diubah fase ini |
| T-412-05 | Information Disclosure | force-kick examRemoved ke worker non-InProgress (UX noise / leak status) | mitigate | Kirim HANYA bila `wasInProgress` (capture sebelum core, Task 1b) |
| T-412-06 | Spoofing | client kirim event participantAdded/examRemoved palsu | accept | Server→client only; tak ada hub method client-invokable untuk event ini (anti-spoofing struktural, RESEARCH §Security) |
| T-412-07 | Tampering (XSS) | RemovalReason bebas-teks masuk ViewBag/payload | mitigate | Server hanya MENYALIN reason; encoding-at-render = Plan 02 (Razor `@`/JS `textContent`); ViewModel typed string (carry T-409-10) |
