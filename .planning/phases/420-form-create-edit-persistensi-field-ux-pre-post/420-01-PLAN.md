| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-420-01 | Tampering | Retake MaxAttempts/RetakeCooldownHours dari form | mitigate | `Math.Clamp(MaxAttempts,1,5)` + `Math.Clamp(RetakeCooldownHours,0,168)` di SETIAP tulis (3 jalur Create + Edit std loop). `[Range]` model = lapisan kedua; clamp server = lapisan terakhir (V5). |
| T-420-02 | Tampering | Mass-assignment via model binding penuh | mitigate | Action pakai `model` typed + penyalinan field eksplisit (whitelist); `NomorSertifikat` tidak di-bind dari form (`NomorSertifikat = null` di Create `:1484`). FORM-02/03/04 hanya menyalin field config yang dimaksud. |
