# Plan 89-03: Views — Admin/KkjMatrix.cshtml + CMP/Kkj.cshtml Dynamic Columns

## Goal

Rewrite `Views/Admin/KkjMatrix.cshtml` to render columns dynamically from `KkjColumn` records per Bagian (instead of the hardcoded 15 `Label_*`/`Target_*` column map). Add a "Kelola Kolom" panel in the admin view for KkjColumn CRUD and a "Kelola Pemetaan Jabatan" panel for PositionColumnMapping CRUD. Rewrite `Views/CMP/Kkj.cshtml` to render target value columns dynamically. Update `CMPController.Kkj()` to pass the correct data for dynamic rendering.

## must_haves

- `Views/Admin/KkjMatrix.cshtml` has NO references to `Label_*` properties or `Target_*` properties from the old hardcoded list
- `Views/Admin/KkjMatrix.cshtml` loads columns via AJAX call to `GET /Admin/GetKkjColumns?bagianId=X` on bagian selection change
- Read-mode table renders header columns dynamically from KkjColumn names (from DB) and cell values from KkjTargetValue records
- Edit-mode table renders editable input cells for each KkjColumn per row
- "Kelola Kolom" section lets admin add/rename/delete KkjColumns for the selected Bagian
- "Kelola Pemetaan Jabatan" section lets admin add/delete PositionColumnMappings for the selected Bagian
- Save button POSTs to `KkjMatrixSave` with the new DTO format (base fields + TargetValues array with KkjColumnId+Value pairs)
- `Views/CMP/Kkj.cshtml` renders target columns dynamically from `ViewBag.Columns` (a list of KkjColumn for the user's section)
- `CMPController.Kkj()` passes `KkjColumn` list and items with included TargetValues
- `dotnet build` succeeds with 0 errors

## Context

### Current KkjMatrix.cshtml state
The view uses a JavaScript `labelFieldKeys` array with 15 hardcoded `{ key: 'Label_*', prop: 'Target_*' }` entries. The `renderReadTable()` and `renderEditRows()` functions iterate this array. After Phase 89, this array is gone. Instead:
- `kkjBagians` JS variable still comes from ViewBag.Bagians but no longer has `Label_*` properties
- Each bagian now has a `Columns` navigation property (loaded by the controller)
- A new JS function fetches columns for the selected bagian via AJAX

### Current CMP/Kkj.cshtml state
The view has a hardcoded 15-column HTML table with explicit `<th>` headers and `<td>` cells referencing `Target_*` model properties (in Razor `@Model` iteration). After Phase 89, columns come from DB dynamically.

### Data flow for KkjMatrix.cshtml
The controller now passes:
- `Model` = `List<KkjMatrixItem>` where each item has `.TargetValues` (collection of `KkjTargetValue`) already included
- `ViewBag.Bagians` = `List<KkjBagian>` where each bagian has `.Columns` (collection of `KkjColumn`) already included

For the JS data:
- `kkjItems` JSON: serialize items with their TargetValues (need to include `KkjColumnId` and `Value` per cell)
- `kkjBagians` JSON: serialize bagians with their Columns (Id, Name, DisplayOrder per column)

### Data flow for CMP/Kkj.cshtml
The CMPController.Kkj() action needs to:
1. Determine the user's section/bagian
2. Load the KkjColumns for that bagian (ordered by DisplayOrder)
3. Load KkjMatrixItems for that bagian with TargetValues included
4. Pass columns via ViewBag.Columns and items via Model

function loadColumnPanel(bagianId) {
    var token = document.querySelector('input[name="__RequestVerificationToken"]').value;

            html = '<p class="text-muted small">Belum ada kolom. Klik "Tambah Kolom" untuk menambahkan.</p>';
        } else {
            html = '<table class="table table-sm table-bordered"><thead><tr><th>Urutan</th><th>Nama Kolom</th><th>Aksi</th></tr></thead><tbody>';

                    + '<td><input type="text" class="form-control form-control-sm col-name-panel-input" value="' + escHtml(col.Name) + '"></td>'
                    + '<td>'
                    + '<button class="btn btn-sm btn-outline-primary me-1 btn-save-col" title="Simpan"><i class="bi bi-check"></i></button>'
                    + '<button class="btn btn-sm btn-outline-danger btn-delete-col" title="Hapus"><i class="bi bi-trash"></i></button>'
                    + '</td></tr>';
            });
            html += '</tbody></table>';
        }
        document.getElementById('columnListContainer').innerHTML = html;

        // Wire save buttons
        document.querySelectorAll('.btn-save-col').forEach(function(btn) {
            btn.addEventListener('click', function() {
                var tr = btn.closest('tr');
                var colId = parseInt(tr.dataset.colId);
                var name = tr.querySelector('.col-name-panel-input').value.trim();
                var order = parseInt(tr.querySelector('.col-order-input').value) || 0;
                $.ajax({
                    url: '/Admin/KkjColumnSave',
                    type: 'POST',
                    contentType: 'application/json',
                    headers: { 'RequestVerificationToken': token },
                    data: JSON.stringify({ Id: colId, Name: name, DisplayOrder: order }),
                    success: function(resp) {
                        if (resp.success) {
                            // Update kkjBagians cache
                            var bagian = kkjBagians.find(function(b) { return b.Id === bagianId; });
                            if (bagian) {
                                var col = bagian.Columns.find(function(c) { return c.Id === colId; });
                                if (col) { col.Name = name; col.DisplayOrder = order; }
                            }
                            alert('Kolom berhasil disimpan.');
                            loadColumnPanel(bagianId);
                        } else {
                            alert('Error: ' + resp.message);
                        }
                    }
                });
            });
        });

        // Wire delete buttons
        document.querySelectorAll('.btn-delete-col').forEach(function(btn) {
            btn.addEventListener('click', function() {
                var tr = btn.closest('tr');
                var colId = parseInt(tr.dataset.colId);
                if (!confirm('Hapus kolom ini? Tidak dapat dihapus jika masih ada data nilai.')) return;
                $.ajax({
                    url: '/Admin/KkjColumnDelete',
                    type: 'POST',
                    data: { id: colId, __RequestVerificationToken: token },
                    success: function(resp) {
                        if (resp.success) {

                            loadColumnPanel(bagianId);
                            renderReadTable(document.getElementById('bagianFilter').value);
                        } else {
                            alert(resp.message);
                        }
                    }
                });
            });
        });
    });
}
