# Plan 93-01: CMP Code Review - Controller & Models Audit

**Goal:** Systematically review CMPController.cs and related models for bugs across all CMP pages (Index, Assessment, Records, Kkj, Mapping, Monitoring)

**Context:** Phase 93 is part of v3.2 Bug Hunting & Quality Audit. This plan follows Phase 92 Homepage Audit pattern: code review → identify bugs → fix → smoke test. Focus on finding bugs, NOT adding new features.

**Requirements Coverage:**
- CMP-01: Assessment page loads without errors for all roles
- CMP-02: Assessment monitoring detail page shows real-time data correctly
- CMP-03: Records page displays assessment history and training records with correct pagination
- CMP-04: KKJ Matrix page loads correctly with section-based filtering
- CMP-05: All CMP forms handle validation errors gracefully (no raw exceptions)
- CMP-06: CMP navigation flows work correctly (Create Assessment → Edit → Delete → Monitor)

## Tasks

### Task 1: Audit CMPController for null safety and error handling

<files>
Controllers/CMPController.cs
</files>

<action>
Read CMPController.cs line-by-line and identify potential bugs:

1. **Null safety check:** Search for all `.First()` calls without `.Any()` guards
   - Known locations: lines 80, 85, 927
   - Check if `availableBagians` can be empty before calling `.First()`

2. **Validation handling:** Find all POST actions and verify ModelState.IsValid checks
   - Search for `[HttpPost]` attributes
   - Verify each has `if (!ModelState.IsValid)` block
   - Identify actions missing validation

3. **Real-time monitoring:** Review IMemoryCache usage for live monitoring
   - Find cache key generation logic
   - Verify TTL is 10 seconds
   - Check cache miss handling

4. **Role-based filtering:** Verify user.RoleLevel usage in section filtering
   - Check Kkj and Mapping actions
   - Verify default RoleLevel fallback
   - Confirm null user handling
</action>

<verify>
Run grep commands to identify potential issues:
```bash
# Find all .First() calls (potential null safety issues)
grep -n "\.First()" Controllers/CMPController.cs

# Find all POST actions
grep -n "\[HttpPost\]" Controllers/CMPController.cs

# Find IMemoryCache usage
grep -n "IMemoryCache\|_cache\." Controllers/CMPController.cs

# Find ModelState validation checks
grep -n "ModelState.IsValid" Controllers/CMPController.cs
```
</verify>

<done>
Bug inventory created in `.planning/phases/93-cmp-section-audit/BUG_INVENTORY.md` with:
- List of null safety issues with line numbers
- POST actions missing validation
- Cache handling issues
- Severity ratings (Critical/High/Medium/Low)
- Categorization by fix type
</done>
