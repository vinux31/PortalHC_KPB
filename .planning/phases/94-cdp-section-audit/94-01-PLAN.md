<files>

</files>

<action>
Add Indonesian date formatting to all date displays in PlanIdp

Steps:
1. Add `@using System.Globalization` to top of `Views/CDP/PlanIdp.cshtml`
2. Find all date formatting calls (search for `.ToString("`, `.ToString("d`, `.ToString("D`)
3. Replace with `CultureInfo.GetCultureInfo("id-ID")` parameter
4. Example fix:
   ```csharp
   // Before
   @item.DeliveredDate.ToString("dd MMM yyyy")

   // After
   @item.DeliveredDate.ToString("dd MMM yyyy", CultureInfo.GetCultureInfo("id-ID"))
   ```
5. Add null checks for nullable DateTime properties:
   ```csharp
   @if (item.DeliveredDate.HasValue)
   {
       @item.DeliveredDate.Value.ToString("dd MMM yyyy", CultureInfo.GetCultureInfo("id-ID"))
   }
   ```
</action>
