### Task 2: Fix CoachCoacheeMapping bugs (Commit 2)

**Fix priorities:** Same as Task 1

**Common fixes to apply:**

**CSRF Token in AJAX:**
```javascript
// BEFORE (missing CSRF token)
fetch('/Admin/CoachCoacheeMappingAssign', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(payload)
});

// AFTER (with CSRF token)
fetch('/Admin/CoachCoacheeMappingAssign', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-Request-VerificationToken': document.querySelector('input[name="__RequestVerificationToken"]').value
    },
    body: JSON.stringify(payload)
});
```
