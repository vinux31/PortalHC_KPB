### Task 98-04-01: Fix critical IsActive filter gaps (DATA-01)

**Action:** Add missing IsActive filters to all user-facing queries identified in plan 98-01

**Fix pattern:**

For each missing IsActive filter gap from 98-01-ISACTIVE-AUDIT.md:

1. **Locate the query in controller:**
   ```csharp
   // BEFORE (missing filter):
   var workers = await _context.Users
       .OrderBy(u => u.FullName)
       .ToListAsync();

   // AFTER (with IsActive filter):
   var workers = await _context.Users
       .Where(u => u.IsActive)
       .OrderBy(u => u.FullName)
       .ToListAsync();
   ```

2. **Cross-entity IsActive checks (parent.IsActive):**
   ```csharp
   // BEFORE (orphan risk):
   var mappings = await _context.CoachCoacheeMappings
       .Where(m => m.CoachId == coachId)
       .ToListAsync();

   // AFTER (orphan prevention):
   var mappings = await _context.CoachCoacheeMappings
       .Include(m => m.Coach)
       .Include(m => m.Coachee)
       .Where(m => m.IsActive && m.Coach.IsActive && m.Coachee.IsActive)
       .ToListAsync();
   ```

3. **ShowInactive toggle pattern:**
   ```csharp
   // BEFORE (always shows inactive):
   var silabus = await _context.ProtonKompetensiList
       .Where(k => k.Bagian == bagian)
       .ToListAsync();

   // AFTER (toggle pattern):
   bool showInactive = Request.Query["showInactive"] == "true";
   var silabus = await _context.ProtonKompetensiList
       .Where(k => k.Bagian == bagian && (showInactive || k.IsActive))
       .ToListAsync();
   ```

### Task 98-04-03: Fix critical AuditLog gaps (DATA-03)

**Action:** Add missing AuditLog calls to all critical actions identified in plan 98-03

**Fix pattern:**

For each missing AuditLog gap from 98-03-AUDITLOG-AUDIT.md:

1. **Delete actions:**
   ```csharp
   // BEFORE (no audit trail):
   [HttpPost]
   [ValidateAntiForgeryToken]
   public async Task<IActionResult> DeleteWorker(int id)
   {
       var user = await _userManager.FindByIdAsync(id.ToString());
       if (user == null) return Json(new { success = false, message = "User not found" });

       var result = await _userManager.DeleteAsync(user);
       if (!result.Succeeded) return Json(new { success = false, message = "Delete failed" });

       return Json(new { success = true, message = "Worker deleted" });
   }

   // AFTER (with audit trail):
   [HttpPost]
   [ValidateAntiForgeryToken]
   public async Task<IActionResult> DeleteWorker(int id)
   {
       var currentUser = await _userManager.GetUserAsync(User);
       var user = await _userManager.FindByIdAsync(id.ToString());
       if (user == null) return Json(new { success = false, message = "User not found" });

       var result = await _userManager.DeleteAsync(user);
       if (!result.Succeeded) return Json(new { success = false, message = "Delete failed" });

       // AUDIT LOG
       await _auditLog.LogAsync(
           currentUser.Id,
           currentUser.FullName,
           "Delete",
           $"Deleted worker '{user.FullName}' (NIP: {user.NIP})",
           user.Id,
           "ApplicationUser"
       );

       return Json(new { success = true, message = "Worker deleted" });
   }
   ```

2. **Import actions:**
   ```csharp
   // BEFORE (no audit trail):
   ```
