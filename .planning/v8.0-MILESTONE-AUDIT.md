### Gap: CLEN-04 x AkhiriUjian

**Severity:** Medium-High

Phase 226 moved NomorSertifikat generation from CreateAssessment to SubmitExam. However, `AdminController.AkhiriUjian` and `AkhiriSemuaUjian` (HC force-close paths) call `GradeFromSavedAnswers` and set `IsPassed`/`Status = Completed` but **do not generate NomorSertifikat**.

**Impact:** If HC force-closes a session where the worker passed, `NomorSertifikat` stays null permanently.

**Fix:** Add cert generation block in AkhiriUjian/AkhiriSemuaUjian after `rowsAffected > 0`, using same pattern as `CMPController.SubmitExam:1394-1417`.
