BAGIAN 2 — CDP

CDP — Reviewer Chain

HC = Final Reviewer (Reviewer ke-3) di approval chain

SLIDE 9 / 23
Coachee Submit Upload evidence deliverable
→
1. Sr Supervisor Review pertama
→
2. Section Head Review kedua
→
3. HC Final Approve / Reject

⚠ Reject = Reset Chain

  • Reject di tahap mana pun = chain ulang dari awal
  • Coachee upload evidence baru
  • Sr Supervisor + Section Head review ulang
  • HC final review lagi

✅ Tidak Boleh Skip Chain

  • HC tunggu Sr Spv + Section Head approve
  • Sistem enforce urutan via endpoint
  • Tombol HC Approve baru aktif setelah Section Head approve
  • Batch HC Approve untuk efisiensi

BAGIAN 2 — CDP

Coaching Proton Dashboard

Monitoring global + drill ke coachee + bottleneck

SLIDE 10 / 23

🌐 Global View

  • Total coachee aktif lintas section
  • Distribusi status deliverable
  • Progress per Section / Unit / Track
  • Filter cascade: Section → Unit, Kategori → Sub-Kategori
  • Drill ke card coachee untuk detail deliverable + session

🚨 Tab Bottleneck Report

  • Deliverable stuck per coachee / unit
  • Filter: Pending Approval > X hari, Rejected, by Coach
  • Identifikasi akar: reviewer lambat, coachee pasif, scope unclear
  • Export untuk meeting eskalasi
💡 Daily routine HC: Buka Dashboard pagi, scan tab Bottleneck dulu. Identifikasi 3-5 deliverable terlama, eskalasi ke Section Head terkait.

BAGIAN 2 — CDP

Histori Proton + Export

Laporan periode + per coachee — Excel & PDF

SLIDE 11 / 23

📚 Histori per Periode

  • Riwayat coaching per periode Proton
  • Filter section, unit, track, periode
  • Drill detail per coachee — deliverable + session log + final assessment
  • Histori tetap accessible setelah periode selesai (audit trail)

📥 Export Options

  • Histori Proton — Excel rekap periode
  • Progress per Coachee — Excel atau PDF
  • Bottleneck Report — Excel stuck items
  • Coaching Tracking — Excel lengkap
  • Workload Summary — Excel beban coach
💡 Untuk laporan bulanan: Kombinasikan Histori Proton + Bottleneck + Workload. Tiga file ini sudah cover 80% kebutuhan reporting HC ke manajemen.
