2. XPAG-01: CreateAssessment Renewal Pre-fill

Status: OK — Pre-fill berfungsi lengkap sejak Phase 201/210. Tidak ada bug ditemukan.

Temuan Controller (AdminController.cs ~line 957-1063)

CreateAssessment GET action menangani dua skenario renewal:

Skenario Field yang di-set Status
Single renew (renewSessionId.Count == 1) model.Title, model.Category (langsung dari sourceSession.Category), model.ValidUntil, ViewBag.SelectedUserIds, ViewBag.RenewalSourceTitle, ViewBag.RenewalSourceUserName, ViewBag.RenewsSessionId OK
Bulk renew (renewSessionId.Count > 1) model.Title, model.Category, ViewBag.RenewalFkMap (JSON dict UserId→SessionId), ViewBag.RenewalFkMapType="session", ViewBag.SelectedUserIds (list), ViewBag.RenewalSourceTitle, ViewBag.RenewalSourceUserName OK
renewTrainingId path model.Title (dari cert.Judul), model.Category via MapKategori(), ViewBag.RenewalFkMap, ViewBag.RenewalFkMapType="training" OK

Temuan View (Views/Admin/CreateAssessment.cshtml)

  • Category dropdown: Menggunakan option value="@cat.Name" dengan selected="@(Model.Category == cat.Name)" — match dengan string yang disimpan di controller. Tidak ada mismatch.
  • RenewalFkMap hidden input: Ada pada line 112-115 dalam blok @if (ViewBag.RenewalFkMap != null).
  • RenewalFkMapType hidden input: Ada bersamaan dengan RenewalFkMap.
  • RenewalSourceUserName alert: Ditampilkan dalam blok IsRenewalMode pada line 64-66.
  • SelectedUserIds pre-select: JavaScript pada line 277 membaca ViewBag.SelectedUserIds dan men-set nilai multi-select atau menampilkan info peserta bulk.
  • Category change event trigger: Setelah DOMContentLoaded, category dispatch event 'change' untuk trigger sub-category cascade.
Catatan: Single-renew dari Assessment menggunakan model.Category = sourceSession.Category langsung (tanpa MapKategori) karena sourceSession.Category sudah berupa display name (misal "Mandatory HSSE Training"). MapKategori hanya digunakan saat cross-type: renew Training-ke-Assessment.

4. XPAG-03: CDP CertificationManagement Toggle Renewed Certs

Status: OK — Toggle berfungsi, state preserved saat AJAX reload. Tidak ada bug ditemukan.

Temuan View (Views/CDP/CertificationManagement.cshtml)

  • Toggle element: <input type="checkbox" id="toggle-renewed" role="switch"> dengan label "Tampilkan Riwayat Renewal" (line 138-139). Default unchecked = renewed rows tersembunyi.
  • applyRenewedToggle() function: Defined di line 195-202. Membaca state toggleRenewed.checked dan men-set style.display pada semua elemen .renewed-row. Saat checked: display="" (visible, opacity 0.5). Saat unchecked: display="none".
  • State preservation after AJAX reload: Dalam refreshTable(), setelah container.innerHTML = html, fungsi applyRenewedToggle() dipanggil di line 293. Toggle state dari toggle element dipertahankan karena element toggle sendiri tidak di-replace (hanya container tabel).
  • Toggle listener: toggleRenewed.addEventListener('change', applyRenewedToggle) di line 204.
  • Reset behavior: Reset button men-set toggleRenewed.checked = false di line 268 lalu memanggil refreshTable() yang kemudian applyRenewedToggle() — semua rows renewed kembali tersembunyi.

Temuan Partial View (Views/CDP/Shared/_CertificationManagementTablePartial.cshtml)

  • renewed-row class: Pada line 49: <tr class="@(row.IsRenewed ? "renewed-row" : "")" @(row.IsRenewed ? "style=\"display:none;\"" : "")> — menambahkan class dan inline style untuk initial hide.

Temuan CDPController BuildSertifikatRowsAsync (line 3200+)

IsRenewed dihitung menggunakan semua 4 FK kombinasi:

FK Set Query Target
AS.RenewsSessionId AssessmentSessions WHERE RenewsSessionId HasValue AND IsPassed == true renewedAssessmentSessionIds
TR.RenewsSessionId TrainingRecords WHERE RenewsSessionId HasValue renewedAssessmentSessionIds (gabung)
AS.RenewsTrainingId AssessmentSessions WHERE RenewsTrainingId HasValue AND IsPassed == true renewedTrainingRecordIds
TR.RenewsTrainingId TrainingRecords WHERE RenewsTrainingId HasValue renewedTrainingRecordIds (gabung)
Semua 4 FK kombinasi sudah dicek — konsisten dengan AdminController BuildRenewalRowsAsync.
