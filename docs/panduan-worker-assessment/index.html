Cara Mengikuti Assessment

Langkah demi langkah — dari menyalakan koneksi sampai melihat hasil & sertifikat.

Panduan untuk Anda yang baru pertama kali mengikuti assessment di Portal HC KPB. Assessment sudah dibuatkan tim HC dan ditugaskan ke akun Anda. Ikuti langkah berikut berurutan — tidak perlu pengalaman teknis.

Yang Perlu Disiapkan

Komputer / laptop yang terhubung internet.

Aplikasi Virtual Intranet Access (VPN untuk masuk jaringan internal Pertamina).

Email dan password login Anda.

Kode / Token assessment dari HC — hanya jika assessment meminta token (Langkah 6).

Belum punya akun / lupa password? Hubungi tim HC / admin. Akun dibuat oleh HC; Anda tidak bisa mendaftar sendiri.
Langkah Demi Langkah
1

Nyalakan Koneksi Intranet (VPN)

  1. Buka aplikasi Virtual Intranet Access.
  2. Login sesuai akun VPN Anda.
  3. Tunggu sampai status Connected / Terhubung.
Penting Website hanya bisa dibuka bila VPN terhubung. Kalau halaman tidak terbuka, cek dulu koneksi VPN ini.
2

Buka Website Portal HC

Buka browser (Chrome / Edge), ketik alamat ini, lalu tekan Enter:

http://10.55.3.3/KPB-PortalHC
3

Login ke Akun Anda

  1. Masukkan Email Anda.
  2. Masukkan Password Anda.
  3. Klik Login / Masuk.

Gagal login? Periksa huruf besar/kecil. Tetap gagal → hubungi HC/admin.

4

Buka Menu Assessment

Dari menu navigasi, buka halaman Assessment. Di sini muncul semua assessment yang perlu Anda kerjakan.

Assessment yang sudah selesai tidak lagi tampil di daftar ini.

5

Pilih Assessment & Pahami Warna Status

Bisa Mulai

Siap dikerjakan sekarang. Tombol Mulai aktif.

Upcoming

Belum waktunya. Tunggu jadwal tiba.

In Progress

Sedang/tertunda. Tekan Lanjutkan.

Completed

Sudah selesai. Tidak perlu dikerjakan lagi.

Pre-Test & Post-Test Kerjakan Pre-Test dulu sampai selesai. Post-Test baru terbuka setelahnya.
6

Mulai Assessment (Token Bila Diminta)

  1. Klik Mulai pada assessment berstatus hijau.
  2. Jika muncul permintaan Kode Akses / Token (ikon gembok 🔐):
    • Masukkan token dari HC.
    • Klik Verifikasi / Lanjut.
  3. Jika tidak diminta token, Anda langsung masuk ke soal.
Token salah? Muncul pesan “Token tidak valid”. Ketik ulang dengan teliti, atau minta token yang benar ke HC.
7

Kerjakan Soal

Jenis soal yang mungkin muncul:

  • Pilihan 1 jawaban — klik 1 lingkaran.
  • Pilihan banyak jawaban — centang lebih dari satu kotak.
  • Essay — ketik jawaban di kolom teks.

Berpindah soal:

  • Tombol Sebelumnya / Berikutnya untuk pindah halaman.
  • Nomor soal di samping bisa diklik. Nomor hijau = sudah dijawab.
Jawaban tersimpan otomatis Tidak ada tombol “Simpan”. Jika halaman ter-refresh atau koneksi sempat putus, jawaban tetap aman — tinggal lanjutkan.
Perhatikan waktu (timer) Waktu tetap berjalan walau browser ditutup. Jika habis, ujian otomatis dikumpulkan. Jangan menunda terlalu lama.
8

Periksa Jawaban Sebelum Mengumpulkan

  1. Klik Review / Periksa Jawaban.
  2. Muncul ringkasan semua soal & jawaban Anda.
  3. Baris merah = belum dijawab. Klik Kembali ke Ujian untuk melengkapi.

Tombol kumpulkan aktif setelah semua soal terjawab (kecuali waktu habis).

9

Kumpulkan (Submit) Ujian

  1. Klik Kumpulkan Ujian.
  2. Pada konfirmasi, klik Ya / Kumpulkan.
Tidak bisa dibatalkan Setelah dikumpulkan, jawaban tidak dapat diubah. Pastikan sudah yakin.
10

Lihat Hasil

Setelah submit, halaman hasil langsung tampil:

LULUS

Selamat, nilai memenuhi batas kelulusan.

TIDAK LULUS

Nilai belum memenuhi. Pelajari kembali materinya.

MENUNGGU PENILAIAN

Ada soal essay yang dinilai manual HC. Nilai & sertifikat muncul setelah HC selesai menilai.

11

Unduh Sertifikat

  1. Jika LULUS dan tersedia sertifikat, muncul tombol Download / Lihat Sertifikat.
  2. Klik untuk membuka, lalu simpan / cetak (PDF).

Jika status masih “Menunggu Penilaian”, sertifikat tersedia setelah HC menilai essay.

Kalau Ada Masalah

Website tidak mau terbuka

Cek koneksi Virtual Intranet Access (VPN) — harus Connected. Periksa juga alamat http://10.55.3.3/KPB-PortalHC sudah benar.

Tidak bisa login / lupa password

Periksa email & password. Tetap gagal → hubungi HC / admin untuk reset.

Assessment tidak muncul di daftar

Mungkin belum ditugaskan, belum waktunya (Upcoming), atau sudah selesai. Hubungi HC.

Token tidak valid

Ketik ulang dengan teliti. Minta token yang benar ke HC.

Waktu habis saat mengerjakan

Ujian otomatis dikumpulkan. Jawaban yang sudah diisi tetap dihitung.

Tidak sengaja keluar / “Keluar Ujian”

Jika sesi menjadi tidak bisa dilanjutkan, hubungi HC untuk mereset sesi ujian Anda.

Catatan Penting

  • Waktu berjalan di server. Menutup browser tidak menghentikan timer.
  • Jawaban tersimpan otomatis — aman walau koneksi sempat putus.
  • Submit bersifat final — tidak bisa diubah.
  • Satu akun untuk satu orang. Jangan bagikan login.
  • Butuh bantuan? Hubungi tim HC / admin.
