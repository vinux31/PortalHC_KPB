§2 Methodology

2.1 Sumber Data
  • Existing Gap (25) — derive dari index.html §10 Gap Analysis (8 fungsi + 8 sistem + 9 logic, dedup ke 25 unique).
  • Net-New Capability (12) — hasil benchmark industri (Open Badges, blockchain credential, expiry pipeline, skill matrix, peer endorsement, badge wallet, revocation list, multi-language, dynamic QR, SCORM, CPD, proctor mode).
  • Benchmark Vendor — Migas (Shell/Chevron/Exxon) + HRIS/LMS (Workday/SuccessFactors/Cornerstone/Moodle) + Cert Std (ISO 17024/IACET/Open Badges 1EdTech/Accredible/Credly).
  • Citation — WebSearch real-time tanggal 2026-05-26 untuk klaim faktual; URL + access date di §10.
2.2 Framework Prioritisasi — ICE

RICE (Reach·Impact·Confidence/Effort) tidak fit karena Reach hampir flat di context internal Pertamina KPB (semua user terjangkau). Kami pakai ICE = Impact × Confidence × Ease / 100.

  • Impact (1-10) — Dampak business / compliance / UX bila item diimplementasi.
  • Confidence (%) — Keyakinan solusi efektif (data + benchmark backing).
  • Ease (1-10) — Kebalikan effort. 10 = ½ PW, 1 = 8+ PW.

§5 Industry Benchmark

Tiga kategori benchmark: Migas/Energi (sektor sama), HRIS/LMS umum (best practice digital cert), Cert Standar Internasional (compliance framework). Per vendor: capability snapshot, gap Portal HC vs vendor, citation.

§5.1 — Migas / Energy

Konteks Regulatori Migas Indonesia
  • Permen ESDM (SKKNI Migas Wajib) — Badan usaha migas wajib mempekerjakan tenaga yang memiliki sertifikat kompetensi sesuai SKKNI. Acuan untuk uji kompetensi + sertifikasi profesi.
  • LSP PPSDM Migas — Satu-satunya LSP milik pemerintah untuk migas. 45 ruang lingkup sertifikasi; bidang hilir/pengilangan mencakup Crude Distilling Unit, Vacuum Distilling Unit, Operasi Boiler, Laboratorium Pengujian, dll.
  • BNSP / SKKNI — Body governing SKKNI nasional, validity sertifikat umumnya 3 tahun.
  • ISO 29001 / 14001 / 45001 — QMS migas mengarahkan competency-based personnel management; relevan ke ExxonMobil OIMS + Chevron OEMS framework.

Capability Snapshot:

  • Shell Learning Academy — competence-based programmes via hands-on roles, job tasks, formal training.
  • PetroSkills Alliance (founded 2001 by Shell + BP + OGCI) — competency-based training industri-wide, scope global.
  • 3-year Graduate Programme dengan job rotation + personalized coaching + leadership development.
  • Internal cert linked to career progression matrix per role (operator/engineer/manager).

Gap vs Portal HC KPB: Portal HC sudah role-based RBAC + assessment, tapi belum ada (a) job-rotation tracking, (b) coaching pairing formal mechanism, (c) graduate program flow distinct dari worker eksisting.

Source: shell.com Learning Academy + petroskills.com — access 2026-05-26

Capability Snapshot:

  • OEMS (launched 2004, updated 2018) — competency tied to risk + assurance + safeguards.
  • Refinery Formal Operator Training — wajib selesai dalam 3 tahun pertama kerja.
  • External verification via Lloyd's Register (LRQA) untuk align ISO 14001:2015 + ISO 45001:2018.
  • OEMS scope: workforce safety, process safety, reliability/integrity, environment, efficiency, security.

Gap vs Portal HC KPB: Portal HC essay-based assessment ada (CMP). Belum ada (a) external auditor verification model LRQA-style, (b) explicit linkage cert ↔ process safety risk register, (c) time-bound completion mandatory tracking (3-year rule Chevron).

Source: chevron.com OEMS + richmond.chevron.com Training Programs — access 2026-05-26

Capability Snapshot:

  • OIMS — 11-element framework, element 6 = "Personnel and Training".
  • Ties competency to incident prevention via PSM (Process Safety Management).
  • Independent verification by Exxon corporate auditor (internal third-party).
  • Documented training matrix per asset (refinery, platform, terminal).

Gap vs Portal HC KPB: Portal HC tidak terhubung dengan PSM/HSSE database (jika ada). Audit log generic (G-10) prerequisite untuk independent verification model OIMS.

Source: corporate.exxonmobil.com — Operations — access 2026-05-26

Capability Snapshot:

  • LSP PPSDM Migas — body sertifikasi resmi pemerintah Indonesia, 45 ruang lingkup.
  • Bidang hilir/pengilangan: CDU, VDU, Operasi Boiler, Lab Pengujian, Operasi Blending, dll.
  • Skema sertifikasi mengacu SKKNI sesuai Permen ESDM (wajib).
  • Open to public via pelatihan masyarakat + partner perusahaan migas (EMCL, dll.).

Implikasi Portal HC KPB: Portal HC dapat berperan sebagai (a) feeder data ke LSP PPSDM Migas untuk uji kompetensi eksternal, (b) tracker re-sertifikasi 3-tahun validity, (c) repository sertifikat LSP untuk worker. Integrasi data (CSV export atau API kalau tersedia) = potensial enhancement.

Source: ppsdmmigas.esdm.go.id + Permen ESDM SKKNI Migas — access 2026-05-26

  • Skills Cloud — ML-driven skill inference + verify dari workforce data, identifikasi skills gap, reskilling measurement.

Gap vs Portal HC KPB: Portal HC standalone — no ML skill inference, no career path engine, no GenAI integration. Skill matrix manual via form. Reference architecture untuk N-04 + N-05.
