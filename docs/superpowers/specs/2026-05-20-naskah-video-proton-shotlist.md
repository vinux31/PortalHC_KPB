## 3. Catatan Produksi

### Lower-third (text overlay)
Setiap fitur PROTON disebut by name = lower-third teks 2 detik untuk reinforcement:

| Time approx | Lower-third |
|-------------|-------------|
| 0:20 | "Coaching Proton · IDP · Histori PROTON" |
| 1:22 | "Fitur: IDP" |
| 1:25 | "Fitur: Deliverable" |
| 1:30 | "Fitur: Coaching Proton" |
| 1:42 | "Final Assessment" |
| 1:52 | "Histori PROTON" |
| 2:15 | "Sertifikasi" |

### Watermark
Logo Portal HC KPB di pojok kanan bawah, opacity 30%, sepanjang durasi.

### Color palette
- Hijau Pertamina (primary): `#009639`
- Biru Pertamina (secondary): `#0066B3`
- Putih, abu-abu netral untuk background
- Aksen kuning/emas (`#FFB81C`) untuk badge & sertifikat

### Typography
- Heading: SF Pro Display / Montserrat Bold
- Body: SF Pro Text / Open Sans Regular

### Pace narasi
~170 kata per menit. Total ~430 kata = 150 detik. Beri jeda 0.5s di antar-segmen untuk breathing room.

### Approval chain
1. Internal review HC (script + storyboard) — sebelum produksi
2. Draft cut review — setelah motion graphic + VO selesai
3. Final cut review — termasuk musik & color grading
4. Distribusi via Portal HC KPB (modul Sosialisasi) + channel internal
