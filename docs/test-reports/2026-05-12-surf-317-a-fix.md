## Root Cause Recap

`Hubs/AssessmentHub.cs SaveMultipleAnswer` inserts **1 `PackageUserResponse` row PER selected option**. MA question dengan 2 correct answers selected = 2 rows sharing same `PackageQuestionId`. `Controllers/CMPController.cs:2190` `packageResponses.ToDictionary(r => r.PackageQuestionId)` throws `ArgumentException: An item with the same key has already been added` → Results page MA = HTTP 500.

**Discovered:** Phase 317 Plan 01 FLOW K (workaround via DB-based score verify K5 + M5).
