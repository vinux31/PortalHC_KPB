🗂️ Master Data

Setup data fondasi: Worker, Coach mapping. Untuk role Admin / HC.

ADMIN 🧪 Drill di Praktik →

8. Worker CRUD Manual

  1. Login sebagai Admin → klik "Kelola Data" di navbar
  2. Pilih "Manage Workers" → list worker existing
  3. Tombol "Create Worker" → form isi NIP, Email, FullName, Role, Section
  4. Edit existing → klik baris → "Edit Worker"
  5. Untuk soft delete: klik "Deactivate" (worker jadi inactive, data preserved)
  6. Untuk hard delete: klik "Delete" (permanen, hilang dari DB — pakai dengan hati-hati)
  7. Reactivate: filter Inactive → klik worker → "Reactivate"
💡 Tip: Untuk audit retention, selalu pakai Deactivate dulu, bukan Delete. Hard delete cuma untuk data typo yang belum dipakai assessment.
ADMIN

9. Worker Bulk Import Excel

  1. "Manage Workers" → tombol "Import Excel"
  2. Klik "Download Template" → buka file Excel kosong dengan kolom NIP/Email/FullName/Role/Section
  3. Isi worker data sesuai template (10-100 row sekaligus OK)
  4. Upload file Excel via tombol "Choose File"
  5. Preview muncul → review data + validation result (warna merah = error, hijau = OK)
  6. Klik "Commit" kalau OK, atau "Cancel" kalau perlu fix Excel dulu
💡 Tip: Kalau preview merah, hover error untuk detail. NIP duplikat = paling sering. Cek di Excel sebelum upload ulang.
ADMIN 🧪 Drill di Praktik →

13. Coach-Coachee Mapping

  1. "Kelola Data" → "Coach-Coachee Mapping"
  2. Tombol "Add Mapping" → pilih Coach + Coachee dari dropdown
  3. Klik "Assign" → mapping tersimpan
  4. Cek workload coach via menu "Coach Workload" — threshold warning kalau coach load > limit
  5. Edit existing mapping: klik baris → ganti coach
  6. Bulk import via tombol "Import Mapping Excel" (alternatif manual)
💡 Tip: Threshold warning signal saja, bukan hard block. Coach overload tetap bisa di-assign manual, sistem cuma kasih warning.
