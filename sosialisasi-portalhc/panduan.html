🗂️ Master Data

Setup data fondasi: Worker, Coach mapping. Untuk role Admin / HC.

ADMIN 🧪 Drill di Praktik →

8. Worker CRUD Manual

  1. Login sebagai Admin → klik "Kelola Data" di navbar
  2. Pilih "Manage Workers" → list worker existing
  3. Tombol "Create Worker" → form isi NIP, Email, FullName, Role, Section
  4. Edit existing → klik baris → "Edit Worker"
  5. Untuk soft delete: klik "Deactivate" (worker jadi inactive, data preserved)
  6. Untuk hard delete: klik "Delete" (permanen, hilang dari DB — pakai dengan hati-hati)
  7. Reactivate: filter Inactive → klik worker → "Reactivate"
💡 Tip: Untuk audit retention, selalu pakai Deactivate dulu, bukan Delete. Hard delete cuma untuk data typo yang belum dipakai assessment.
ADMIN

9. Worker Bulk Import Excel

  1. "Manage Workers" → tombol "Import Excel"
  2. Klik "Download Template" → buka file Excel kosong dengan kolom NIP/Email/FullName/Role/Section
  3. Isi worker data sesuai template (10-100 row sekaligus OK)
  4. Upload file Excel via tombol "Choose File"
  5. Preview muncul → review data + validation result (warna merah = error, hijau = OK)
  6. Klik "Commit" kalau OK, atau "Cancel" kalau perlu fix Excel dulu
💡 Tip: Kalau preview merah, hover error untuk detail. NIP duplikat = paling sering. Cek di Excel sebelum upload ulang.
ADMIN 🧪 Drill di Praktik →

13. Coach-Coachee Mapping

  1. "Kelola Data" → "Coach-Coachee Mapping"
  2. Tombol "Add Mapping" → pilih Coach + Coachee dari dropdown
  3. Klik "Assign" → mapping tersimpan
  4. Cek workload coach via menu "Coach Workload" — threshold warning kalau coach load > limit
  5. Edit existing mapping: klik baris → ganti coach
  6. Bulk import via tombol "Import Mapping Excel" (alternatif manual)
💡 Tip: Threshold warning signal saja, bukan hard block. Coach overload tetap bisa di-assign manual, sistem cuma kasih warning.

📝 CMP Assessment

Workflow inti CMP — buat soal, sesi assessment, monitoring, grading, results.

ASESOR 🧪 Drill di Praktik →

19. Manage Package Questions (Bank Soal)

  1. "Kelola Data" → "Manage Package Questions" → pilih package
  2. "Create Question" → isi: text soal, type (MC/Essay/Interview), choices, kunci jawaban
  3. Edit existing → klik soal → "Edit Question"
  4. Delete kalau ada typo (sebelum dipakai assessment)
💡 Tip: Untuk tipe MC, kunci jawaban wajib di-set sebelum publish. Tanpa kunci, auto-grade gagal.
ASESOR

20. Import Questions Excel (Bulk)

  1. Di Manage Package Questions → "Import Excel"
  2. Download template → isi soal MC bulk
  3. Upload → preview validasi → commit
💡 Tip: Per package, jangan campur tipe soal (MC + Essay) di-import bareng. Bikin batch terpisah biar template tidak conflict.
ASESOR

21. Manage Assessment (List + Tabs)

  1. "Kelola Data" → "Manage Assessment"
  2. 3 tab tersedia: Assessment (aktif), Training (record manual), History (archived)
  3. Filter by date range, status, package
  4. Klik baris → detail assessment + actions (Edit, Monitor, Export Results)
ASESOR 🧪 Drill di Praktik →

22. Create Assessment — Single Mode

  1. "Manage Assessment" → tombol "Create Assessment"
  2. Form: nama sesi, package, kategori (OJ/IHT/dll), schedule date, durasi
  3. Pilih mode: Single (1 sesi, MC auto-grade)
  4. Assign users — pilih dari dropdown atau bulk by section
  5. Optional: set passing score, manual grading toggle untuk Essay
  6. Save → assessment masuk list dengan status "Scheduled"
💡 Tip: Mode "Mixed" = MC auto + Essay manual grading. Pilih ini kalau soal bercampur.
ASESOR 🧪 Drill di Praktik →

23. Create Assessment — Pre-Post Mode

  1. "Create Assessment" → pilih mode Pre-Post
  2. Setup Pre group (assessment sebelum training)
  3. Setup Post group (assessment sesudah training) — soal sama atau ekivalen
  4. Pre-Post pair link otomatis via PrePostGroupId
  5. Save → 2 sesi terbuat, linked untuk gain score analysis
💡 Tip: Pre dan Post harus pakai soal level kompetensi sama. Beda soal → gain score tidak valid.
ASESOR LIVE DEMO

25. Assessment Monitoring (Live)

  1. "Manage Assessment" → klik sesi yang sedang berjalan → "Monitor"
  2. Real-time view: list taker, status (Not Started / In Progress / Submitted)
  3. Refresh otomatis tiap 30 detik (atau manual via tombol "Refresh")
  4. Untuk Essay/Interview: klik "Grade" → manual grading per submission
📺 Catatan: Live demo dinamis — paling baik dilihat di portal real saat acara. Mockup tidak menangkap esensi real-time.
ASESOR 🧪 Drill di Praktik →

26. Manual Grading (Essay / Interview)

  1. "Monitor" sesi → tab "Pending Grading" → list submission Essay/Interview
  2. Klik 1 submission → view jawaban + rubric
  3. Input skor (0-100) + feedback text
  4. Save → status berubah ke "Graded", masuk hasil agregat
💡 Tip: Konsistensi penilaian penting. Sebelum grading batch, baca rubric + sample 2-3 submission dulu untuk kalibrasi.
SEMUA ROLE (terkait)

29. Results Page (MA Mixed Mode)

  1. "Manage Assessment" → klik sesi → "View Results"
  2. List taker dengan skor, grade, pass/fail status
  3. Untuk Pre-Post: tampil gain score (selisih Post − Pre)
  4. Untuk Mixed mode (MC auto + Essay manual): skor total agregat
  5. Export tombol → Excel detail per taker
SEMUA ROLE

31. Certificate PDF Export

  1. "View Results" → klik taker yang Pass → tombol "Generate Certificate"
  2. PDF certificate ter-generate (template Pertamina branding)
  3. Download / Print langsung dari browser
  4. Atau bulk: tombol "Export All Certificates" → ZIP file
💡 Tip: Certificate auto-include valid until date (renewal). Cek expiry dashboard untuk renewal alert.
