⚡ Quick Start — Daily Ops

4 fitur dasar untuk semua role. Mulai di sini kalau baru kenal portal.

DAILY · SEMUA ROLE 🧪 Drill di Praktik →

1. Login ke Portal

  1. Buka URL Dev http://10.55.3.3/KPB-PortalHC di Chrome/Edge
  2. Halaman login muncul — input Email kantor + Password
  3. Klik tombol Sign In
  4. Berhasil → masuk ke Dashboard
  5. Lupa password? Hubungi admin (Tim IT KPB)
💡 Tip: Bookmark URL di browser. AD/LDAP belum aktif — saat ini login pakai form email/password biasa.
DAILY · SEMUA ROLE

2. Navigasi Dashboard + Top Navbar

  1. Setelah login, landing di Dashboard
  2. Top navbar punya menu sesuai role kamu (role-based)
  3. Avatar pojok kanan = dropdown Profile + Settings + Logout
  4. Notification bell di sebelah avatar = alert sistem
  5. Icon ? di navbar = link ke Panduan portal lengkap
💡 Tip: Menu "Kelola Data" hanya muncul kalau role kamu = Admin. Kalau tidak muncul, bukan bug — cek badge role samping nama.
SELF-HELP · SEMUA ROLE

3. Buka Panduan via Icon ?

  1. Klik icon ? di top navbar
  2. Buka halaman index 9 panduan HTML (Admin, Assessment, Coaching, dll)
  3. Pilih topik yang dibutuhkan
  4. Search by keyword di halaman index kalau panduan banyak
💡 Tip: Bookmark URL documents/guides/ kalau sering buka panduan.
DAILY · SEMUA ROLE

4. Notification Bell

  1. Klik icon 🔔 di top navbar (samping avatar)
  2. Dropdown muncul dengan list notifikasi terbaru
  3. Badge angka = jumlah unread
  4. Klik notifikasi → navigate ke halaman terkait (assessment, coaching, dll)
  5. Notifikasi otomatis di-mark read setelah dibuka
💡 Tip: Cek bell setiap login — notifikasi assessment baru atau follow-up coaching biasanya muncul di sini.
