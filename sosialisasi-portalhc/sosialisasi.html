Apa Itu HC Portal KPB?

Sistem informasi berbasis web untuk Tim Human Capital Kilang Pertamina Balikpapan dalam mengelola kompetensi dan pengembangan pekerja secara terintegrasi, transparan, dan data-driven.

🎯

Terintegrasi

Semua modul HC dalam 1 portal

🔍

Transparan

Audit trail + role-based access

📈

Data-Driven

Analytics + report real-time

3 Pilar Manfaat

Nilai utama PortalHC untuk operasional KPB

⚙️

1. Efisiensi Operasional

Bulk import worker, auto-grading MC, certificate PDF instant. Tugas yang dulu 1 hari, sekarang 1 jam.

🔒

2. Compliance & Audit

AuditLog otomatis tiap action, soft delete preserve history, export ke Excel untuk audit eksternal.

📊

3. Insight Pengembangan

Dashboard fail-rate, gain score Pre-Post, coaching progress — manajemen punya data untuk decide intervention.

Struktur Role Pengguna

6 level akses (10 role total) — makin tinggi tangga, makin luas authority

Level 1AdminFull system control
Level 2HCHC operational lead
Level 3Direktur · VP · ManagerManagement (3 role)
Level 4SectionHead · SrSupervisorSection supervisory (2 role)
Level 5Coach · SupervisorCoaching (2 role)
Level 6CoacheeOperational (default)

Modul CMP

Competency Management Program — assessment & sertifikasi kompetensi pekerja

📝 Question & Assessment

  • Question bank per package + bulk import Excel
  • Create Assessment single / Pre-Post / Mixed mode
  • Schedule + assign user + grading auto/manual
  • Live monitoring sesi berjalan

🏆 Result & Certification

  • Results page MA mixed (Manual + Auto)
  • Pre-Post gain score analysis
  • Certificate PDF auto-generate
  • Renewal tracking + ExpiringSoon alert

Modul CDP

Continuous Development Program — coaching, IDP, sertifikasi pekerja

👥

Coaching Proton

Manage coaching session, submit deliverable, evidence approval flow coach-coachee

🎯

Plan IDP (Individual Development Plan)

Setup target kompetensi per worker, silabus-based mapping, monitor progress

🔗

Coach-Coachee Mapping

Assign coach ke coachee, threshold workload monitoring, bulk import Excel

Coaching Proton Journey

Perjalanan pengembangan pekerja Tahun 1 → Tahun 3

1
Tahun 1 — Fundamental: Onboarding, basic kompetensi, IDP awal, coach mapping
2
Tahun 2 — Develop: Advanced kompetensi, deliverable submission, mid-cycle assessment
3
Tahun 3 — Certify: Final assessment, sertifikasi, transition ke role baru
