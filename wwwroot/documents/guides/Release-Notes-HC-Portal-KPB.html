1. Anti-Copy Protection

Security Semua Peserta Ujian

Fitur perlindungan konten soal ujian dari aktivitas copy-paste. Selama ujian berlangsung, peserta tidak dapat menyalin teks soal maupun opsi jawaban.

Manfaat

  • Mencegah kebocoran soal melalui copy-paste
  • Menjaga integritas dan kerahasiaan bank soal
  • Tidak mengganggu pengalaman mengerjakan ujian — perlindungan bersifat silent

Cara Kerja

Perlindungan diterapkan secara otomatis pada area soal dan opsi jawaban saat ujian dimulai:

  • Keyboard shortcut diblokir: Ctrl+C, Ctrl+A, Ctrl+U, Ctrl+S, Ctrl+P
  • Aksi clipboard dinonaktifkan: Copy, Cut, Paste, klik kanan, drag & drop
  • Seleksi teks dinonaktifkan pada area soal (CSS user-select: none)
  • Tombol navigasi, input jawaban, dan kontrol lainnya tetap berfungsi normal
Catatan
Perlindungan hanya aktif pada area soal & opsi jawaban. Area lain seperti tombol navigasi dan input jawaban essay tetap dapat digunakan secara normal.

2. Maintenance Mode

Operasional Admin HC

Mode pemeliharaan sistem yang memungkinkan Admin menonaktifkan akses pengguna sementara — baik untuk keseluruhan sistem maupun modul tertentu.

Manfaat

  • Melakukan update atau migrasi data tanpa gangguan dari pengguna
  • Menampilkan pesan informatif kepada pengguna tentang estimasi waktu selesai
  • Dapat ditargetkan ke modul spesifik (CMP, CDP, Proton, dll.) tanpa mematikan seluruh sistem

Cara Kerja

  1. Admin membuka menu Admin → Maintenance
  2. Aktifkan maintenance dengan mengisi:
    • Pesan: Informasi yang ditampilkan ke pengguna
    • Estimasi Selesai: Waktu perkiraan maintenance berakhir
    • Scope: Pilih "Semua" atau modul tertentu (CMP, CDP, Proton, Admin, Home)
  3. Klik Aktifkan — pengguna biasa akan melihat halaman maintenance
  4. Admin & HC tetap dapat mengakses sistem selama maintenance aktif
Perhatian
Hanya role Admin yang dapat mengaktifkan/menonaktifkan maintenance mode. Role HC dapat mengakses sistem saat maintenance aktif, tetapi tidak dapat mengubah konfigurasi maintenance.
Halaman konfigurasi Maintenance Mode — pilih cakupan, pesan, dan estimasi selesai

3. User Impersonation

Admin Tool Admin

Fitur yang memungkinkan Admin untuk melihat sistem dari sudut pandang pengguna lain tanpa perlu mengetahui password mereka. Berguna untuk troubleshooting masalah yang dilaporkan pengguna.

Manfaat

  • Troubleshooting lebih cepat — lihat persis apa yang dilihat pengguna
  • Aman: mode read-only, tidak bisa mengubah data atas nama pengguna lain
  • Auto-expire setelah 30 menit untuk keamanan

Cara Kerja

Terdapat dua mode impersonation:

ModeKeteranganKegunaan
Simulasi Role Melihat sistem sebagai role tertentu (HC, Coach, dll.) Verifikasi tampilan menu & akses per role
Simulasi User Melihat sistem sebagai user spesifik Troubleshoot masalah user tertentu
  1. Buka menu Admin → Impersonate
  2. Pilih mode: Simulasi Role atau Simulasi User
  3. Banner kuning muncul di atas halaman menandakan mode aktif
  4. Klik Stop Impersonation di banner untuk kembali ke akun Admin
Keamanan
Selama impersonation, semua aksi tulis (POST/PUT/DELETE) diblokir secara otomatis. Admin hanya dapat melihat data, tidak dapat mengubah apapun atas nama pengguna lain.
Halaman Impersonate — pilih mode View As HC, View As User, atau cari user spesifik

4. Tipe Assessment Pre-Post Test

Assessment HC

Tipe assessment baru yang mengukur efektivitas training dengan membandingkan hasil Pre-Test (sebelum training) dan Post-Test (sesudah training) secara otomatis.

Manfaat

  • Mengukur dampak training secara kuantitatif
  • Sistem otomatis membuat dua sesi ujian (Pre & Post) dari satu kali setup
  • Terintegrasi dengan laporan Gain Score untuk analisis peningkatan per peserta

Cara Kerja

  1. Buat assessment baru, pilih tipe Pre-Post Test
  2. Sistem otomatis membuat dua session:
    • Pre-Test — diberikan sebelum training dimulai
    • Post-Test — diberikan setelah training selesai
  3. Soal yang digunakan sama untuk kedua sesi
  4. Hasil perbandingan dapat dilihat di menu Analytics → Gain Score
Terintegrasi
Data Pre-Post Test secara otomatis tersedia di laporan Gain Score (Fitur 7), menampilkan peningkatan skor per peserta dan per elemen teknis.
Form Buat Assessment — dropdown Tipe Assessment dengan pilihan Standard dan Pre-Post Test

5. Tipe Soal MC / MA / Essay

Assessment HC

Dukungan tiga tipe soal dalam assessment untuk fleksibilitas evaluasi yang lebih lengkap.

Tipe Soal

TipeKeteranganPenilaian
Multiple Choice (MC) Pilihan ganda, satu jawaban benar. Peserta memilih dengan radio button. Otomatis
Multiple Answer (MA) Pilihan ganda, lebih dari satu jawaban benar. Label "Pilih semua yang benar" ditampilkan. Otomatis
Essay Jawaban uraian bebas dengan batas karakter (default 2000). Dilengkapi rubrik penilaian. Manual oleh HC

Cara Kerja

  • Saat membuat soal di package, pilih Tipe Soal (MC/MA/Essay)
  • Untuk soal Essay, isi Rubrik Penilaian sebagai panduan scoring
  • Peserta melihat badge tipe soal saat mengerjakan ujian
  • Soal MC & MA dinilai otomatis; soal Essay perlu penilaian manual oleh HC
  • Character counter real-time ditampilkan untuk soal Essay
Info
Assessment yang mengandung soal Essay akan menampilkan indikator Essay Pending pada dashboard HC sampai semua jawaban essay dinilai.
Halaman Manage Questions — daftar soal dengan badge tipe, dan form Tambah Soal dengan dropdown Pilihan Ganda (MC) / Multiple Answer (MA) / Essay

6. Mobile Optimization Exam

UX Semua Peserta Ujian

Halaman ujian kini dioptimasi untuk perangkat mobile sehingga peserta dapat mengerjakan ujian dengan nyaman melalui smartphone.

Manfaat

  • Peserta dapat mengerjakan ujian kapan saja, di mana saja melalui HP
  • Navigasi soal mudah dijangkau dengan satu tangan (bottom navigation)
  • Layout responsif — otomatis menyesuaikan ukuran layar

Fitur Mobile

  • Sticky Footer Navigation: Tombol Previous, Next, Navigator, dan Submit selalu terlihat di bawah layar
  • Question Navigator Drawer: Panel geser untuk navigasi ke soal tertentu (offcanvas)
  • Responsive Layout: Sidebar navigasi soal otomatis tersembunyi di layar kecil, digantikan drawer
  • Adaptive Content: Jumlah soal per halaman disesuaikan untuk mobile
  • Touch-Friendly: Tombol dan area klik diperbesar untuk kemudahan sentuh

7. Item Analysis & Gain Score Report

Analytics HC

Laporan analitik untuk mengevaluasi kualitas soal (Item Analysis) dan efektivitas training (Gain Score) secara mendalam.

Item Analysis

Menganalisis setiap butir soal berdasarkan respons peserta:

MetrikKeterangan
Difficulty Index Tingkat kesulitan soal (0.0 – 1.0). Nilai tinggi = soal mudah.
Discrimination Index Kemampuan soal membedakan peserta pandai dan kurang pandai (Kelley D-index).
Distractor Analysis Distribusi pilihan jawaban — berapa persen memilih opsi A, B, C, D.
Peringatan Low-N
Jika jumlah responden kurang dari 30 orang, sistem menampilkan peringatan Low-N. Hasil analisis dengan responden sedikit mungkin kurang reliabel.

Gain Score

Mengukur peningkatan skor dari Pre-Test ke Post-Test. Tersedia dalam dua tampilan:

  • Per Peserta: Nama, NIP, Section, Skor Pre, Skor Post, Gain Score
  • Per Elemen Teknis: Rata-rata skor Pre, Post, dan Gain per elemen
  • Per Grup: Perbandingan rata-rata antar grup/kelas training
Tab Item Analysis — pilih assessment untuk melihat Difficulty Index, Discrimination Index, dan Distractor Analysis
Tab Gain Score Report — toggle Per Pekerja / Per Elemen Kompetensi dengan opsi Export

8. Coach Workload

Coaching Admin HC

Dashboard untuk memantau dan mengelola beban kerja coach sehingga distribusi coachee merata dan tidak ada coach yang kelebihan beban.

Manfaat

  • Visibilitas langsung terhadap jumlah coachee per coach
  • Peringatan otomatis untuk coach yang mendekati atau melampaui batas kapasitas
  • Rekomendasi penyeimbangan workload otomatis

Fitur Utama

KomponenKeterangan
Dashboard Metrics Total Coach Aktif, Total Coachee, Rasio Rata-rata, Jumlah Overloaded
Threshold Setting Batas maksimum coachee per coach (default: 5) dan batas warning (default: 4)
Reassignment Saran otomatis untuk redistribusi coachee dari coach overloaded
Filter & Export Filter by section, export ke Excel
Hak Akses
HC dapat melihat dashboard workload. Hanya Admin yang dapat mengubah threshold setting.
Dashboard Coach Workload — metric cards, grafik beban, tabel detail, dan saran penyeimbangan

9. Budget Training

Training HC

Modul pengelolaan anggaran training yang mencakup pencatatan rencana dan realisasi budget, lengkap dengan visualisasi dan perbandingan.

Manfaat

  • Tracking budget rencana vs realisasi dalam satu tampilan
  • Visualisasi chart untuk distribusi budget per kategori dan tipe
  • Import data budget dari file CSV/Excel

Fitur Utama

Dua Tab Utama:

  • Data Budget: Tabel daftar item budget dengan filter (tahun, tipe, kategori, pencarian). Mendukung create, import, dan export.
  • Ringkasan: Summary metrics (Total Rencana, Total Realisasi, Variance) dan chart visualisasi:
    • Chart perbandingan Rencana vs Realisasi per kategori
    • Chart distribusi budget per tipe
    • Top spending items

Aksi yang tersedia:

  • Create: Tambah item budget satu per satu
  • Import: Upload file CSV atau Excel untuk input massal
  • Export: Download data budget ke Excel
Tab Data Budget — tabel item budget dengan filter, sorting, dan tombol Export/Import/Tambah
Tab Ringkasan — summary cards, chart Budget vs Realisasi per Kategori, Distribusi per Type, dan Top 10 Item
