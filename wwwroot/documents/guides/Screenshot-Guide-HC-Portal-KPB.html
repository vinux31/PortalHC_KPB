1. Akun & Pengaturan

Halaman login, akses ditolak, dashboard utama, profil, dan pengaturan akun.

1.1 Halaman Login
Halaman awal untuk masuk ke HC Portal KPB.
Publik Tidak memerlukan login

Langkah Penggunaan

  1. Masukkan NIP pada kolom pertama.
  2. Masukkan Password pada kolom kedua.
  3. Klik tombol Masuk untuk login ke sistem.
  4. Jika berhasil, Anda akan diarahkan ke halaman Dashboard.
Tombol / ElemenFungsi
MasukMemproses login dengan NIP dan password yang dimasukkan
Informasi
Jika lupa password, hubungi Admin atau HC untuk melakukan reset password.
1.2 Access Denied (Akses Ditolak)
Halaman yang muncul ketika pengguna tidak memiliki izin akses.
Semua

Halaman ini muncul ketika pengguna mencoba mengakses fitur yang tidak sesuai dengan role-nya. Misalnya, pekerja biasa mencoba membuka halaman Admin.

Perhatian
Jika Anda merasa seharusnya memiliki akses, hubungi Admin untuk pengecekan role Anda.
1.3 Dashboard
Halaman utama setelah login. Menampilkan ringkasan, notifikasi, dan akses cepat ke modul.
Semua Semua pengguna yang sudah login

Isi Halaman

  • Sapaan waktu — Menampilkan salam (Selamat Pagi/Siang/Sore) beserta nama pengguna.
  • Event mendatang — Daftar coaching dan assessment yang dijadwalkan.
  • Progress bar — Ringkasan kemajuan CDP, Assessment, dan Coaching.
  • Alert sertifikat (Admin/HC) — Peringatan sertifikat yang akan kedaluwarsa.
Tombol / ElemenFungsi
Navbar: CMPNavigasi ke modul Competency Management Portal
Navbar: CDPNavigasi ke modul Career Development Portal
Navbar: Kelola DataNavigasi ke modul Admin (hanya tampil untuk Admin/HC)
Navbar: PanduanNavigasi ke halaman panduan penggunaan
Notification BellMenampilkan notifikasi terbaru (coaching, assessment, dll.)
Profile MenuAkses ke Profil, Pengaturan, dan Logout
1.4 Profil Pengguna
Menampilkan data pribadi pengguna secara read-only.
Semua

Informasi yang Ditampilkan

  1. Nama Lengkap, NIP, dan Email
  2. Jabatan, Direktorat, Seksi, dan Unit Kerja
  3. Nomor Telepon dan Role dalam sistem
Tips
Untuk mengubah data profil, gunakan halaman Pengaturan (Settings). Data seperti NIP dan Role hanya bisa diubah oleh Admin.
1.5 Pengaturan Akun
Halaman untuk mengubah password, profil, dan konfigurasi tanda tangan digital (PSign).
Semua

Langkah Penggunaan

  1. Pilih tab yang ingin diubah: Edit Profil, Ubah Password, atau PSign.
  2. Edit Profil — Ubah nama, email, nomor telepon, lalu klik Simpan.
  3. Ubah Password — Masukkan password lama, password baru, dan konfirmasi, lalu klik Ubah Password.
  4. PSign — Upload tanda tangan digital untuk digunakan pada sertifikat/dokumen.
Tombol / ElemenFungsi
SimpanMenyimpan perubahan profil
Ubah PasswordMemproses perubahan password
Upload PSignMengunggah file tanda tangan digital

2.2 Dokumen KKJ
Viewer dokumen Kerangka Kompetensi Jabatan per bagian dalam format tab.
Semua

Halaman viewer dokumen Kerangka Kompetensi Jabatan. Menampilkan file KKJ aktif yang dikelompokkan per bagian (departemen) dalam format tab.

  1. Pilih tab bagian untuk melihat file KKJ dari departemen tersebut.
  2. Klik Download pada file yang ingin diunduh (format PDF atau Excel).
2.3 Assessment
Daftar assessment yang ditugaskan kepada pekerja.
Semua

Langkah Penggunaan

  1. Lihat daftar assessment yang ditugaskan beserta status, jadwal, dan waktu ujian.
  2. Klik Mulai Ujian pada assessment yang berstatus Belum Dikerjakan dan sudah masuk jadwal.
  3. Untuk assessment yang sudah selesai, klik Lihat Hasil untuk melihat skor.
TombolFungsi
Mulai UjianMemulai sesi ujian assessment (hanya aktif saat jadwal terbuka)
Lihat HasilMelihat hasil dan skor assessment yang sudah dikerjakan
VerifikasiProses verifikasi identitas sebelum memulai ujian
Perhatian
Ujian memiliki batas waktu. Pastikan koneksi internet stabil sebelum memulai ujian.

2.4 Halaman Ujian (Start Exam)

Semua Pekerja yang ditugaskan

Halaman ujian aktif dengan tampilan soal, pilihan jawaban, timer, dan progress tracking.

  1. Baca soal yang ditampilkan dengan seksama.
  2. Pilih jawaban dengan mengklik opsi jawaban (A/B/C/D/E).
  3. Gunakan tombol Sebelumnya / Selanjutnya untuk navigasi antar soal.
  4. Perhatikan timer di bagian atas — ujian otomatis berakhir saat waktu habis.
  5. Klik Selesai & Review untuk meninjau jawaban sebelum submit.
TombolFungsi
SebelumnyaKembali ke soal sebelumnya
SelanjutnyaLanjut ke soal berikutnya
Nomor SoalNavigasi langsung ke soal tertentu (panel samping)
Selesai & ReviewMenuju halaman ringkasan jawaban sebelum submit final

2.5 Ringkasan Ujian (Exam Summary)

Semua

Halaman review sebelum submit final. Menampilkan daftar semua soal beserta jawaban yang dipilih.

  1. Periksa daftar jawaban — soal yang belum dijawab ditandai.
  2. Klik Edit pada soal tertentu untuk mengubah jawaban.
  3. Klik Submit Ujian untuk mengirim jawaban secara final.
TombolFungsi
EditKembali ke soal tertentu untuk mengubah jawaban
Submit UjianMengirim semua jawaban (tidak bisa diubah setelahnya)

2.6 Hasil Assessment (Results)

Semua

Menampilkan hasil setelah ujian selesai: skor total, status lulus/tidak lulus, dan detail jawaban per soal.

  1. Lihat skor total dan status kelulusan di bagian atas.
  2. Scroll ke bawah untuk melihat detail jawaban per soal (benar/salah).
  3. Jika lulus, klik Lihat Sertifikat untuk melihat dan mengunduh sertifikat.

2.7 Sertifikat (Certificate)

2.10 Records Tim (Records Team)
Tampilan records untuk beberapa pekerja sekaligus (perspektif tim/unit).
Admin HC

Tampilan records untuk beberapa pekerja sekaligus (perspektif tim/unit). Mendukung export data ke Excel.

2.11 Analytics Dashboard
Dashboard analitik dengan grafik dan statistik performa assessment.
Admin HC

Isi Halaman

  1. Fail Rate per Bagian & Kategori — Persentase kegagalan assessment per bagian dan kategori.
  2. Trend Assessment — Grafik tren lulus/gagal per bulan untuk analisis waktu.
  3. Elemen Teknis Breakdown — Analisis avg/min/max scoring per elemen teknis.
  4. Gain Score Analysis — Perbandingan skor Pre-Test vs Post-Test untuk assessment Pre-Post.
  5. Item Analysis — Difficulty Index, Discrimination Index, dan analisis Distractors per soal.
  6. Multi-level Filtering — Filter bertingkat: bagian, unit, kategori, sub-kategori dengan drill-down.
  7. Export Excel — Unduh seluruh data analytics ke format Excel.

2.12 Budget Training

Admin HC

Halaman pengelolaan anggaran pelatihan per bagian dan unit kerja. Diakses via CMP → Budget Training.

  1. Summary Cards — Total anggaran, realisasi, dan persentase penyerapan.
  2. Filter — Saring data berdasarkan tahun, bagian, unit, dan kategori pelatihan.
  3. Inline Edit — Klik cell realisasi untuk langsung mengubah nilai tanpa membuka form.
  4. Bulk Operations — Pilih beberapa baris lalu gunakan bulk delete atau bulk update.
  5. Import Excel — Upload data anggaran dari file Excel.
  6. Export Excel — Unduh data anggaran ke file Excel.
  7. Chart — Grafik perbandingan anggaran vs realisasi per kategori.

2.13 Sertifikasi Management

Admin HC

Halaman manajemen sertifikat kompetensi pekerja. Diakses via CMP → Sertifikasi.

  1. Grouped View — Daftar sertifikat dikelompokkan per jenis sertifikasi.
  2. Filter & Search — Cari berdasarkan nama pekerja, jenis sertifikat, atau status.
  3. Detail Worker — Klik nama pekerja untuk melihat semua sertifikat yang dimiliki.
  4. Masa Berlaku — Status sertifikat: Active, Expiring Soon (30 hari), Expired.
  5. Notifikasi Expired — Sistem otomatis mengirim notifikasi 30 hari sebelum kadaluarsa.

3.X Coach Workload

Admin HC

  4. Rebalancing Suggestions — Rekomendasi perpindahan coachee untuk menyeimbangkan beban.
  5. Export Report — Unduh laporan workload ke Excel.

3.X Proton History (Timeline)

Semua

  3. Export Histori — Unduh riwayat coaching ke file Excel.
  4. Coaching Transcript Export — Export sesi coaching ke PDF atau Excel.
