1. Akun & Pengaturan

Halaman login, dashboard utama, profil, dan pengaturan akun.

1.1 Halaman Login
Halaman awal untuk masuk ke HC Portal. Pengguna memasukkan NIP dan password untuk mengakses sistem.
1.2 Dashboard
Halaman utama setelah login. Menampilkan ringkasan informasi, notifikasi, dan akses cepat ke modul CMP, CDP, dan Admin.
1.3 Profil Pengguna
Halaman profil menampilkan data pribadi pengguna seperti nama, NIP, jabatan, unit kerja, dan informasi lainnya.
1.4 Pengaturan Akun
Halaman pengaturan untuk mengubah password dan preferensi akun pengguna.

2. CMP — Competency Management Portal

Modul pengelolaan kompetensi: assessment, rekaman, dan analitik.

2.1 CMP Hub
Halaman utama modul CMP. Menyediakan akses ke fitur Dokumen KKJ, Assessment, Records, dan Analytics.
2.2 Assessment
Halaman ujian assessment kompetensi. Pekerja mengerjakan soal sesuai paket yang telah ditetapkan dengan batas waktu tertentu.
2.3 Records
Riwayat hasil assessment pekerja. Menampilkan skor, status kelulusan, dan detail per sesi ujian.
2.4 Analytics
Dashboard analitik CMP dengan grafik dan statistik performa assessment di seluruh unit kerja.

3. CDP — Career Development Portal

Modul pengembangan karir: IDP, Coaching Proton, dan dashboard perkembangan.

3.1 CDP Hub
Halaman utama modul CDP. Menyediakan akses ke Plan IDP, Coaching Proton, Dashboard, dan Deliverable.
3.2 Plan IDP
Halaman perencanaan Individual Development Plan (IDP). Pekerja menyusun rencana pengembangan kompetensi mereka.
3.3 Coaching Proton
Halaman Coaching Proton untuk mencatat sesi coaching antara coach dan coachee, termasuk progress dan feedback.
3.4 CDP Dashboard
Dashboard perkembangan karir yang menampilkan progress IDP, statistik coaching, dan pencapaian deliverable.

4. Admin — Kelola Data

Modul administrasi untuk pengelolaan data pekerja, assessment, dan mapping.

4.1 Admin Hub
Halaman utama modul Admin. Menyediakan akses ke semua fitur kelola data: Manajemen Pekerja, Assessment, Mapping, dan lainnya.
4.2 Manajemen Pekerja
Halaman kelola data pekerja. Admin dan HC dapat menambah, mengedit, mengimpor, dan mengelola data seluruh pekerja.
4.3 Kelola Assessment
Halaman admin untuk mengelola paket soal, jadwal assessment, dan konfigurasi ujian kompetensi.
4.4 Mapping
Halaman mapping untuk mengatur hubungan coach-coachee, assignment pekerja ke unit, dan konfigurasi lainnya.

5. Proton Data & Panduan

Kelola data silabus Proton dan halaman panduan penggunaan.

5.1 Proton Data
Halaman kelola data Proton: silabus, guidance, dan override. Diakses oleh Admin dan HC untuk mengatur materi coaching.
5.2 Halaman Panduan
Halaman panduan penggunaan HC Portal. Berisi dokumen-dokumen panduan yang dapat diakses dan diunduh oleh semua pengguna.
